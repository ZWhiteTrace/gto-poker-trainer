# Bluff Catching 原則：何時該抓詐唬？

## 什麼是 Bluff Catching？

**Bluff Catching（抓詐唬）** 是用能贏對手詐唬、但輸給取值牌的手牌跟注。

```
範例：
牌面：A♠ K♦ 8♣ 4♥ 2♠
你的手牌：Q♠ Q♥

分析：
- 你贏對手的詐唬（錯失的聽牌）
- 你輸給 AK, AA, KK, A8, A4 等
- QQ 是典型的 bluff catcher
```

---

## 何時需要抓詐唬？

### 情境判斷

```
對手河牌大注 → 他的範圍極化
極化範圍 = 堅果 or 空氣

問題：他有多少空氣？

如果空氣夠多 → 你需要抓詐唬
如果空氣太少 → 你應該棄牌
```

### 極化範圍的特徵

當對手的下注表示極化範圍時：
- 大尺度下注（66%+ pot）
- 河牌激進行動
- 牌面發展不利於對手範圍

```
範例牌面發展：
翻牌：K♣ J♥ 9♠（濕潤）
轉牌：3♦
河牌：2♣

分析：
- 很多聽牌沒有完成
- 對手大注 = 有 KJ+，或在詐唬
- 這是抓詐唬的好場景
```

## Bluff Catcher 的選擇

### 好的 Bluff Catcher

```
特點：
✓ 能贏大部分詐唬
✓ 有阻擋效應（阻擋對手取值牌）
✓ 沒有攤牌價值（對手不會用更差的牌跟注）
```

#### 範例 1：阻擋對手強牌

```
牌面：A♠ K♦ Q♣ 4♥ 2♠
你有：A♣ T♣

分析：
- 你有 A 阻擋 AK, AQ, AA
- 減少對手堅果組合
- 這是好的 bluff catcher
```

#### 範例 2：贏所有詐唬

```
牌面：K♠ 8♦ 5♣ 3♥ 2♠
你有：K♦ J♦

分析：
- 你贏所有錯失的聽牌
- 只輸給 K8, 88, 55, 33, 22
- KJ 是有效的 bluff catcher
```

### 差的 Bluff Catcher

```
特點：
✗ 輸給部分詐唬
✗ 沒有阻擋效應
✗ 有攤牌價值但不夠抓詐唬
```

#### 範例：太弱的牌

```
牌面：A♠ K♦ 8♣ 4♥ 2♠
你有：7♦ 7♣

分析：
- 77 輸給 88, A8, K8 等中等牌
- 這些可能不是取值範圍
- 77 不夠強抓詐唬
```

## 數學基礎：MDF 與跟注頻率

### 最小防守頻率 (MDF)

**MDF** = 你需要跟注的頻率，讓對手的純詐唬 EV 為 0

```
公式：
MDF = Pot / (Pot + Bet)

範例：
底池 $100，對手下注 $75
MDF = 100 / (100 + 75) = 57%

意思：你需要跟注 57% 的時間
讓對手無法透過純詐唬獲利
```

### 常見下注尺度的 MDF

| 下注尺度 | MDF |
|----------|-----|
| 33% pot | 75% |
| 50% pot | 67% |
| 66% pot | 60% |
| 75% pot | 57% |
| 100% pot | 50% |
| 150% pot | 40% |

### MDF 的應用

```
情境：
底池 $100，對手河牌下注 $100（滿池）
MDF = 50%

分析：
- 你需要跟注 50% 的時間
- 對手只有 50% 詐唬會成功
- 如果你的範圍有 10 個組合
- 你應該用 5 個組合跟注
```

## 範圍分析

### 步驟 1：估計對手範圍

```
牌面：K♠ Q♦ 7♣ 3♥ 2♠
對手在河牌下注 75% pot

他的可能範圍：
取值：KQ, K7, 77, 33, 22, AA
詐唬：JT（順子聽牌沒中）, 同花聽牌, Ax
```

### 步驟 2：計算組合

```
取值組合：
- KQ: 9 組合
- K7s: 2 組合
- 77: 3 組合
- 33: 3 組合
- 22: 3 組合
- AA: 6 組合
總計: 26 組合

詐唬組合：
- JT: 12 組合（部分可能放棄）
- A♠ x♠: ~10 組合
- 其他: ~8 組合
估計: ~20 組合

詐唬比例: 20/46 ≈ 43%
```

### 步驟 3：決定是否跟注

```
MDF = 57%（對手下注 75% pot）
對手詐唬比例估計 = 43%

問題：對手詐唬夠多嗎？

如果你認為對手詐唬頻率 > 43%
→ 抓詐唬有利

如果對手很緊，只詐唬 20%
→ 棄牌
```

## 阻擋效應詳解

### 正面阻擋

**持有阻擋對手取值牌的牌**

```
牌面：A♠ K♦ T♣ 4♥ 2♠
你有：A♣ J♣

阻擋：
- AK: 本來 12 組合 → 9 組合
- AA: 本來 6 組合 → 3 組合
- ATs: 本來 3 組合 → 2 組合

你的 A 減少對手堅果組合
→ 好的 bluff catcher
```

### 負面阻擋

**持有阻擋對手詐唬牌的牌**

```
牌面：K♠ Q♦ 8♣ 4♥ 2♠
你有：J♥ T♥

問題：
- JT 阻擋 J9, T9 等詐唬
- 對手詐唬的牌變少
- 這不是好的 bluff catcher

你應該棄牌，選擇其他牌抓詐唬
```

### 阻擋效應總結

| 持有 | 對抓詐唬的影響 |
|------|----------------|
| 阻擋取值牌 | 有利 ✓ |
| 阻擋詐唬牌 | 不利 ✗ |
| 兩者都阻擋 | 中性 |

## 實戰決策框架

### 決策流程

```
1. 識別極化範圍
   對手大注 → 他要嘛很強，要嘛在詐唬

2. 計算 MDF
   根據下注尺度計算需要跟注的頻率

3. 估計詐唬頻率
   對手有多少組合是詐唬？

4. 評估阻擋效應
   你的牌阻擋什麼？

5. 做出決定
   綜合以上因素決定跟注或棄牌
```

### 快速判斷

```
跟注傾向：
✓ 對手激進，詐唬頻率高
✓ 你的牌阻擋取值
✓ 牌面有很多沒完成的聽牌
✓ 對手之前表現薄弱

棄牌傾向：
✗ 對手保守，很少詐唬
✗ 你的牌阻擋詐唬
✗ 牌面沒有什麼聽牌
✗ 對手之前表現強勢
```

## 常見場景分析

### 場景 1：河牌大注

```
你：A♠ T♠
牌面：A♦ K♣ 7♥ 5♠ 2♦
對手：河牌下注 $80 進 $100 底池

分析：
- 你有頂對弱踢腳
- 對手下注 80% pot
- MDF ≈ 55%

對手取值：AK, A7, 77, 55
對手詐唬：QJ（順子聽牌）, 同花聽牌

你的 A 阻擋 AK, AA
→ 傾向跟注
```

### 場景 2：連續下注後的河牌

```
你：Q♠ Q♥
牌面：K♦ T♣ 6♥ 3♠ 2♦
對手：翻牌、轉牌、河牌都下注

分析：
- 對手三條街都下注 = 範圍很強
- QQ 在這牌面只贏詐唬
- 對手詐唬頻率在三條街後通常低

風險評估：
- 對手可能有 Kx, TT, 66, 33
- 三條街詐唬需要勇氣
→ 傾向棄牌（除非對手非常激進）
```

### 場景 3：錯失明顯聽牌

```
你：K♦ K♣
牌面：Q♠ J♥ 9♣ 3♦ 2♠
對手：河牌下注滿池

分析：
- 翻牌有順子聽牌（KT, T8）
- 河牌沒有完成順子
- 對手可能用錯失的聽牌詐唬

MDF = 50%
對手詐唬範圍很寬
→ 傾向跟注
```

## 對手類型調整

### vs 緊玩家
```
特點：很少詐唬
調整：減少抓詐唬頻率
只用最強的 bluff catcher 跟注
```

### vs 鬆激進玩家
```
特點：詐唬頻率高
調整：增加抓詐唬頻率
用更寬的範圍跟注
```

### vs 不明玩家
```
策略：使用接近 GTO 的 MDF
不要過度調整
收集信息
```

## 常見錯誤

### 1. 用太弱的牌抓詐唬
```
❌ 用底對抓詐唬（可能輸給中等牌）
✅ 用頂對弱踢腳抓詐唬
```

### 2. 忽略阻擋效應
```
❌ 隨機選牌跟注
✅ 選擇阻擋對手取值的牌
```

### 3. 不調整對手類型
```
❌ 對保守老人頻繁抓詐唬
✅ 根據對手傾向調整
```

### 4. 過度抓詐唬
```
❌ 每次都跟注「抓詐唬」
✅ 用 MDF 計算合理頻率
```

## 總結

### Bluff Catching 核心原則

1. **識別極化範圍** = 對手大注時
2. **使用 MDF** 計算跟注頻率
3. **評估阻擋效應** 選擇最佳 bluff catcher
4. **估計詐唬頻率** 根據牌面和對手
5. **調整對手類型** 不要一成不變
