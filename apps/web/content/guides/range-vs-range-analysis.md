# 範圍對範圍分析：從手牌思維到範圍思維

## 從「他有什麼牌」到「他可能有什麼範圍」

**初級思維**：「他有 AA 嗎？他有同花嗎？」

**進階思維**：「他的範圍是什麼？我的範圍在這個牌面表現如何？」

```
思維轉變：

❌ 他可能有 AK
✅ 他的範圍中有 16 組合 AK，加上 30 組合其他強牌...

❌ 我有頂對，應該下注
✅ 我的範圍在這個牌面有優勢，應該選擇合適的下注策略
```

---

## 範圍基礎概念

### 什麼是範圍？

**範圍 (Range)** 是玩家在特定情況下可能持有的所有手牌組合。

```
範例：UTG 開池範圍（約 15%）

高對：AA, KK, QQ, JJ, TT, 99
中對：88, 77, 66
強牌：AKs, AKo, AQs, AQo, AJs
同花牌：ATs, KQs, KJs, QJs, JTs

總計約 120 種組合
```

### 組合計算

| 手牌類型 | 組合數 | 範例 |
|----------|--------|------|
| 對子 | 6 | AA = 6 種組合 |
| 同花 | 4 | AKs = 4 種組合 |
| 非同花 | 12 | AKo = 12 種組合 |
| 指定花色 | 1 | A♠K♠ = 1 種組合 |

### 阻擋效應對組合的影響

```
牌面：A♠ K♦ 7♣
你的手牌：A♥ Q♥

AK 組合變化：
- 原本：16 組合
- A 被阻擋：剩 12 組合
- 牌面有 A, K：剩 9 組合
```

## 範圍優勢與劣勢

### 什麼是範圍優勢？

當你的範圍在特定牌面上整體更強時，你有**範圍優勢**。

```
範例 1：翻牌 A-K-7

你：UTG 開池
對手：BB 跟注

範圍比較：
UTG 範圍：大量 AK, AA, KK, AQ, AJ
BB 範圍：少量 AK（會 3-bet），較多 A7, K7, 小對子

結論：UTG 有明顯範圍優勢
```

### 範圍優勢的表現

| 指標 | 說明 |
|------|------|
| 權益領先 | 你的範圍平均權益更高 |
| 堅果優勢 | 你的範圍有更多強牌 |
| 封頂 vs 無封頂 | 對手範圍有上限，你沒有 |

### 範圍劣勢

```
範例 2：翻牌 7-6-5（兩張同花）

你：BTN 開池
對手：BB 跟注

範圍比較：
BTN 範圍：高牌多，這牌面擊中少
BB 範圍：很多 87, 98, 76, 65, 54, 同花連牌

結論：BB 有範圍優勢，BTN 應該收斂
```

## 牌面紋理分析

### 乾燥牌面 (Dry)

```
特徵：
- 高牌 + 低牌
- 很少聽牌
- 範圍容易定義

範例：A♠ 7♦ 2♣

分析：
- 進攻者通常有明顯範圍優勢
- 可以高頻率 C-bet
- 防守者範圍較弱
```

### 濕潤牌面 (Wet)

```
特徵：
- 連接牌
- 同花可能
- 很多聽牌

範例：J♠ T♠ 8♥

分析：
- 雙方範圍都可能擊中
- 需要謹慎評估
- 下注需要保護權益
```

### 配對牌面

```
特徵：
- 有一對在牌面
- 減少強牌組合
- Trips 變得重要

範例：K♠ K♦ 4♣

分析：
- 持有 K 的人很強
- 但 K 的組合已經很少
- 範圍封頂概念重要
```

## 範圍封頂 (Range Capping)

### 什麼是範圍封頂？

當玩家的行動排除了某些強牌，他的範圍就被**封頂**了。

```
範例：

翻牌：Q♠ J♥ 4♣
你 C-bet，對手只是跟注

對手範圍分析：
- 他沒有加注 = 可能沒有 QJ, JJ, 44
- 他的範圍被封頂於 Qx, Jx, 聽牌
- 你可以在之後的街施壓
```

### 識別封頂範圍

| 行動 | 封頂暗示 |
|------|----------|
| 翻牌只跟注 | 通常沒有 set, 兩對 |
| 轉牌過牌 | 通常放棄或中等牌 |
| 面對 3-bet 只跟注 | 通常不是 AA, KK |

### 利用封頂範圍

```
策略：
當對手範圍封頂，你可以：
1. 更頻繁詐唬
2. 用更大尺度下注
3. 代表超強牌

範例：
對手在 K-Q-7-3-2 只跟注三條街
他的範圍被封頂於 Kx, Qx
你可以大注詐唬，代表 KQ, 77, 33
```

## 極化 vs 合併範圍

### 極化範圍

```
定義：只有很強或很弱的牌

結構：
[堅果] + [空氣]
沒有中間牌

適用場景：
- 河牌決策
- 大尺度下注
- 對抗弱範圍
```

### 合併範圍

```
定義：包含各種牌力

結構：
[強牌] + [中等牌] + [弱牌]
連續分佈

適用場景：
- 小尺度下注
- 範圍優勢明顯
- 不需要極化
```

### 實戰應用

```
牌面：K♠ 7♦ 2♣

你是 CO 開池者，BB 跟注

選項 A：小注 1/3 pot（合併範圍）
- 用整個範圍下注
- 包含 Kx, 口袋對, A-high

選項 B：中注 2/3 pot（極化範圍）
- 用強牌取值（KQ+, sets）
- 用詐唬（後門聽牌, A-high）

Solver 建議：此牌面適合高頻小注
```

## 範圍分析工具

### 手動分析步驟

```
步驟 1：確定起始範圍
- 翻前行動決定
- 使用標準範圍作為基礎

步驟 2：根據牌面過濾
- 什麼牌擊中？
- 什麼牌有聽牌？
- 什麼牌完全沒有？

步驟 3：根據行動過濾
- 什麼牌會下注？
- 什麼牌會加注？
- 什麼牌會棄牌？

步驟 4：計算組合
- 計算每種手牌的組合數
- 考慮阻擋效應
```

### 範例分析

```
情境：
UTG 開池，BTN 跟注
翻牌：A♥ K♦ 7♠
UTG C-bet 50%，BTN 跟注

BTN 跟注範圍分析：

跟注的牌：
- AQ, AJ, AT：~24 組合
- KQ, KJ：~24 組合
- QJ（順子聽牌）：16 組合
- 77：3 組合
- 口袋對 88-QQ：30 組合

會加注的牌（移除）：
- AK：會加注
- AA, KK：可能加注
- 77：可能加注

會棄牌的牌（移除）：
- 低同花連牌
- 完全沒擊中的牌

總結：BTN 範圍約 80 組合，被封頂於 AQ/KQ
```

## 實戰範例

### 範例 1：翻牌範圍對抗

```
你：CO 開池 A♠ Q♠
對手：BB 跟注
翻牌：A♦ 8♣ 3♥

範圍分析：
你的範圍：所有 Ax, 高對, 同花連牌
對手範圍：Ax（較少 AK、AQ 會 3bet）, 口袋對, 連牌

範圍優勢：你有優勢
- 你有更多 AK, AQ
- 對手有 A8, A3 但數量少

策略：
- 高頻 C-bet
- 小尺度（1/3 pot）足夠
- 用整個範圍下注
```

### 範例 2：轉牌範圍變化

```
承上例
轉牌：T♣

新的分析：
- T 可能幫助對手（JT, T9, T8）
- 你的 AQ 仍然領先
- 但對手範圍沒有被封頂

策略調整：
- 繼續下注（你仍有範圍優勢）
- 可以增加尺度（50-66%）
- 準備面對加注
```

### 範例 3：河牌極化決策

```
承上例
河牌：2♠

牌面：A♦ 8♣ 3♥ T♣ 2♠

你的選擇：
A) 下注取值
B) 過牌

分析：
- 你有 AQ（頂對好踢腳）
- 對手範圍：Ax（可能跟注）, 錯失聽牌（不跟）
- AQ 是取值範圍的底部

決定：小注取值（1/3 pot）
- 可以被 AT, A9, A8 跟注
- 可以被錯失聽牌棄牌
- 不需要大注
```

## GTO 範圍平衡

### 平衡的概念

```
平衡 = 你的範圍中有適當比例的取值和詐唬

不平衡的問題：
- 只用強牌下注 → 對手知道你下注=強
- 只用弱牌詐唬 → 對手總是跟注

平衡的目標：
- 讓對手無法從你的行動判斷牌力
- 即使被讀也不影響 EV
```

### 建構平衡範圍

```
下注範圍應該包含：
1. 強取值（明確贏錢）
2. 中等取值（薄值）
3. 詐唬（權益+阻擋）
4. 過牌範圍（攤牌價值+陷阱）

比例原則：
取值 : 詐唬 ≈ 根據下注尺度
66% pot → 60:40
100% pot → 67:33
```

## 常見錯誤

### 1. 只看自己的牌
```
❌ 「我有 AK，應該下注」
✅ 「我的範圍在這個牌面有優勢，應該下注」
```

### 2. 忽略範圍封頂
```
❌ 對手只跟注三街還怕他有堅果
✅ 認識到對手範圍被封頂
```

### 3. 不考慮阻擋效應
```
❌ 隨機選擇詐唬牌
✅ 選擇阻擋對手強牌的牌詐唬
```

### 4. 用同一策略對所有對手
```
❌ 對所有人用平衡策略
✅ 根據對手範圍傾向調整
```

## 總結

### 範圍思維核心

1. **用範圍思考**，而非單一手牌
2. **分析牌面**對雙方範圍的影響
3. **識別範圍優勢**和封頂情況
4. **平衡你的範圍**以保持不可讀
5. **計算組合**做出精確決策

### 範圍分析清單

```
□ 我的翻前範圍是什麼？
□ 對手的翻前範圍是什麼？
□ 這個牌面對誰有利？
□ 對手的行動封頂他的範圍了嗎？
□ 我應該用什麼範圍下注/過牌？
```
