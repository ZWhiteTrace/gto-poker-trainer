# 錦標賽階段策略：從開賽到決賽桌

## 錦標賽的獨特性

**錦標賽 (Tournament)** 與現金局有根本性的不同：

```
關鍵差異：

現金局：
- 可以隨時買入
- 籌碼 = 真實金錢
- 目標：最大化每手 EV

錦標賽：
- 只有一次買入
- 籌碼 = 生存權
- 目標：最大化獎金 EV
```

---

## ICM：錦標賽決策的核心

### 什麼是 ICM？

**ICM (Independent Chip Model)** 是將籌碼轉換為獎金價值的模型。

```
核心概念：

籌碼不等於金錢：
- 第一個籌碼最有價值
- 邊際籌碼價值遞減
- 輸掉所有籌碼 = 出局

範例：
3 人剩餘，獎金 50/30/20
玩家 A：5000 籌碼
玩家 B：3000 籌碼
玩家 C：2000 籌碼

ICM 價值 ≠ 籌碼比例：
A 的價值 < 50%
B 的價值 > 30%
C 的價值 > 20%
```

### ICM 的影響

```
風險厭惡：

因為籌碼價值非線性：
- 贏的籌碼價值 < 輸的籌碼價值
- 應該避免邊際決策
- 生存優先

範例：
決賽桌泡沫邊緣
你有中等籌碼
面對大籌碼的全下

chip EV：可能是跟注
ICM EV：通常是棄牌
（因為出局損失太大）
```

## 錦標賽階段劃分

### 早期階段 (Early Stage)

```
特點：
- 籌碼深（100BB+）
- 離獎金很遠
- 可以打得像現金局

盲注結構：
籌碼 / 大盲 > 50

策略要點：
1. 可以玩投機牌（小對子、同花連牌）
2. 不需要冒險
3. 建立籌碼和形象
4. 觀察對手
```

### 中期階段 (Middle Stage)

```
特點：
- 籌碼中等（30-60BB）
- 開始有淘汰壓力
- 需要積累籌碼

盲注結構：
籌碼 / 大盲 = 30-50

策略要點：
1. 開始偷盲
2. 不要讓籌碼太少
3. 找機會加倍
4. 避免邊際決策
```

### 泡沫階段 (Bubble)

```
特點：
- 即將進入獎金圈
- 短籌碼拼命求生
- 中籌碼最難打

盲注結構：
接近獎金圈

策略要點：

大籌碼：
- 攻擊中籌碼
- 他們不敢反擊
- 積極偷盲

中籌碼：
- 避免和大籌碼對抗
- 攻擊短籌碼
- 等待更好機會

短籌碼：
- 找機會全下
- 利用 fold equity
- 不要等到沒有籌碼
```

### 獎金圈 (In The Money)

```
特點：
- 已經確保獎金
- 可以稍微放鬆
- 但目標是更高名次

策略要點：
1. 泡沫後可以調整
2. 但不要過度激進
3. 記住下一個 pay jump
4. 繼續積累籌碼
```

### 決賽桌 (Final Table)

```
特點：
- 最高獎金在這裡
- 每個名次差距大
- ICM 影響最大

策略要點：
1. 每個決定都考慮 ICM
2. 利用籌碼優勢
3. 注意籌碼結構
4. 大對小有巨大優勢
```

## Push/Fold 策略

### 何時進入 Push/Fold 模式

```
籌碼深度指標：
< 15BB：主要 push/fold
< 10BB：幾乎只能 push/fold

原因：
- 沒有足夠籌碼翻後遊玩
- 開池加注後被 3-bet 很難棄牌
- 全下可以最大化 fold equity
```

### 基本 Push 範圍

```
不同籌碼深度的參考（BTN vs 盲注）：

15BB Push 範圍（約 25%）：
22+, A2s+, A7o+, K9s+, KTo+, Q9s+, QTo+
J9s+, JTo, T9s, 98s, 87s, 76s, 65s

10BB Push 範圍（約 40%）：
22+, A2+, K2s+, K5o+, Q4s+, Q8o+
J6s+, J8o+, T7s+, T9o, 97s+, 87s, 76s

5BB Push 範圍（約 60%）：
幾乎任何兩張牌從後位
```

### 跟注全下範圍

```
一般原則：
跟注範圍比 push 範圍窄

原因：
- 你沒有 fold equity
- 需要攤牌贏

BB vs BTN 5BB 全下的跟注範圍：
22+, A2s+, A5o+, K7s+, K9o+, Q9s+, QTo+
JTs, T9s
```

## 籌碼結構的影響

### 大籌碼策略

```
優勢：
- 可以壓制小籌碼
- 中籌碼不敢反擊
- 可以更積極

策略：
1. 頻繁偷盲（特別是對中籌碼）
2. 利用 ICM 壓力
3. 對短籌碼保持警惕
4. 不要輕易把籌碼給其他大籌碼
```

### 中籌碼策略

```
困境：
- 被大籌碼壓制
- 不想冒險出局
- 但需要籌碼

策略：
1. 選擇性攻擊
2. 避開大籌碼
3. 找短籌碼對抗
4. 等待好牌
```

### 短籌碼策略

```
優勢：
- ICM 影響較小
- 可以更激進
- 別人怕你全下

策略：
1. 找機會全下
2. 不要等太久
3. 利用你的「all-in 權」
4. 盡量在有 fold equity 時行動
```

## 特殊場景

### Satellite（衛星賽）

```
特點：
- 所有晉級者獲得相同獎勵
- 不需要所有籌碼
- 只需要晉級

策略：
- 更加保守
- 確保晉級比籌碼量重要
- 泡沫時非常緊

範例：
剩 15 人晉級 10 人
你是第 8 籌碼
→ 極度保守，等待淘汰
```

### Shootout

```
特點：
- 每桌只有一人晉級
- 像單桌比賽
- 必須打到最後

策略：
- 打法更像現金局
- 需要積累籌碼
- 沒有 ICM 考量（直到最後幾人）
```

### PKO（Progressive Knockout）

```
特點：
- 淘汰對手有賞金
- 每人頭上有價值
- 影響決策

策略：
- 大籌碼更有誘因攻擊
- 短籌碼被攻擊更多
- 要計算賞金的影響

計算：
跟注 EV = 贏的籌碼 + 賞金 - 輸的籌碼
```

## 常見錯誤

### 1. 忽略 ICM

```
❌ 像現金局一樣計算 EV
✅ 考慮籌碼的真實價值
```

### 2. 泡沫太激進

```
❌ 泡沫期用邊緣牌冒險
✅ 選擇性攻擊，避免大對抗
```

### 3. 短籌碼等太久

```
❌ 等到 3BB 才全下
✅ 在 10-15BB 時就要行動
```

### 4. 不調整對手範圍

```
❌ 認為對手和平常一樣打
✅ 考慮對手的籌碼和壓力
```

## 總結

### 錦標賽階段核心

1. **早期**：像現金局，建立籌碼
2. **中期**：開始偷盲，維持籌碼
3. **泡沫**：利用 ICM，選擇性攻擊
4. **獎金圈**：繼續積累，目標更高
5. **決賽桌**：每個決定考慮 ICM

### 籌碼管理原則

```
□ 不要讓籌碼跌到 < 15BB
□ 短籌碼要主動全下
□ 中籌碼避免邊緣決策
□ 大籌碼利用 ICM 壓力
□ 考慮 pay jump 的影響
```
