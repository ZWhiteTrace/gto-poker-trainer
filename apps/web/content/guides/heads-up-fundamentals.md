# 單挑基礎：一對一的撲克對決

## 單挑的獨特性

**單挑 (Heads-Up)** 是撲克最純粹的形式：只有你和一個對手。

```
核心差異：

常規桌（6-max）：
- 多人競爭
- 可以等待好牌
- 位置輪轉慢

單挑：
- 只有一個對手
- 必須頻繁參與
- 每手都在盲注位置
```

---

## 單挑的基本結構

### 位置與行動順序

```
翻前：
按鈕（SB）先行動
大盲（BB）後行動

翻後：
大盲（BB）先行動
按鈕（SB/BTN）後行動

關鍵理解：
按鈕 = 翻前劣勢 + 翻後優勢
大盲 = 翻前優勢 + 翻後劣勢
```

### 盲注結構

```
按鈕投入小盲注
大盲投入大盲注

特點：
- 每手都要投錢
- 不能等待
- 必須打得寬
```

---

## 翻前策略

### 按鈕（SB）策略

```
開池範圍：約 80-100%

原因：
- 翻後有位置優勢
- 對手很難防守所有
- 棄牌 = 白送大盲

具體範圍：
打所有：22+, A2+, K2+, Q2+
只棄最差的垃圾牌
```

#### 開池尺度

```
常見選擇：

加注到 2-2.5BB：
標準尺度，適合大部分情況

加注到 3BB：
較大尺度，壓縮對手範圍

跛入（Limp）：
部分策略會混合跛入
優點：控制底池，保留範圍
```

### 大盲（BB）策略

```
防守範圍：非常寬

原因：
- 已投入 1BB
- 底池賠率好
- 對手範圍很寬

具體策略：

面對 2.5x 開池：
跟注：約 60% 的牌
3-bet：約 15% 的牌
棄牌：約 25% 最弱的牌
```

---

## 3-Bet 策略

### BB 的 3-Bet

```
3-Bet 範圍（約 15%）：

取值 3-Bet：
AA, KK, QQ, JJ, TT, 99
AKs, AKo, AQs, AQo, AJs
KQs

詐唬 3-Bet：
A5s, A4s, A3s, A2s（阻擋牌）
K9s, K8s, Q9s
76s, 65s, 54s（可玩性）
```

### 按鈕面對 3-Bet

```
一般建議：

4-Bet：AA, KK, QQ, AKs + 詐唬
跟注：JJ, TT, 99, AQs, AJs, KQs
棄牌：弱牌

特別注意：
單挑中對手 3-bet 範圍寬
不要太容易放棄
```

---

## 翻後策略

### C-Bet 策略

```
按鈕 C-Bet：

高頻率下注：
- 你有位置優勢
- 對手必須先行動
- 可以用 50-70% 範圍 C-bet

尺度選擇：
乾燥牌面：25-33% pot
濕潤牌面：50-66% pot
```

### 大盲面對 C-Bet

```
防守策略：

繼續率：需要防守 50-60%

方式：
- 跟注：中等牌、好聽牌
- 加注：強牌、部分詐唬
- 棄牌：最弱的牌

特別策略：
Donk Bet：某些牌面可以先下注
Check-Raise：設陷阱、半詐唬
```

## 特殊技巧

### 跛入策略 (Limping)

```
按鈕跛入的考量：

優點：
- 控制底池
- 保持範圍隱蔽
- 允許更寬的參與

缺點：
- 讓大盲免費看牌
- 無法建立初始攻勢

混合策略：
有些玩家會混合跛入
例：跛入 50%，加注 50%
```

### 小球策略 (Small Ball)

```
概念：
使用較小的下注尺度
更頻繁地參與底池

適用場景：
- 對手容易被壓制
- 你的讀牌能力強
- 想要控制變異

實施：
- 小尺度開池（2x）
- 小尺度 C-bet（25-33%）
- 多看牌，少全下
```

### 激進風格 (LAG)

```
概念：
寬範圍 + 高攻擊性

適用場景：
- 對手太被動
- 對手不適應
- 籌碼深度足夠

風險：
- 需要好的翻後技巧
- 變異較大
- 對手可能調整
```

## 讀牌與調整

### 快速調整的重要性

```
單挑特點：
- 樣本累積快
- 對手傾向明顯
- 必須快速調整

觀察重點：
1. 翻前加注頻率
2. C-bet 頻率
3. 面對下注的反應
4. 詐唬傾向
```

### 常見對手類型

#### 被動玩家

```
特徵：
- 很少加注
- 傾向跟注
- 很少詐唬

調整：
- 減少詐唬
- 增加取值下注
- 用更薄的牌取值
```

#### 激進玩家

```
特徵：
- 頻繁加注
- 喜歡詐唬
- 施壓不斷

調整：
- 設陷阱（過牌-加注）
- 準備跟注更多
- 用更寬範圍繼續
```

#### 緊弱玩家

```
特徵：
- 翻前太緊
- 容易棄牌
- 只打強牌

調整：
- 頻繁偷盲
- 更多 C-bet
- 尊重他的加注
```

## 籌碼深度的影響

### 深籌碼（100BB+）

```
特點：
- 翻後空間大
- 投機牌價值高
- 技術差距明顯

策略：
- 可以打更多牌
- 重視位置
- 翻後操作空間大
```

### 中籌碼（30-60BB）

```
特點：
- 開始收緊
- 3-bet 變得重要
- 全下是選項

策略：
- 減少投機牌
- 重視 3-bet 動態
- 準備全下
```

### 短籌碼（<20BB）

```
特點：
- Push/Fold 為主
- 沒有翻後空間
- 高變異

策略：
- 使用 push/fold 圖表
- 位置極為重要
- 不要等太久
```

## 心理戰

### 保持侵略

```
單挑中被動 = 失敗

原因：
- 盲注不停消耗
- 被動讓對手控制
- 機會成本高

心態：
- 保持攻擊性
- 但不是魯莽
- 選擇好的時機
```

### 處理下風

```
單挑變異大：
- 經常輸大底池
- 情緒影響明顯
- 需要強大心理

應對：
- 認識到變異是正常的
- 不要因一手牌傾斜
- 專注於做出好決策
```

### 利用對手情緒

```
觀察對手狀態：

傾斜信號：
- 開始打得更鬆
- 行動變快
- 明顯沮喪

利用：
- 增加取值
- 減少詐唬
- 讓他犯更多錯
```

## 常見錯誤

### 1. 翻前太緊

```
❌ 像 6-max 一樣只打 20% 的牌
✅ 單挑需要打 70%+ 的牌
```

### 2. 翻後太被動

```
❌ 經常過牌，等強牌
✅ 積極下注，控制底池
```

### 3. 不調整對手

```
❌ 用固定策略
✅ 根據對手快速調整
```

### 4. 不尊重深度

```
❌ 深籌碼和淺籌碼打法一樣
✅ 根據籌碼深度調整策略
```

## 總結

### 單挑核心原則

1. **打得寬**：每手都要參與
2. **保持侵略**：被動等於失敗
3. **利用位置**：按鈕是翻後優勢
4. **快速調整**：讀取並適應對手
5. **控制情緒**：處理高變異

### 單挑檢查清單

```
翻前：
□ 按鈕開池夠寬嗎？（80%+）
□ 大盲防守夠多嗎？（60%+）
□ 3-bet 範圍平衡嗎？

翻後：
□ 利用位置優勢了嗎？
□ C-bet 頻率適當嗎？
□ 讀取對手傾向了嗎？

心理：
□ 保持專注了嗎？
□ 情緒穩定嗎？
□ 準備好打持久戰嗎？
```
