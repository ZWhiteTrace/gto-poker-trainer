# 深籌碼策略：100BB+ 的高階玩法

## 什麼是深籌碼？

**深籌碼 (Deep Stack)** 通常指 100BB 以上的籌碼量。

```
籌碼深度分類：

超深籌碼：200BB+
深籌碼：100-200BB
標準籌碼：40-100BB
短籌碼：<40BB

深籌碼的特點：
- 翻後空間巨大
- 投機牌價值提高
- 位置更加重要
- 技術差距被放大
```

---

## 深籌碼的優勢與特性

### 投機牌價值增加

```
為什麼投機牌變強：

小對子（22-66）：
- 標準籌碼：隱含賠率約 10:1
- 深籌碼：隱含賠率可達 20:1+
- 中 set 可以贏大底池

同花連張（76s, 87s）：
- 可以贏對手整個籌碼
- 中牌時很隱蔽
- 翻後操作空間大

結論：
深籌碼時要更願意玩投機牌
```

### 位置的重要性

```
深籌碼位置價值：

有位置的優勢放大：
- 更多資訊收集機會
- 可以控制底池大小
- 詐唬機會更多

沒位置的劣勢放大：
- 難以知道何時領先
- 容易被壓制
- 大底池決策困難

建議：
深籌碼時更嚴格選擇開池範圍
特別是在沒有位置時
```

---

## 翻前策略調整

### 開池策略

```
深籌碼開池調整：

前位（UTG, MP）：
- 保持緊的範圍
- 強調強牌和可遊玩性
- 減少容易被主導的牌

範例（150BB UTG）：
開池：77+, ATs+, AJo+, KQs
謹慎：22-66, A2s-A9s

後位（CO, BTN）：
- 可以稍微放寬
- 但要考慮被 3-bet 後的可玩性
- 避免尷尬的 SPR
```

### 3-Bet 策略

```
深籌碼 3-bet 考量：

範圍要更極化：
價值端：QQ+, AK
詐唬端：A5s-A2s, KQo 部分

為什麼不用中等牌 3-bet：
- 建立太大底池
- 面對 4-bet 進退兩難
- 中牌後難以判斷

中等牌處理：
- JJ, TT, AQo
- 更傾向跟注
- 看翻牌再決定
```

### 面對 3-Bet

```
深籌碼 vs 3-bet：

你開池被 3-bet 時：

跟注更有吸引力：
- 有足夠隱含賠率
- 翻後可以操作
- 不需要馬上決生死

4-bet 要更謹慎：
- 只用超強牌
- AA, KK 必然 4-bet
- QQ, AK 可以混合

範例（150BB CO 開池被 BTN 3-bet）：
4-bet：AA, KK, A5s
Call：QQ, JJ, TT, AKs, AKo, AQs
Fold：弱牌
```

## 翻牌圈策略

### SPR 的概念

```
SPR = Stack to Pot Ratio
= 有效籌碼 / 翻牌底池

深籌碼特點：
- SPR 通常 > 10
- 很難在翻牌承諾所有籌碼
- 需要多條街建立底池

SPR 決策指導：
SPR > 13：很難只靠一對贏
SPR 8-13：頂對可以取值但要小心
SPR < 8：頂對好 kicker 可以全進
```

### C-Bet 策略調整

```
深籌碼 C-bet：

使用較小尺寸：
- 25-33% pot 更常見
- 控制底池
- 保持範圍平衡

為什麼不大注：
- 建立太大底池
- 對手跟注後 SPR 變低
- 失去翻後操作空間

範例：
翻牌 K♠8♦3♥
你有 A♠K♥
底池 $30，有效籌碼 $300

深籌碼做法：bet $10
標準籌碼做法：bet $15-20
```

### 慢打考量

```
深籌碼慢打更常見：

什麼牌適合慢打：
- 暗三
- 堅果同花
- 頂兩對

為什麼深籌碼慢打更好：
- 對手有足夠籌碼詐唬
- 可以後續街取值
- 避免過早趕走對手

範例：
你有 7♠7♦
翻牌：A♠7♥2♦
底池 $40，有效籌碼 $400

考慮過牌讓對手詐唬
或小注引誘加注
```

## 轉牌和河牌策略

### 底池控制

```
什麼是底池控制：

定義：
刻意保持底池較小
用中等強度的牌

何時使用：
- 頂對中等 kicker
- 超對在濕潤牌面
- 不確定是否領先

方法：
- 過牌一條街
- 使用小尺寸下注
- 不要過度建立底池
```

### 多條街計劃

```
深籌碼需要計劃：

翻牌前就要想：
- 中什麼牌打幾條街
- 籌碼如何分配
- 何時放棄

範例計劃：
你有 A♠K♦，100bb effective
翻牌：K♥7♠2♦

計劃 A（沒有危險牌）：
翻牌 bet 1/3 pot
轉牌 bet 1/2 pot
河牌 bet 2/3 pot

計劃 B（出現危險牌）：
翻牌 bet 1/3 pot
轉牌 check 控制
河牌 根據行動決定
```

### 詐唬頻率調整

```
深籌碼詐唬考量：

詐唬更有效：
- 對手要用更多籌碼跟注
- 更多籌碼 = 更多壓力
- 錯誤代價更大

但也更危險：
- 失敗損失更大
- 需要更好的故事
- 對手可能加注

建議：
- 選擇好的詐唬牌
- 有阻擋效應
- 有 equity 後盾
```

## 特殊場景

### 3-Bet Pots

```
3-bet 底池的深籌碼處理：

特點：
- 底池已經很大
- SPR 降低但仍深
- 需要謹慎行動

翻牌策略：
- 範圍 c-bet 頻率降低
- 更多 check 和控制
- 強牌可以開始建池

範例（你 3-bet 被 call）：
底池 $60，有效籌碼 $270
SPR ≈ 4.5

這已經接近淺籌碼
頂對好 kicker 可以較激進
```

### 多人底池

```
深籌碼多人底池：

更加謹慎：
- 更多人 = 更可能有人中牌
- 底池增長快
- 詐唬更難成功

策略調整：
- 減少 c-bet 頻率
- 只用強牌繼續
- 位置更加關鍵

建議：
不要在深籌碼多人底池詐唬
等待有牌的時候取值
```

### 面對強手

```
深籌碼對強手更困難：

原因：
- 技術差距被放大
- 更多決策點
- 更容易犯錯

調整：
1. 更緊的翻前範圍
2. 更簡單的翻後策略
3. 避免複雜的底池
4. 保護好籌碼

不要和強手玩大底池
除非你有堅果或接近堅果
```

## 常見錯誤

### 1. 過度建池

```
錯誤：
每手牌都想打大底池
用中等牌 4-bet

問題：
- 輸大底池給更強的牌
- 沒有利用深籌碼優勢
- 變異太大

正確：
選擇性建立大底池
用合適的牌在合適的時機
```

### 2. 忽略 SPR

```
錯誤：
不考慮 SPR 做決定
翻牌就承諾所有籌碼

問題：
- 頂對在高 SPR 時脆弱
- 容易被 set 或兩對擊敗
- 失去後續街資訊

正確：
根據 SPR 調整策略
高 SPR 時更謹慎
```

### 3. 投機牌打法錯誤

```
錯誤：
中了強牌就慢打到河牌
或者中了聽牌就過度激進

問題：
慢打太多：失去價值
過度激進：建立太大底池

正確：
平衡取值和控制
根據牌面和對手調整
```

### 4. 不調整範圍

```
錯誤：
用標準籌碼的範圍打深籌碼
不考慮手牌可玩性

問題：
- 玩太多容易被主導的牌
- 翻後經常面對困難決策

正確：
深籌碼時考慮：
- 同花性
- 連接性
- 被主導風險
```

## 總結

### 深籌碼核心原則

```
1. 投機牌價值提高
   小對子、同花連張更值得玩

2. 位置更重要
   沒位置時更謹慎

3. 注意 SPR
   高 SPR 時一對不夠強

4. 控制底池
   不要用中等牌建大底池

5. 多條街計劃
   翻前就要想好怎麼玩
```

### 深籌碼檢查清單

```
每手牌問自己：
□ 有效籌碼多深？
□ 這手牌適合深籌碼嗎？
□ 我的計劃是什麼？
□ SPR 是多少？
□ 我要建多大的底池？
```
