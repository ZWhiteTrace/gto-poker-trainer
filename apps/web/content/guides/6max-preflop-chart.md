# 6-Max 翻前範圍圖表完整指南

6-Max（六人桌）是線上撲克最流行的格式。本指南提供完整的翻前開池範圍圖表，幫助你掌握每個位置的正確策略。

## 為什麼 6-Max 範圍很重要？

在六人桌中，位置優勢更加明顯：
- 每個位置都需要行動更頻繁
- 範圍比滿桌（9-max）更寬
- 盲注每輪輪到你的次數更多

## 各位置 RFI（率先加注）範圍

### UTG（Under The Gun）
作為最早行動的位置，UTG 需要最緊的範圍：

**開池範圍（約 15%）：**
- 所有口袋對：22+
- 強 Ax：ATs+, AJo+
- 強 Kx：KQs, KJs
- 同花連牌：JTs+

### HJ（Hijack）
比 UTG 稍寬，但仍需謹慎：

**開池範圍（約 18%）：**
- 新增：A9s, KTs, QJs, T9s
- 可加入一些 Axs（A5s-A2s 有阻擋效果）

### CO（Cutoff）
位置優勢開始顯現：

**開池範圍（約 25%）：**
- 新增：A2s+, KJo, QJo
- 所有同花連牌：54s+
- 同花一間連牌：86s+

### BTN（Button）
最佳位置，範圍最寬：

**開池範圍（約 40-45%）：**
- 幾乎所有 Ax
- 所有 Kxs, 多數 Kxo
- 所有同花連牌
- 同花 Gapper

### SB（Small Blind）
特殊位置 - 建議採用極化策略：

**開池範圍（約 35-40%）：**
- 強牌 3-bet 或加注
- 弱牌直接棄牌
- 減少 limp 頻率

## 實用圖表速查

```
位置     開池%    關鍵手牌
-----------------------------------
UTG      15%     77+, ATs+, KQs
HJ       18%     66+, A9s+, KJs+
CO       25%     55+, A2s+, KTs+
BTN      45%     22+, Axs, K5s+
SB       38%     33+, A7s+, K9s+
```

## 常見錯誤

1. **UTG 開池太寬** - 不要用 A5o 或 KTo 在 UTG 開池
2. **BTN 太緊** - 很多玩家在按鈕位打得太緊，錯過獲利機會
3. **忽略籌碼深度** - 深籌碼可以玩更多投機牌

## 進階調整

當面對特定對手時，可以調整範圍：
- 面對緊凶型：可以更寬
- 面對鬆凶型：收緊範圍
- 面對魚玩家：開池更寬以獲得位置
