# MDF 詳解：最小防守頻率的完整理解

## 什麼是 MDF？

**MDF (Minimum Defense Frequency)** 是你面對下注時，需要繼續（跟注或加注）的最低頻率，以防止對手透過純詐唬獲利。

```
核心概念：
如果你棄牌太多 → 對手可以用任何牌詐唬獲利
如果你棄牌剛好 → 對手的純詐唬 EV = 0
```

---

## MDF 公式

### 基本公式

```
MDF = Pot / (Pot + Bet)

或等價地：

MDF = 1 - (Bet / (Pot + Bet))
```

### 計算範例

```
範例 1：
底池 $100，對手下注 $50
MDF = 100 / (100 + 50) = 100 / 150 = 67%

範例 2：
底池 $100，對手下注 $100
MDF = 100 / (100 + 100) = 100 / 200 = 50%

範例 3：
底池 $100，對手下注 $150
MDF = 100 / (100 + 150) = 100 / 250 = 40%
```

## 常見下注尺度的 MDF

| 下注尺度 | MDF | 計算 |
|----------|-----|------|
| 25% pot | 80% | 100/125 |
| 33% pot | 75% | 100/133 |
| 50% pot | 67% | 100/150 |
| 66% pot | 60% | 100/166 |
| 75% pot | 57% | 100/175 |
| 100% pot | 50% | 100/200 |
| 150% pot | 40% | 100/250 |
| 200% pot | 33% | 100/300 |

**快速記憶**：
- 半池下注 = 67% 防守
- 滿池下注 = 50% 防守
- 超池下注 = 防守更少

## MDF 的推導

### 為什麼是這個公式？

```
設對手純詐唬（0% 權益）：

如果你棄牌：
對手贏得底池 = Pot

如果你跟注：
對手輸掉下注 = -Bet

對手詐唬的 EV：
EV = (棄牌率 × Pot) - (跟注率 × Bet)

讓 EV = 0：
棄牌率 × Pot = 跟注率 × Bet
(1 - 跟注率) × Pot = 跟注率 × Bet
Pot - 跟注率 × Pot = 跟注率 × Bet
Pot = 跟注率 × (Pot + Bet)
跟注率 = Pot / (Pot + Bet) = MDF
```

## MDF 的實戰應用

### 作為防守者

```
使用 MDF 決定防守範圍：

1. 計算 MDF
2. 排序你的範圍（從強到弱）
3. 選擇前 MDF% 的牌繼續
4. 其餘棄牌

範例：
MDF = 60%
你的範圍有 100 種組合
→ 用前 60 種組合繼續
```

### 範圍排序

```
排序標準：

1. 對抗對手範圍的權益
2. 可玩性（後續街道的遊玩）
3. 阻擋效應
4. 有無後門權益

高優先繼續：
- 強成牌
- 強聽牌
- 有阻擋效應的 bluff catcher

低優先繼續：
- 弱聽牌
- 無阻擋的 bluff catcher
- 純空氣
```

## MDF 的限制

### 1. 假設對手純詐唬

```
MDF 假設：
對手下注 = 0% 權益的空氣

現實：
對手下注範圍包含取值和詐唬

如果對手只有取值：
你可以棄牌 100%

如果對手詐唬太多：
你應該繼續 > MDF
```

### 2. 不考慮後續行動

```
MDF 只考慮當前決策：

但實際上：
- 翻牌跟注 ≠ 最終投資
- 轉牌、河牌還有決策
- 隱含賠率和反向隱含賠率

因此：
翻牌/轉牌的 MDF 可以調整
河牌的 MDF 最準確
```

### 3. 不考慮位置

```
MDF 不區分位置：

但有位置時：
- 更容易實現權益
- 可以控制底池
- 信息優勢

沒位置時：
- 權益難以實現
- 更容易犯錯

調整：
有位置 → 可以繼續更多
沒位置 → 可以繼續較少
```

## Alpha 值：對手視角

### 什麼是 Alpha？

**Alpha** 是對手詐唬需要的成功率，使詐唬 EV = 0。

```
公式：
Alpha = Bet / (Pot + Bet)

注意：Alpha = 1 - MDF
```

### Alpha 的意義

```
範例：
底池 $100，對手下注 $100
Alpha = 100 / 200 = 50%

意思：
對手的詐唬需要 50% 成功率才能打平
如果你棄牌 > 50%，對手詐唬獲利
如果你棄牌 < 50%，對手詐唬虧損
```

### 尺度與 Alpha 的關係

| 下注尺度 | Alpha | 解讀 |
|----------|-------|------|
| 33% pot | 25% | 詐唬只需 25% 成功 |
| 50% pot | 33% | 詐唬需 33% 成功 |
| 100% pot | 50% | 詐唬需 50% 成功 |
| 200% pot | 67% | 詐唬需 67% 成功 |

**洞見**：
- 小注詐唬風險低
- 大注詐唬需要對手棄牌更多

## 實戰場景分析

### 場景 1：河牌面對大注

```
底池：$100
對手河牌下注：$75
MDF = 100 / 175 = 57%

你的範圍（假設 20 種組合）：
- 強牌（5 組合）：跟注/加注
- Bluff catchers（8 組合）：選擇性跟注
- 弱牌（7 組合）：棄牌

需要繼續：20 × 57% = 11-12 組合
→ 強牌 5 + 最好的 bluff catchers 6-7
```

### 場景 2：翻牌面對 C-bet

```
底池：$20
對手 C-bet：$7（35% pot）
MDF = 20 / 27 = 74%

但這是翻牌：
- 還有兩條街
- 不確定性高
- 可以適度調整

實戰調整：
- 繼續 65-70%（略低於 MDF）
- 用聽牌和中等成牌繼續
- 純空氣可以棄牌
```

### 場景 3：面對超池下注

```
底池：$100
對手下注：$200（200% pot）
MDF = 100 / 300 = 33%

分析：
- 只需繼續 33%
- 對手需要你棄牌 67% 才獲利
- 這是極化下注

應對：
- 用最強的 33% 繼續
- 中等牌可以棄牌
- 不需要過度防守
```

## MDF 與範圍構建

### 防守範圍構建

```
步驟：
1. 確定你的全部範圍
2. 計算 MDF
3. 選擇繼續的牌
4. 確保範圍平衡

範例：
你的 BB 防守範圍面對 BTN C-bet
範圍：30% 的手牌 ≈ 400 組合
MDF = 67%
繼續：約 270 組合

選擇：
- 所有對子和更強
- 好的聽牌
- 最好的高牌
```

### 不要過度防守

```
常見錯誤：
「我要防守 MDF 所以不能棄牌」

正確思維：
- MDF 是對抗純詐唬的
- 如果對手很少詐唬，棄牌更多
- 如果對手詐唬很多，繼續更多

實戰調整：
vs 緊玩家 → 繼續 < MDF
vs 激進玩家 → 繼續 ≥ MDF
```

## MDF 與詐唬頻率

### 對手詐唬比例

```
平衡的取值/詐唬比例：

公式：
取值 : 詐唬 = 防守率 : 棄牌率

範例（100% pot 下注）：
MDF = 50%
取值 : 詐唬 = 50 : 50 = 1 : 1

範例（50% pot 下注）：
MDF = 67%
取值 : 詐唬 = 67 : 33 ≈ 2 : 1
```

### 判斷對手是否過度詐唬

```
如果你認為對手詐唬 > 平衡比例：
→ 你應該繼續 > MDF
→ 抓詐唬有利可圖

如果你認為對手詐唬 < 平衡比例：
→ 你可以繼續 < MDF
→ 棄牌不會被剝削
```

## 常見錯誤

### 1. 機械應用 MDF

```
❌ 「MDF 是 60% 所以我必須繼續 60%」
✅ MDF 是基準，根據對手調整
```

### 2. 忽略範圍強度

```
❌ 不管什麼牌都防守 MDF
✅ 選擇最好的牌繼續
```

### 3. 翻牌過度防守

```
❌ 翻牌用 MDF 防守所有
✅ 翻牌可以適度收緊
```

### 4. 不考慮對手類型

```
❌ 對所有人用同樣的防守頻率
✅ vs 緊=棄多，vs 激進=繼續多
```

## 進階概念

### 多條街的 MDF

```
如果對手每條街都下注：

翻牌 MDF：67%
轉牌 MDF：67%
河牌 MDF：67%

三條街總防守率：
67% × 67% × 67% ≈ 30%

意思：
從翻牌到河牌，你只需要繼續 30% 的範圍
```

### 調整後的 MDF

```
考慮額外因素：

1. 對手範圍強度
   強 → 繼續更少
   弱 → 繼續更多

2. 你的範圍強度
   強 → 繼續更多
   弱 → 繼續更少

3. 位置
   有位置 → 繼續更多
   沒位置 → 繼續更少
```

## 總結

### MDF 核心原則

1. **MDF = Pot / (Pot + Bet)**
2. **防守 MDF 讓對手純詐唬 EV = 0**
3. **選擇最好的牌繼續**
4. **根據對手類型調整**
5. **河牌最適用，翻牌需調整**

### MDF 快速參考

```
半池下注 → 繼續 67%
滿池下注 → 繼續 50%
超池下注 → 繼續更少

vs 緊 → 繼續較少
vs 激進 → 繼續較多
```

### MDF 決策清單

```
□ 計算 MDF
□ 評估我的範圍有哪些牌
□ 排序牌（權益、可玩性、阻擋）
□ 選擇前 MDF% 的牌繼續
□ 根據對手類型調整
```
