# 面對 3-Bet 完全策略指南

## 什麼是 3-Bet？

在撲克中，3-Bet 是指在有人開池加注（open raise）後，你進行的再次加注。這是翻前最重要的對抗工具之一。

### 術語解釋

- **Open Raise（1st Raise）**：第一個加注
- **3-Bet（2nd Raise）**：對 open raise 的再加注
- **4-Bet（3rd Raise）**：對 3-bet 的再加注

---

## 為什麼學習面對 3-Bet 很重要？

當你開池後被 3-bet，這是 MTT 和現金桌中最常見的困境之一：

1. **頻率高**：好的對手會頻繁 3-bet 你的開池
2. **決策關鍵**：錯誤的回應會嚴重傷害你的勝率
3. **底池膨脹**：3-bet pot 比 SRP 大得多，錯誤代價更高
4. **範圍定義**：你的回應會定義你在翻後的範圍

---

## 面對 3-Bet 的四種選擇

### 1. 4-Bet（再加注）

**什麼時候 4-bet：**

| 牌型 | 目的 | 說明 |
|------|------|------|
| AA, KK | 價值 | 永遠 4-bet，建立大底池 |
| QQ, AKs | 價值 | 標準 4-bet，但要準備面對 5-bet |
| AQs, AJs | 混合 | 部分 4-bet bluff（有阻隔效應） |
| A5s, A4s | Bluff | 純 bluff 用（阻隔 AA/AK，有花潛力） |

**4-Bet 尺度建議：**
- 有位置：2.2-2.5x 3-bet 大小
- 無位置：2.5-3x 3-bet 大小

### 2. Call（平跟）

**什麼時候 Call：**

- **中等口袋對**：JJ, TT, 99（有位置時）
- **強 broadways**：AQo, KQs（有位置時）
- **同花連張**：適合中 set 的口袋對和有隱含賠率的牌

**Call 的關鍵考量：**
- 籌碼深度 > 40bb 才有足夠隱含賠率
- 有位置比無位置更容易 call
- 對手的 3-bet 頻率影響你的 call 範圍

### 3. Fold（棄牌）

**什麼時候 Fold：**

- **邊緣開池牌**：K9o, Q8o 等
- **無位置的弱牌**：大部分 suited connectors OOP
- **面對緊的 3-better**：收緊你的防守範圍

### 4. 5-Bet All-in

**什麼時候 5-bet：**
- 有效籌碼 < 100bb
- 對手 4-bet 後只剩 AA, KK, AKs

## 位置對策略的影響

### 有位置（IP）面對 3-Bet

當你在 BTN 或 CO 被盲注位置 3-bet：

**優勢：**
- 翻後有位置
- 可以看對手動作再決定
- bluff 和薄價值更容易

**建議範圍（vs BB 3-bet）：**
```
4-bet: AA, KK, QQ, AKs, A5s-A4s
Call: JJ-88, AQs-ATs, KQs-KJs, QJs, JTs
Fold: 其餘
```

### 無位置（OOP）面對 3-Bet

當你在早位開池被後位 3-bet：

**劣勢：**
- 翻後無位置
- 需要更強的牌才能繼續
- bluff 成本更高

**建議範圍（UTG vs BTN 3-bet）：**
```
4-bet: AA, KK, QQ, AKs
Call: JJ-TT, AQs
Fold: 大部分其他牌
```

## 根據對手調整

### vs 緊的 3-Better（3-bet < 6%）

- **收緊 call 範圍**：只用頂級牌防守
- **減少 4-bet bluff**：他們的範圍太強
- **更多 fold**：不要送錢給緊手

### vs 鬆的 3-Better（3-bet > 10%）

- **擴大 call 範圍**：可以用更多投機牌
- **增加 4-bet bluff**：利用他們的寬範圍
- **少 fold 好牌**：不要被過度剝削

### vs 極端緊（3-bet < 4%）

- **幾乎只 4-bet 頂級牌**：AA, KK
- **Call 很少**：只有 set mine 價值的對子
- **大量 fold**：尊重他們的範圍

## 常見 3-Bet Pot 場景

### 場景 1：100bb 深籌

**你：BTN 開池 2.5bb**
**BB：3-bet 到 10bb**

| 你的手牌 | 建議動作 | 說明 |
|---------|---------|------|
| AA | 4-bet 到 25bb | 建立價值 |
| KK | 4-bet 到 25bb | 建立價值 |
| QQ | 4-bet/Call 混合 | 根據對手傾向 |
| JJ | Call | 翻後打得好 |
| AKs | 4-bet 到 25bb | 價值 + 平衡 |
| AQs | Call | 太好不能 fold，太弱不能 4-bet |
| TT | Call | Set mine + 位置 |
| 99 | Call/Fold | 邊界，對手依賴 |
| A5s | 4-bet/Fold | Bluff 或放棄 |

### 場景 2：40bb 有效

短籌碼情況下決策簡化：

| 你的手牌 | 建議動作 | 說明 |
|---------|---------|------|
| AA, KK | 4-bet all-in | 籌碼不夠玩翻後 |
| QQ, AKs | 4-bet all-in | 翻前鎖定權益 |
| JJ, AQs | Call/Jam | 根據對手 |
| 其他 | Fold | 不值得 commit |

## 翻後策略（3-Bet Pot）

### 作為 Caller（被動方）

**翻後原則：**

1. **Check 到 3-better**：讓他們繼續代表範圍
2. **中 set 要慢打嗎？**：通常不，在 3-bet pot 底池大，直接 raise
3. **什麼時候 donk bet？**：幾乎不要，除非有特定策略

### 作為 3-Better

**翻後原則：**

1. **C-bet 頻率**：3-bet pot 中 c-bet 頻率應該較低（~50%）
2. **尺度**：標準 33% pot（因為 SPR 低）
3. **什麼牌 check**：中等 showdown value（如 JJ on A-high board）

- 「籌碼深度支持 call 嗎？」

## 常見錯誤

### 1. 過度 Fold

- **錯誤**：用 AQo 面對 BTN 3-bet 直接 fold
- **正確**：有位置時可以 call，甚至 4-bet

### 2. 不理解位置影響

- **錯誤**：OOP 和 IP 用同樣的範圍
- **正確**：無位置要收緊防守範圍

### 3. 忽略對手傾向

- **錯誤**：對所有人用同一套策略
- **正確**：vs Nit 多 fold，vs LAG 多 4-bet

### 4. Call 太多導致翻後困難

- **錯誤**：用 K9s 在 OOP call 3-bet
- **正確**：這類牌在 3-bet pot OOP 很難玩好

---

## 總結

面對 3-bet 是翻前最重要的決策之一：

1. **了解四種選擇**：4-bet、Call、Fold、5-bet
2. **位置是關鍵**：IP 可以防守更寬
3. **籌碼深度影響策略**：短籌碼簡化決策
4. **根據對手調整**：不要機械執行範圍
5. **大量練習**：讓正確決策變成直覺

正確的 3-bet 防守策略可以顯著降低你被剝削的程度，並在錦標賽和現金桌中提升你的長期勝率！
