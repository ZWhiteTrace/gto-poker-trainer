# 短籌碼策略：生存與反擊的藝術

## 什麼是短籌碼？

**短籌碼 (Short Stack)** 通常指 20BB 以下的籌碼量。

```
籌碼深度分類：

深籌碼：100BB+
標準籌碼：40-100BB
中等籌碼：20-40BB
短籌碼：10-20BB
超短籌碼：<10BB

短籌碼的特點：
- 翻後空間有限
- 很多決策變成全下或棄牌
- 位置更加重要
```

---

## 短籌碼的優勢與劣勢

### 優勢

```
1. 減少翻後複雜度
   - 決策更簡單
   - 減少技術差距

2. 增加 Fold Equity
   - 全下威脅大
   - 對手不願意跟

3. 變異較小
   - 每手牌風險有限
   - 更穩定的結果
```

### 劣勢

```
1. 無法實現隱含賠率
   - 沒有足夠籌碼贏大底池
   - 投機牌價值降低

2. 盲注壓力大
   - 每圈消耗比例高
   - 必須頻繁行動

3. 被大籌碼壓制
   - 他們可以無視你的全下
   - ICM 壓力（錦標賽）
```

## Push/Fold 策略

### 何時使用 Push/Fold

```
適用情況：

<10BB：幾乎只能 push/fold
10-15BB：主要 push/fold + 少量 open raise
15-20BB：混合策略

為什麼：
- 開池加注後籌碼已經投入太多
- 面對 3-bet 無法棄牌
- 不如直接全下增加 fold equity
```

### Push 範圍基礎

```
位置越靠後，範圍越寬：

UTG（10BB push）：
約 15% 範圍
77+, ATs+, AJo+, KQs

BTN（10BB push）：
約 40% 範圍
22+, A2s+, A7o+, K5s+, K9o+
Q8s+, QTo+, J8s+, JTo
T8s+, 98s, 87s, 76s

SB vs BB（10BB push）：
約 50-60% 範圍
幾乎任何 Ax, Kx, Qx
大部分同花連張
```

### 跟注範圍基礎

```
跟注範圍比 push 範圍窄：

原因：
- 沒有 fold equity
- 必須靠攤牌贏

BB vs SB 全下（10BB）：
約 35% 範圍
22+, A2s+, A5o+, K7s+, K9o+
Q9s+, QTo+, JTs

BB vs BTN 全下（10BB）：
約 25% 範圍
55+, A7s+, ATo+, KTs+, KQo
```

## 不同籌碼深度的策略

### 超短籌碼（<10BB）

```
策略：純 Push/Fold

範例（5BB 在 BTN）：
Push：約 60% 的牌
- 幾乎任何 Ax
- 任何 Kx 同花
- 任何 Qx 同花
- 連張如 54o+

不要：
- 開池 limp
- 小額加注
- 等待「更好的牌」
```

### 短籌碼（10-15BB）

```
策略：主要 Push/Fold + 少量 Open

何時 open raise（到 2-2.5x）：
- 前位有強牌
- 想控制底池
- 對手會 3-bet fold

何時直接 push：
- 後位
- 對手會 flat
- 籌碼接近 10BB

範例（12BB 在 CO）：
Push：A2s-A9s, 22-77, K9s, QTs
Raise/Fold：TT+, ATs+, AQo+
Raise/Call：JJ+, AKs, AKo
```

### 中等短籌碼（15-20BB）

```
策略：混合 Open + Push

一般原則：
- 強牌開池加注
- 中等牌可以 push
- 弱牌棄牌

3-bet 策略：
- 主要全下或棄牌
- 很少小 3-bet

範例（18BB 在 BTN）：
Open 到 2.5x：JJ+, AQs+, AKo
Push：22-TT, A2s-AJs, AJo-AQo
Fold：大部分其他牌
```

## 面對不同情況的調整

### 面對 Limpers

```
策略：隔離加注或全下

籌碼 10BB：
直接全下大部分可玩牌
範圍可以更寬（有死錢）

籌碼 15BB：
可以 raise 到 4-5x
或直接全下

關鍵：
不要跟著 limp
要麼全下，要麼棄牌
```

### 面對 Open Raise

```
作為短籌碼的選擇：

1. 全下（主要選擇）
   - 強牌為價值
   - 中等牌為 fold equity
   - 沒有小 3-bet

2. 跟注（很少）
   - 只有特定情況
   - 深籌碼才考慮

3. 棄牌
   - 大部分牌

範例（10BB）：
全下：55+, A7s+, A9o+, KQs
棄牌：其他
```

### 面對 3-bet

```
你開池被 3-bet：

15-20BB 時：
要麼全下，要麼棄牌
不要跟注

計算 pot odds：
開池 2.5x，對手 3-bet 到 8x
你需要跟 5.5x 進入 11.5x 底池
賠率約 48%

結論：
只有超強牌才繼續
大部分棄牌
```

## 錦標賽 vs 現金局

### 錦標賽短籌碼

```
額外考量：

1. ICM 壓力
   - 泡沫期更保守
   - 獎金跳躍影響決策

2. 盲注上升
   - 必須在籌碼消耗前行動
   - 不能等太久

3. 桌上短籌碼數量
   - 如果你不是最短，可以等
   - 如果你是最短，必須行動
```

### 現金局短籌碼

```
策略選擇：

1. Buy in short 策略
   - 故意買入 20-30BB
   - 減少變異
   - 簡化決策

2. 短籌碼後的處理
   - 可以隨時補碼
   - 或離開桌子
   - 不需要等待
```

## 心理因素

### 保持紀律

```
挑戰：
- 急於加倍
- 放寬範圍
- 情緒決策

應對：
1. 堅持計劃好的範圍
2. 不因為「等太久」而打弱牌
3. 接受有時候就是拿不到好牌
```

### 利用形象

```
建立激進形象：

優點：
- 對手更寬地跟你
- 價值牌得到更多支付

方法：
- 適當展示詐唬
- 保持頻繁攻擊
- 讓對手猜測
```

## 常見錯誤

### 1. 等太久

```
錯誤：
15BB 時還在等 AA
「我只想拿好牌」

問題：
- 盲注消耗籌碼
- 變成 8BB 時沒有 fold equity
- 最終被迫用垃圾牌全下

正確：
10-15BB 就要積極
選擇好位置行動
```

### 2. 開池太小

```
錯誤：
15BB 時開池到 2x
「我想控制底池」

問題：
- 給對手太好的賠率
- 面對 3-bet 進退兩難
- 浪費籌碼

正確：
要麼開池到 2.5x+
要麼直接全下
```

### 3. 翻後過度遊玩

```
錯誤：
12BB 開池，被跟注後
翻牌過牌-跟注-轉牌過牌-跟注

問題：
- 籌碼被消耗
- 沒有使用 fold equity
- 翻後技術沒有優勢

正確：
翻前全下或
翻後第一條街全下
```

### 4. 忽略位置

```
錯誤：
UTG 和 BTN 用同樣範圍 push

問題：
- UTG 面對更多對手
- 被跟注機率更高
- 需要更強的牌

正確：
前位收緊
後位放寬
```

## 總結

### 短籌碼核心原則

```
1. 簡化決策
   Push 或 Fold
   不要複雜化

2. 尊重位置
   後位更寬
   前位更緊

3. 不要等太久
   10-15BB 必須行動
   不要等到絕望

4. 利用 Fold Equity
   這是短籌碼的武器
   在還有時候使用

5. 調整範圍
   根據情況調整
   不是固定不變
```

### 短籌碼檢查清單

```
每手牌問自己：
□ 我的籌碼是多少 BB？
□ 我的位置在哪裡？
□ 這手牌在我的 push 範圍嗎？
□ 我有足夠的 fold equity 嗎？
□ 我應該行動還是等待？
```
