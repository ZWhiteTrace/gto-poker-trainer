# 底池賠率簡易教學：撲克數學入門

## 什麼是底池賠率？

**底池賠率 (Pot Odds)** 是你需要跟注的金額與你可能贏得的總金額之間的比率。

**公式**：
```
底池賠率 = 需要跟注的金額 / (當前底池 + 對手下注 + 你的跟注)
```

**轉換為百分比**：
```
需要的勝率 = 跟注金額 / 總底池
```

---

## 基本計算範例

### 範例 1：簡單計算

**情況**：
- 底池：$100
- 對手下注：$50
- 你需要跟注：$50

**計算**：
```
總底池 = $100 + $50 + $50 = $200
需要的勝率 = $50 / $200 = 25%
```

**結論**：如果你的勝率 > 25%，跟注是有利可圖的

## Outs 與勝率

### 什麼是 Outs？

**Outs** 是能讓你的牌變強的剩餘牌數量。

### 常見 Outs 數量

| 聽牌類型 | Outs | 範例 |
|----------|------|------|
| 同花聽牌 | 9 | 4 張同花，需要第 5 張 |
| 雙頭順子 | 8 | 5-6-7-8，需要 4 或 9 |
| 卡順 | 4 | 5-6-_-8，只能用 7 |
| 兩高牌 | 6 | AK 未中，需要 A 或 K |
| Set 聽牌 | 2 | 口袋對等第 3 張 |
| 同花+順子 | 15 | 同花順聽牌 |

---

## 2-4 法則：快速估算勝率

### 法則說明

- **轉牌後** (還有 1 張牌)：Outs × 2 = 勝率 %
- **翻牌後** (還有 2 張牌)：Outs × 4 = 勝率 %

### 範例

| Outs | 轉牌後勝率 (×2) | 翻牌後勝率 (×4) |
|------|-----------------|-----------------|
| 4 | 8% | 16% |
| 8 | 16% | 32% |
| 9 | 18% | 36% |
| 12 | 24% | 48% |
| 15 | 30% | 60% |

**注意**：此法則是估算，實際勝率略有偏差

## 實戰決策流程

### 步驟 1：計算底池賠率

```
對手下注 $30 進入 $60 底池
總底池 = $60 + $30 + $30 = $120
需要勝率 = $30 / $120 = 25%
```

### 步驟 2：計算你的 Outs

```
你有同花聽牌
Outs = 9
```

### 步驟 3：估算勝率

```
轉牌後：9 × 2 = 18%
翻牌後：9 × 4 = 36%
```

### 步驟 4：做出決定

```
情況 A：轉牌（1 張牌）
- 需要 25%，你有 18%
- 結論：單純底池賠率不足，考慮隱含賠率

情況 B：翻牌（2 張牌）
- 需要 25%，你有 36%
- 結論：跟注有利！
```

## 隱含賠率 (Implied Odds)

### 什麼是隱含賠率？

**隱含賠率**考慮的是：如果你擊中，還能從對手那裡贏得多少額外籌碼。

**公式**：
```
需要的勝率 = 跟注金額 / (當前底池 + 預期額外贏得)
```

### 範例

**情況**：
- 你需要跟注 $50
- 當前底池 $150
- 底池賠率需要 25% 勝率
- 你只有 18% 勝率（同花聽牌）

**但是**：
- 對手還有 $500 籌碼
- 如果你擊中，預期能贏 $200

**計算隱含賠率**：
```
實際需要勝率 = $50 / ($150 + $200) = 14.3%
你有 18%，跟注變成有利！
```

---

## 反向隱含賠率 (Reverse Implied Odds)

### 什麼是反向隱含賠率？

當你擊中時，可能仍然輸給更好的牌，這會降低你的期望值。

### 範例

**情況**：
- 你有 J♠ T♠ 在 Q♠ 9♠ 3♥ 牌面
- 你聽同花（9 outs）
- 如果擊中小同花...

**風險**：
- 對手可能有更大的同花聽牌 (A♠ x♠, K♠ x♠)
- 你擊中卻輸更多

**結論**：小同花聽牌的隱含賠率較低

## 常見錯誤

### 1. 忽略位置影響
❌ 只看數學，不考慮後續行動
✅ 有位置時隱含賠率更高

### 2. 高估隱含賠率
❌ 假設對手總會支付
✅ 對手可能識破並不支付

### 3. 忘記扣除 Outs
❌ 對手可能拿著你的 Outs
✅ 考慮阻擋效應

### 4. 忽略反向隱含賠率
❌ 小同花當堅果打
✅ 小同花可能被主導

## 練習題

### 題目 1
底池 $100，對手下注 $50，你有同花聽牌（9 outs），轉牌。
應該跟注嗎？

<details>
<summary>答案</summary>

需要勝率：$50 / $200 = 25%
你的勝率：9 × 2 = 18%
**單純底池賠率不足，但有足夠隱含賠率可以跟注**
</details>

### 題目 2
底池 $80，對手全下 $80，你有雙頭順子（8 outs），翻牌。
應該跟注嗎？

<details>
<summary>答案</summary>

需要勝率：$80 / $240 = 33%
你的勝率：8 × 4 = 32%
**接近臨界點，考慮額外因素（如後門同花）後可以跟注**
</details>

4. **反向隱含賠率降低價值**
