# 幾何下注尺度：數學化的下注規劃

## 什麼是幾何下注？

**Geometric Bet Sizing（幾何下注尺度）** 是一種數學化的下注規劃方法，讓你在多條街上用固定比例下注，最終正好全進。

```
核心概念：
每條街下注相同百分比的底池
→ 最後一條街正好全進

這樣的好處：
- 範圍平衡
- 最大化取值
- 減少決策困難
```

---

## 幾何下注公式

### 基本計算

```
設：
S = 有效籌碼
P = 初始底池
n = 剩餘街數
x = 每條街下注比例（相對於底池）

公式：
S/P = (1 + x)^n

解 x：
x = (S/P)^(1/n) - 1
```

### 計算範例

```
場景：
有效籌碼：$100
翻牌底池：$20
剩餘街數：3（翻牌、轉牌、河牌）

S/P = 100/20 = 5

x = 5^(1/3) - 1
x = 1.71 - 1
x ≈ 0.71 = 71%

答案：每條街下注 71% pot
三條街後正好全進
```

## 實戰應用

### 範例 1：標準取值線

```
場景：
你有 AA，BTN vs BB
翻牌底池：$10
有效籌碼：$50
SPR = 5

幾何計算：
x = 5^(1/3) - 1 ≈ 71%

下注計劃：
翻牌：$7（70% pot）
    → 底池變成 $24

轉牌：$17（71% pot）
    → 底池變成 $58

河牌：$26（全進，約 45% pot）
    → 對手面對滿池跟注

結果：完美規劃三條街取值
```

### 範例 2：調整後的幾何

```
場景：
你有 Set，CO vs BB
翻牌底池：$15
有效籌碼：$100
SPR ≈ 6.7

純幾何：約 88% 每街

但考慮：
- 翻牌對手範圍寬
- 轉牌對手會棄很多
- 河牌可以打更大

調整後：
翻牌：50%（讓對手跟注）
轉牌：75%
河牌：全進（超池）

策略：前面小保留，後面大收割
```

## 幾何下注的優勢

### 1. 範圍平衡

```
當你所有牌都用同一尺度：

取值牌：正常取值
詐唬牌：正常詐唬頻率

對手無法通過尺度讀牌

如果你：
強牌 → 大注
弱牌 → 小注

對手可以根據尺度調整
```

### 2. 最大化取值

```
數學證明：
幾何尺度最大化對手的跟注損失

為什麼？
- 每條街都給對手「合理」的賠率
- 對手無法輕易棄牌
- 累積下來付出最多
```

### 3. 簡化決策

```
傳統思考：
「翻牌下多少？轉牌下多少？河牌還剩多少？」

幾何思考：
「SPR 是 X，每街下 Y%」

決策變簡單，錯誤變少
```

## 何時偏離幾何尺度？

### 翻牌使用小注

```
情況：
- 乾燥牌面
- 你的範圍有很多空氣
- 想用高頻率 c-bet

策略：
翻牌 33%（小注高頻）
轉牌/河牌調整

為什麼可以：
- 乾燥牌面對手範圍窄
- 小注一樣有效
- 後續可以加大
```

### 河牌使用超池

```
情況：
- 堅果牌
- 對手範圍被壓縮
- 對手不太會棄牌

策略：
前面較小，河牌超池

範例：
翻牌：50%
轉牌：60%
河牌：200%（超池）

利用對手在河牌的跟注傾向
```

### 面對被動對手

```
情況：
- 對手很少加注
- 可以安全過牌

策略：
- 翻牌過牌
- 轉牌/河牌雙倍取值

這不是幾何，但可能更有效
```

## 詐唬的幾何規劃

### 多條街詐唬

```
如果你計劃三條街詐唬：

必須用幾何尺度！

原因：
- 代表強牌
- 每條街都給壓力
- 最後全進最有說服力

不能：
翻牌大注 → 轉牌小注 → 河牌大注
這樣的線路很奇怪
```

### 詐唬頻率計算

```
使用 MDF 反推：

每街 70% pot 下注
對手 MDF ≈ 59%

三條街：
59% × 59% × 59% ≈ 20%

意思：
對手只有最好的 20% 會跟到河牌
你的詐唬成功率約 80%

但你需要在河牌被抓時輸錢
所以需要適當的取值/詐唬比例
```

## 進階：非對稱幾何

### 遞增幾何

```
小 → 中 → 大

適用情況：
- 你的牌越來越強
- 對手範圍越來越弱
- 建立更大的河牌底池

範例：
翻牌：33%
轉牌：75%
河牌：150%
```

### 遞減幾何

```
大 → 中 → 小

適用情況：
- 需要快速保護
- 對手可能 slowplay
- 避免讓對手追牌

範例：
翻牌：100%
轉牌：50%
河牌：33%
```

## 幾何下注與牌面

### 濕潤牌面

```
牌面：9♠ 8♠ 7♣

特徵：
- 很多聽牌
- 牌力變化大

策略：
- 使用較大幾何尺度
- 快速建立底池
- 不給對手便宜追牌

建議：75-100% 每街
```

### 乾燥牌面

```
牌面：K♥ 7♣ 2♦

特徵：
- 聽牌少
- 牌力穩定

策略：
- 可以使用較小尺度
- 高頻率 c-bet
- 後續慢慢建立

建議：33-50% 翻牌，後面調整
```

## 常見錯誤

### 1. 翻牌太大，河牌太小

```
❌ 錯誤：
翻牌：100%
轉牌：75%
河牌：25%

問題：
- 河牌沒壓力
- 取值不完整
- 線路不一致
```

### 2. 不考慮籌碼深度

```
❌ 錯誤：
「我每條街都下 50% pot」
但 SPR = 15

問題：
50% × 3 條街不會全進
浪費了堅果的取值潛力
```

### 3. 詐唬尺度不一致

```
❌ 錯誤：
取值線：翻 75% → 轉 75% → 河全進
詐唬線：翻 50% → 轉 100% → 河 33%

問題：
對手可以通過尺度讀牌
```

## 實用計算技巧

### 快速估算

```
三條街，SPR 5-6：
→ 約 70-75% 每街

兩條街，SPR 3：
→ 約 70-75% 每街

一條街，SPR 1.5：
→ 約 150%（超池）
```

### 逆向計算

```
如果你知道河牌要全進多少：

河牌全進 = 100% pot
需要轉牌後 SPR = 1

往回推：
轉牌下多少才能達到 SPR 1？
翻牌下多少才能設置好轉牌？
```

## 總結

### 幾何下注核心原則

1. **數學化規劃多條街下注**
2. **每街相同比例，最終全進**
3. **範圍平衡，難以被讀**
4. **根據牌面調整基礎幾何**
5. **詐唬必須用一致的線路**

### 幾何下注決策清單

```
□ 計算當前 SPR
□ 確定剩餘街數
□ 計算幾何尺度
□ 考慮是否需要調整
□ 執行並保持一致
```
