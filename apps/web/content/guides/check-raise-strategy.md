# Check-Raise 策略指南：反擊的藝術

## 什麼是 Check-Raise？

**Check-Raise（過牌加注）** 是撲克中最強大的反擊武器。

```
定義：
1. 你先過牌 (check)
2. 對手下注
3. 你加注 (raise)
= Check-Raise

為什麼強大：
- 展現極大的力量
- 建立大底池
- 給對手施壓
- 可以是取值或詐唬
```

---

## Check-Raise 的目的

### 價值型 Check-Raise

```
目的：用強牌建立大底池

範例：
你在 BB 跟注 BTN 開牌
翻牌：9♠7♠2♥
你的牌：9♥9♦

分析：
- 你有暗三
- 對手會用很多牌 C-bet
- Check-raise 建立底池
- 對手可能認為你在詐唬
```

### 詐唬型 Check-Raise

```
目的：讓對手棄掉更好的牌

範例：
你在 BB 跟注 BTN 開牌
翻牌：K♠Q♥7♦
你的牌：T♠9♠

分析：
- 你有順子聽牌
- 對手會頻繁 C-bet
- Check-raise 可能讓 Ax, JJ 棄牌
- 有 equity 作為後盾
```

### 保護型 Check-Raise

```
目的：拒絕免費的 equity

範例：
你在 BB 跟注 BTN 開牌
翻牌：J♥8♠4♦
你的牌：J♣T♣

分析：
- 你有頂對中等 kicker
- 很多轉牌對你不利
- Check-raise 可以保護
- 讓聽牌付出代價
```

---

## 何時使用 Check-Raise

### 有利的牌面

```
好的 Check-Raise 牌面：

1. 低連接牌面
   - 8-7-3, 9-6-4
   - 防守者有更多暗三和兩對

2. 配對牌面
   - 7-7-2, J-J-5
   - 更容易有三條

3. 對手範圍優勢小的牌面
   - 不是 A 高或 K 高
   - 更平衡的對抗
```

```
不好的 Check-Raise 牌面：

1. A 高牌面
   - A-K-7, A-Q-5
   - 對手有太多 Ax

2. 極乾燥低牌面
   - 3-2-2
   - 很難有可信的強牌

3. 高度連接牌面
   - 對手可能有更多聽牌
   - 容易被跟注
```

### 對手類型考量

```
對愛 C-bet 的對手：
✓ Check-raise 更有效
✓ 他們用弱牌下注
✓ 增加詐唬 check-raise

對謹慎的對手：
✗ 他們只用強牌 C-bet
✗ Check-raise 要更偏價值
✗ 減少詐唬比例
```

## Check-Raise 尺寸

### 標準尺寸

```
基本公式：
Check-raise 到 3x 對手下注

範例：
底池 $20
對手 C-bet $10
你 check-raise 到 $30-$35
```

### 尺寸調整

```
使用更大尺寸（3.5-4x）：

- 濕潤牌面（拒絕免費牌）
- 詐唬時（增加 fold equity）
- 對愛跟注的對手

使用較小尺寸（2.5-3x）：

- 乾燥牌面
- 純取值時（保持對手範圍寬）
- 籌碼較淺時
```

## Check-Raise 範圍構建

### 翻牌圈 Check-Raise 範圍

```
價值端：
- 暗三（必須 check-raise）
- 兩對（通常 check-raise）
- 頂對好 kicker（部分）
- 超對（在低牌面）

詐唬端：
- 同花聽牌
- 兩頭順子聽牌
- 底對 + 後門聽牌
- 超牌 + 後門聽牌

平衡比例：
約 2:1 價值對詐唬
（因為後續街還能詐唬）
```

### 範例：BB vs BTN 在 8♠7♥4♦

```
Check-Raise 範圍：

價值牌（約 4-5%）：
44, 77, 88, 87s, 84s
T9s, 65s（強聽牌）

詐唬牌（約 3-4%）：
A5s, A6s（後門聽牌）
T9o, 65o（順子聽牌）
J9s, 96s（後門 + 對子聽牌）

總計：約 7-9% 範圍

其餘牌：
- 跟注或棄牌
- 取決於具體組合
```

## 不同街的 Check-Raise

### 翻牌圈 Check-Raise

```
特點：
- 最常見的 check-raise
- 建立底池或收取死錢
- 有後續街可以調整

目標：
- 價值：讓對手用差牌跟注
- 詐唬：讓對手棄牌
- 保護：拒絕免費牌
```

### 轉牌圈 Check-Raise

```
特點：
- 底池已經較大
- 代表極強的範圍
- 詐唬需要好的故事

適用場景：
- 翻牌過牌-跟注後
- 轉牌帶來好牌
- 對手繼續下注

範圍：
- 更偏向價值
- 詐唬要非常謹慎
```

### 河牌圈 Check-Raise

```
特點：
- 最強的行動
- 代表堅果或接近堅果
- 很少人會詐唬

適用場景：
- 你有超強牌
- 對手必然會下注
- 跟注無法最大化價值

注意：
- 需要好的讀牌
- 對手可能不下注
- 風險較高
```

## Check-Raise vs 其他打法

### Check-Raise vs 領先下注 (Donk Bet)

```
Check-Raise 優勢：
- 讓對手先投錢
- 有更多資訊
- 可以選擇不 raise

Donk Bet 適用場景：
- 牌面對你極有利
- 對手不會 C-bet
- 保護弱牌
```

### Check-Raise vs 過牌-跟注

```
Check-Raise 優勢：
- 建立更大底池
- 給對手更多壓力
- 保護手牌

過牌-跟注 優勢：
- 保持對手範圍寬
- 隱藏手牌強度
- 適合慢打
```

## 面對 Check-Raise 的應對

### 作為 C-bet 者

```
面對 check-raise 的選擇：

1. 棄牌
   - 大部分弱牌
   - 沒有聽牌的空氣

2. 跟注
   - 強聽牌
   - 中等成手牌
   - 準備看轉牌

3. 再加注 (3-bet)
   - 頂級價值牌
   - 偶爾詐唬（需要 blocker）

一般建議：
棄牌頻率應該較高（50-60%）
因為 check-raise 範圍通常很強
```

### 防守頻率

```
根據尺寸調整：

對手 check-raise 到 3x：
- 需要約 40% 繼續

對手 check-raise 到 4x：
- 需要約 35% 繼續

但要考慮：
- 對手的實際範圍
- 你的手牌 equity
- 位置和籌碼深度
```

## 實戰範例

### 範例 1：價值 Check-Raise

```
情況：
你在 BB，BTN 開牌你跟注
你的牌：7♥7♦
翻牌：A♠7♣3♦
底池 $20

BTN C-bet $10

分析：
- 你有暗三，極強
- BTN 會用很多 Ax C-bet
- Check-raise 建立底池

決策：
Check-raise 到 $32
準備在後續街繼續取值
```

### 範例 2：詐唬 Check-Raise

```
情況：
你在 BB，CO 開牌你跟注
你的牌：J♠T♠
翻牌：K♥8♠4♠
底池 $18

CO C-bet $9

分析：
- 你有同花聽牌
- 15 outs 如果算後門順
- Check-raise 可能讓 QQ, JJ 棄牌
- 被跟注也有 35%+ equity

決策：
Check-raise 到 $28
用好聽牌半詐唬
```

### 範例 3：不適合 Check-Raise

```
情況：
你在 BB，BTN 開牌你跟注
你的牌：A♥4♥
翻牌：A♠K♦Q♥
底池 $20

BTN C-bet $13

分析：
- 你有弱頂對
- 牌面極度對對手有利
- BTN 有很多 AK, AQ, KQ
- Check-raise 沒有好的對象

決策：
跟注或棄牌
不要 check-raise
```

## 常見錯誤

### 1. Check-Raise 太頻繁

```
錯誤：
每次有強牌都 check-raise
範圍變得過於透明

問題：
- 對手容易讀牌
- 過牌時被認定為弱
- 失去平衡

正確：
強牌要混合打法
有時跟注慢打
保持範圍不透明
```

### 2. 尺寸太小

```
錯誤：
底池 $20，對手下注 $10
你 check-raise 到 $20（最小加注）

問題：
- 給對手太好的賠率
- 無法保護手牌
- 詐唬沒有 fold equity

正確：
至少 raise 到 3x
通常 $30-$35
```

### 3. 詐唬牌選擇不當

```
錯誤：
用純空氣 check-raise
沒有 equity 沒有 blocker

問題：
- 被跟注時完全沒有退路
- 後續街很難繼續

正確：
選擇有 equity 的牌
聽牌或後門聽牌
有一些 blocker
```

### 4. 忽略對手類型

```
錯誤：
對不會 C-bet 的對手 check
期待他下注讓你 check-raise

問題：
- 對手過牌，你失去價值
- 免費給對手看牌

正確：
對被動對手領先下注
不要過度依賴 check-raise
```

## 進階概念

### 阻擋效應在 Check-Raise 中的應用

```
好的詐唬 blocker：

A♠x：
- 阻擋 AA, AK, AQ
- 減少對手頂級價值牌

K♠x 在 K 高牌面：
- 阻擋 KK, AK
- 減少對手可能的強牌

底對 + 聽牌：
- 阻擋對手的暗三
- 有 equity 作為後盾
```

### 頻率和平衡

```
GTO 指導原則：

翻牌圈 check-raise 頻率：
約 10-15% 的防守範圍

其中：
- 約 50-60% 價值
- 約 40-50% 詐唬

目標：
讓對手的 C-bet 不能自動獲利
但也不能讓對手隨意棄牌
```

## 總結

### Check-Raise 核心原則

```
1. 選擇好的牌面
   低連接牌面最適合

2. 正確的尺寸
   至少 3x 對手下注

3. 平衡範圍
   價值和詐唬都要有

4. 選擇好的詐唬牌
   有 equity 和 blocker

5. 考慮對手類型
   對愛 C-bet 的對手更有效
```

### Check-Raise 檢查清單

```
決定 check-raise 前：
□ 牌面適合 check-raise 嗎？
□ 對手會頻繁 C-bet 嗎？
□ 我的尺寸足夠大嗎？
□ 我的範圍平衡嗎？
□ 詐唬時有 equity 嗎？
□ 我準備好面對 3-bet 嗎？
```
