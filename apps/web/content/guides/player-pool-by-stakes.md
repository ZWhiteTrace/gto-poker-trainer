# 不同級別玩家池分析

從 NL2 到 NL500+，每個級別的玩家池都有獨特的特性和常見錯誤。理解這些差異可以幫助你在每個級別做出最優的剝削調整，並為升級做好準備。

## 級別概述

| 級別 | 盲注 | 買入 | 玩家特性 |
|------|------|------|----------|
| NL2 | $0.01/$0.02 | $2 | 純新手，基礎錯誤 |
| NL5 | $0.02/$0.05 | $5 | 休閒玩家為主 |
| NL10 | $0.05/$0.10 | $10 | 開始有認真玩家 |
| NL25 | $0.10/$0.25 | $25 | 半職業/嚴肅休閒 |
| NL50 | $0.25/$0.50 | $50 | 職業玩家出現 |
| NL100 | $0.50/$1.00 | $100 | 高競爭環境 |
| NL200+ | $1/$2+ | $200+ | 職業主導 |

## NL2-NL5：微級別

### 玩家池特徵
- **VPIP 極高**：經常看到 40-60%
- **很少 3-bet**：整體 3-bet 頻率 <4%
- **不會棄牌**：跟注站遍地
- **不理解位置**：各位置開池差異小
- **純粹娛樂**：不在乎輸贏

### 常見錯誤
1. Limp 進池（而非開池加注）
2. 用太寬的範圍跟注 3-bet
3. 不做 C-bet（或過度 C-bet）
4. 河牌 check 堅果
5. 沒有 fold equity 概念

### 剝削策略

**翻前：**
- 收緊開池範圍（他們會跟注）
- 增大開池尺寸（3-4BB 而非 2.5BB）
- 用更強範圍 3-bet（減少詐唬）

**翻後：**
- 停止詐唬（他們不會棄牌）
- 價值下注三條街
- 用更大的尺寸取值
- 當他們加注時相信他們有牌

### 升級指標
- 10BB/100 以上 win rate（持續 50k+ 手）
- 理解基本位置概念
- 能夠控制 tilt

## NL10：過渡級別

### 玩家池特徵
- 開始有研究過策略的玩家
- VPIP 稍微收緊（25-35%）
- 出現一些 3-bet
- 仍然很多跟注站
- 少數人用 HUD

### 常見錯誤
1. 過度 C-bet（不根據牌面調整）
2. 4-bet 範圍太緊
3. 轉牌放棄太快
4. 不會多條街取值
5. River 詐唬頻率太低

### 剝削策略

**翻前：**
- 可以開始加入 3-bet 詐唬
- 利用位置偷盲
- 但仍然減少對跟注站的詐唬

**翻後：**
- 對 C-bet 多 float
- 河牌詐唬開始有效
- 但價值下注仍然是主要盈利來源

### 升級指標
- 5-8BB/100 win rate
- 理解範圍概念
- 開始使用 HUD 數據

## NL25：分水嶺

### 玩家池特徵
- 明顯分成兩群：
  - 認真學習的玩家
  - 仍然很魚的休閒玩家
- 3-bet 頻率正常化（5-8%）
- 開始有聚合報告分析
- 一些玩家使用 solver

### 常見錯誤
1. 過度依賴 solver 輸出（不理解原理）
2. 盲注防守範圍不平衡
3. 多人底池打得太鬆
4. 對大注碼跟注太寬
5. 不會根據對手調整

### 剝削策略

**翻前：**
- 需要更平衡的 3-bet 範圍
- 開始利用 HUD 數據做針對性調整
- 對不同玩家類型使用不同策略

**翻後：**
- 需要理解牌面結構
- Check-raise 更有效
- 阻擋下注開始重要

### 升級指標
- 4-6BB/100 win rate
- 熟練使用 HUD
- 理解 solver 輸出

## NL50：職業入門

### 玩家池特徵
- 很多半職業玩家
- 魚的比例下降
- 大多數人有研究過策略
- 平均水平明顯提高
- 位置戰更激烈

### 常見錯誤
1. 過於 GTO 化（不夠剝削）
2. 盲注戰策略弱
3. 不會利用 population reads
4. 資金管理不當
5. 處理波動能力差

### 剝削策略

**關鍵轉變：**
- 更需要針對性策略
- Population exploits 很重要
- 需要建立自己的筆記系統
- 選桌變得關鍵

**具體調整：**
- 對 Reg 更 GTO
- 對 Fish 極端剝削
- 盲注戰需要專門研究

### 升級指標
- 3-5BB/100 win rate
- 完整的 solver 訓練
- 良好的心態管理

## NL100：高競爭區

### 玩家池特徵
- 職業玩家為主
- 很少的純魚
- 大家都有 HUD
- 大家都研究過 solver
- 邊際優勢變小

### 常見錯誤
1. 不夠的學習時間
2. 遊戲選擇差
3. 頻率不準確
4. 不夠針對性的筆記
5. 低估對手能力

### 成功要素

**技術：**
- 精確的頻率控制
- 多尺寸策略
- 完整的 solver 理解

**非技術：**
- 優秀的選桌能力
- 心態管理
- 持續學習

### 升級建議
- 2-4BB/100 win rate
- 加入學習群組
- 考慮教練

## NL200+：高級別

### 玩家池特徵
- 純職業環境
- 極少的軟玩家
- 大家互相熟悉
- 策略對話頻繁
- 很快適應新策略

### 生存法則

1. **持續進化**
   - 每週學習 10+ 小時
   - 追蹤最新策略趨勢
   - 加入高級學習群

2. **選桌至關重要**
   - 跟蹤魚的時間表
   - 多平台套利
   - 不要和強 Reg 硬剛

3. **心態管理**
   - 大的 Downswing 是正常的
   - 需要更大的資金緩衝
   - 專業的心理支持

## 升級策略

### 何時升級？
- 目前級別 win rate 穩定（50k+ 手樣本）
- 資金充足（30 買入以上新級別）
- 心態準備好接受更大波動
- 有時間投入學習

### 升級方法

**激進法：**
- 達到標準直接升
- 快速測試新級別
- 失敗就降回

**保守法：**
- 更大的樣本量
- 更多的資金
- 混合兩個級別打

### 升級失敗怎麼辦？
1. 不要 Tilt，這很正常
2. 分析是因為運氣還是水平
3. 降級回去不丟人
4. 專注於修補弱點
5. 準備好再來

## 不同平台的差異

### PokerStars
- 競爭最激烈
- 玩家整體水平高
- 但玩家數量大，可選桌

### GGPoker
- 亞洲玩家多
- 休閒玩家多
- 晚間時段較軟

### 其他平台
- 小平台通常較軟
- 但遊戲數量少
- 需要權衡

## 總結

**每個級別的核心：**

| 級別 | 關鍵技能 |
|------|----------|
| NL2-5 | 停止詐唬，瘋狂取值 |
| NL10 | 開始平衡，使用 HUD |
| NL25 | 針對性調整，理解 solver |
| NL50 | 選桌，心態管理 |
| NL100+ | 持續學習，專業化 |

**記住：**
- 每個級別都有可以剝削的對象
- 升級不是目標，賺錢才是
- 穩定比快速升級更重要
- 資金管理是生存的基礎
