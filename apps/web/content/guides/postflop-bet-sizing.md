# 翻後下注尺寸完全指南

## 為什麼下注尺寸重要？

**下注尺寸 (Bet Sizing)** 是撲克盈利的核心技術之一。

```
下注尺寸的影響：

正確的尺寸：
- 最大化價值
- 有效詐唬
- 保護手牌
- 平衡範圍

錯誤的尺寸：
- 價值牌取值不足
- 詐唬沒有 fold equity
- 讓對手正確跟注
- 範圍不平衡
```

---

## 下注尺寸基礎概念

### 底池比例

```
常見下注尺寸：

25% pot（1/4 底池）：
- 小型探測
- 多人底池
- 乾燥牌面

33% pot（1/3 底池）：
- 標準小注
- 範圍下注
- 控制底池

50% pot（半池）：
- 中等尺寸
- 平衡價值/詐唬
- 最常用

66-75% pot（2/3-3/4 池）：
- 較大尺寸
- 價值取向
- 保護手牌

100%+ pot（全池或超池）：
- 極化範圍
- 堅果或詐唬
- 最大壓力
```

### SPR 與下注尺寸

```
SPR 定義：
Stack to Pot Ratio = 有效籌碼 / 底池

SPR 影響尺寸選擇：

高 SPR（>10）：
- 使用較小尺寸
- 多條街取值
- 控制底池增長

中 SPR（4-10）：
- 標準尺寸
- 2-3 條街承諾

低 SPR（<4）：
- 可以更大尺寸
- 準備全下
- 簡化決策
```

## 翻牌圈下注尺寸

### C-Bet 尺寸

```
乾燥牌面（如 K-7-2 彩虹）：

小尺寸（25-33%）：
- 對手很少有牌
- 不需要大注保護
- 頻率可以更高

範例：
底池 $10
下注 $3（30%）
你的範圍很寬都可以這樣下

濕潤牌面（如 J-T-8 兩張同花）：

大尺寸（66-75%）：
- 很多聽牌需要被拒絕
- 對手有更多繼續的牌
- 需要收取更多價值

範例：
底池 $10
下注 $7（70%）
保護你的成手牌
```

### 不同牌面的尺寸

```
高牌面（A-K-x, K-Q-x）：

如果你是翻前加注者：
- 使用較小尺寸（25-40%）
- 你有範圍優勢
- 頻繁下注

如果你是跟注者：
- Check-raise 更多
- 下注更極化

連接牌面（9-8-7, T-9-8）：

使用較大尺寸（50-75%）：
- 很多聽牌
- 需要保護
- 拒絕免費牌

配對牌面（7-7-x, K-K-x）：

使用較小尺寸：
- 對手很少有牌
- 大注會趕走所有人
```

## 轉牌圈下注尺寸

### 繼續下注尺寸

```
翻牌下注被跟注後：

評估對手範圍：
- 他們跟了什麼牌？
- 轉牌對誰有利？
- 我應該繼續嗎？

尺寸選擇：

沒有變化的牌面：
- 類似翻牌尺寸
- 或稍大（50-66%）
- 繼續計劃

危險牌出現：
- 考慮過牌
- 或使用較小尺寸
- 控制損失

完成聽牌：
- 如果你完成：大注取值
- 如果對手可能完成：謹慎
```

### 轉牌特殊情況

```
轉牌全下考量：

何時考慮全下：
- SPR 變低（<3）
- 強牌需要保護
- 對手範圍適合

計算：
底池 $40
籌碼 $100
下注 $30 → 河牌 $100 in $100 pot
下注 $40 → 河牌 $60 in $120 pot
直接全下 → 最大壓力
```

## 河牌圈下注尺寸

### 價值下注尺寸

```
河牌價值下注原則：

對手會跟什麼？
- 分析跟注範圍
- 選擇最大化價值的尺寸

薄價值（頂對普通 kicker）：
- 較小尺寸（33-50%）
- 讓弱牌能跟注
- 不要趕走客戶

厚價值（堅果或接近堅果）：
- 較大尺寸（75-100%+）
- 對手有強牌會付錢
- 最大化價值
```

### 詐唬下注尺寸

```
河牌詐唬原則：

需要多少 fold equity？

50% pot 下注：
- 需要 33% 棄牌
- 較小風險
- 但 fold equity 有限

100% pot 下注：
- 需要 50% 棄牌
- 風險更大
- fold equity 更高

選擇依據：
- 對手的跟注傾向
- 你的阻擋效應
- 故事是否合理
```

### 極化下注

```
什麼是極化：

河牌範圍只有：
- 堅果級別
- 或完全沒牌

使用大尺寸：
- 100-150% pot
- 讓對手困難
- 贏大底池或收死錢

範例：
底池 $100
你有堅果同花或完全空氣
下注 $100-150
對手的中等牌很難跟注
```

## 面對不同對手的尺寸調整

### 面對跟注站

```
跟注站特徵：
- 很少棄牌
- 一路跟到河牌
- 需要看到攤牌

調整：

價值時：
- 使用更大尺寸
- 他們會付錢
- 不用擔心趕走

詐唬時：
- 不要詐唬
- 或使用極小尺寸
- 保留資本
```

### 面對緊手

```
緊手特徵：
- 經常棄牌
- 只用強牌繼續
- 對下注敏感

調整：

價值時：
- 使用較小尺寸
- 讓他們能跟注
- 不要趕走

詐唬時：
- 較小尺寸就足夠
- 他們容易棄牌
- 省錢達到目的
```

### 面對激進玩家

```
激進玩家特徵：
- 喜歡加注
- 會在任何時候攻擊
- 難以預測

調整：

誘導加注：
- 使用較小尺寸
- 讓他們加注
- 設置陷阱

避免被詐唬：
- 使用較大尺寸
- 減少他們的操作空間
- 讓他們付代價
```

## 不同場景的尺寸指南

### 單挑底池

```
單挑底池特點：
- 對手範圍明確
- 可以更頻繁下注
- 尺寸更靈活

翻牌：
- 常用 25-50%
- 乾燥牌面偏小
- 濕潤牌面偏大

轉牌：
- 常用 50-66%
- 根據牌面調整
- 考慮 SPR

河牌：
- 根據目的選擇
- 價值/詐唬分開考慮
```

### 多人底池

```
多人底池特點：
- 更可能有人中牌
- 需要更強的牌繼續
- 詐唬更難成功

尺寸調整：

翻牌：
- 使用較小尺寸（25-33%）
- 或直接過牌
- 不要給對手正確賠率

轉牌/河牌：
- 只用強牌下注
- 尺寸可以較大
- 取值導向
```

### 3-Bet 底池

```
3-bet 底池特點：
- 底池已經較大
- SPR 較低
- 決策更快

翻牌：
- 常用 25-33%
- 或直接過牌
- 範圍更緊

轉牌：
- 考慮全下 SPR
- 強牌直接全下
- 弱牌放棄

河牌：
- 通常已經全下
- 或小底池結束
```

## 常見下注尺寸錯誤

### 1. 尺寸透露手牌

```
錯誤：
強牌大注，弱牌小注
總是用同樣模式

問題：
- 對手容易讀牌
- 失去詐唬價值
- 被剝削

正確：
同一牌面用同樣尺寸
不管手牌強弱
保持平衡
```

### 2. 詐唬尺寸太小

```
錯誤：
詐唬時下 1/4 底池
「反正是詐唬，少輸一點」

問題：
- 沒有 fold equity
- 對手正確跟注
- 浪費籌碼

正確：
詐唬和價值用同樣尺寸
需要足夠的壓力
讓對手困難
```

### 3. 價值尺寸太大

```
錯誤：
有頂對就全池下注
「我想贏大底池」

問題：
- 趕走所有弱牌
- 只被更強的跟注
- 變成自己詐唬自己

正確：
根據對手跟注範圍
選擇能被弱牌跟注的尺寸
```

### 4. 忽略 SPR

```
錯誤：
不管 SPR 都用同樣尺寸
翻牌下半池造成尷尬 SPR

問題：
- 轉牌/河牌尺寸奇怪
- 無法有效全下
- 失去最大價值

正確：
計劃好每條街的尺寸
讓 SPR 有意義
全下時機自然
```

## 進階概念

### 幾何級數下注

```
概念：
計劃好讓河牌剛好全下

計算方法：
設定河牌全下
反推每條街尺寸

範例：
有效籌碼 $200
翻牌底池 $30

想河牌全下：
翻牌 bet $20 → pot $70
轉牌 bet $50 → pot $170
河牌 bet $130 (全下)

每條街約 66-75% pot
```

### 阻擋下注

```
什麼是阻擋下注：
用小注阻止對手下更大

適用場景：
- 你的牌不夠強打大底池
- 但又不想讓對手免費看
- 控制底池大小

尺寸：
- 通常 20-33% pot
- 足夠便宜
- 但能達到目的

注意：
- 可能被加注
- 只在特定情況使用
```

### 超池下注

```
什麼是超池：
下注超過底池大小

適用場景：
- 極化範圍
- 堅果或純詐唬
- 河牌最常見

效果：
- 最大化價值
- 最大化詐唬壓力
- 讓對手極度困難

風險：
- 損失更大
- 需要好的讀牌
- 範圍要夠極化
```

## 總結

### 下注尺寸核心原則

```
1. 有目的地下注
   每次下注都要知道為什麼
   價值、詐唬、保護

2. 根據牌面調整
   乾燥小注，濕潤大注
   連接牌面更大

3. 考慮對手類型
   跟注站大注取值
   緊手小注就夠

4. 保持平衡
   同樣牌面用同樣尺寸
   不要透露手牌

5. 計劃多條街
   想好河牌怎麼打
   反推翻牌尺寸
```

### 尺寸檢查清單

```
每次下注前：
□ 我的目的是什麼？
□ 這個尺寸能達到目的嗎？
□ 對手會怎麼反應？
□ 我的後續計劃是什麼？
□ 這個尺寸會透露我的牌嗎？
```
