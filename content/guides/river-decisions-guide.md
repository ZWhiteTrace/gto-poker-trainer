# 河牌決策指南：撲克的最終決策

## 河牌的特殊性

**河牌**是撲克中最純粹的決策點：

1. **沒有更多牌** - 權益固定，不再變化
2. **最大底池** - 通常是最大金額的決策
3. **純博弈** - 只有取值、詐唬、跟注、棄牌

```
河牌的核心問題：
- 作為進攻者：下注取值還是詐唬？
- 作為防守者：跟注抓詐唬還是棄牌？
```

---

## 河牌下注策略

### 取值下注

**目的**：讓比你差的牌跟注

```
決策問題：
「有哪些比我差的牌會跟注？」

如果答案是「很多」→ 下注取值
如果答案是「很少」→ 考慮過牌
```

#### 取值下注尺度

| 對手類型 | 建議尺度 | 原因 |
|----------|----------|------|
| 跟注站 | 大尺度 75-100% | 他會跟注 |
| 標準玩家 | 中尺度 50-66% | 平衡跟注率 |
| 緊玩家 | 小尺度 33-50% | 誘導跟注 |

#### 薄取值 (Thin Value)

```
定義：用只贏一小部分對手範圍的牌下注

範例：
牌面：K♠ 8♦ 5♣ 3♥ 2♠
你有：K♣ J♣

分析：
- 你贏 KT, K9, Kx 弱踢腳
- 你輸 KQ, K8, 88, 55
- 這是薄取值

決定因素：
- 對手會用什麼範圍跟注？
- 他會用 KT 跟注嗎？
```

**薄取值的風險**：
- 可能被加注困擾
- 只有更好的牌跟注 = 變成詐唬自己

---

### 河牌詐唬

**目的**：讓比你好的牌棄牌

```
詐唬條件：
1. 你的牌沒有攤牌價值
2. 對手範圍有足夠的棄牌
3. 你的故事線合理
4. 尺度足夠施壓
```

#### 好的詐唬牌

```
特點：
✓ 錯失的聽牌（已經沒有價值）
✓ 阻擋對手的跟注牌
✓ 不阻擋對手的棄牌
```

**範例**：
```
牌面：K♠ Q♦ 7♣ 4♥ 2♠
你有：A♣ J♣（錯失順子聽牌）

分析：
- AJ 沒有攤牌價值
- A 阻擋 AK, AQ
- 可以代表之前慢打的強牌
→ 好的詐唬候選
```

#### 河牌詐唬尺度

```
一般原則：
詐唬尺度 = 取值尺度（保持一致）

如果你取值用 66%，詐唬也用 66%

極化詐唬：
用更大尺度（75-150%）讓對手棄更多牌
但對手需要的勝率也更低
```

---

### 河牌過牌

**何時過牌？**

```
1. 攤牌價值足夠：
   - 你的牌贏詐唬但不值得取值
   - 過牌讓對手詐唬

2. 下注無利可圖：
   - 只有更好的牌跟注
   - 風險 > 收益

3. 控池：
   - 底池已經夠大
   - 不需要冒險
```

## 河牌防守策略

### 面對河牌下注

```
三個選擇：
1. 跟注（你認為能贏得夠多）
2. 棄牌（你認為落後太多）
3. 加注（你有超強牌，或詐唬）
```

### MDF（最小防守頻率）

```
計算公式：
MDF = Pot / (Pot + Bet)

範例：
底池 $100，對手下注 $80
MDF = 100 / 180 = 55.6%

意思：你應該跟注 55.6% 的時間
```

### Bluff Catcher 選擇

```
選擇跟注的牌：
✓ 能贏所有詐唬
✓ 阻擋對手取值牌
✓ 沒有阻擋對手詐唬牌

棄牌的牌：
✗ 輸給部分詐唬
✗ 阻擋對手詐唬
✗ 對手很少詐唬
```

## 河牌加注

### 取值加注

```
條件：
1. 你有非常強的牌
2. 對手可能用次級強牌跟注
3. 對手不太可能是詐唬

範例：
牌面：K♠ Q♦ Q♣ 7♥ 2♠
你有：Q♥ Q♦（四條！）
對手下注 $100 進 $150 底池

分析：
- 你有堅果
- 對手可能有 Kx, 葫蘆
- 加注取值
```

### 詐唬加注

```
條件：
1. 對手範圍被封頂（capped）
2. 你有好的阻擋效應
3. 對手會棄掉足夠的牌

範例：
牌面：8♣ 7♦ 5♠ K♥ 2♣
對手在河牌第一次下注（代表 one pair）
你有：A♠ 6♠（錯失順子）

分析：
- 6 阻擋 64（堅果順子）
- 對手可能只有 Kx, 88
- 加注代表強牌
```

**警告**：河牌詐唬加注風險極高，需要謹慎使用

## 河牌尺度策略

### 極化範圍下注

```
極化 = 你要嘛有堅果，要嘛在詐唬

大尺度下注（66%+）：
- 讓對手用中等牌做困難決定
- 最大化取值 + 詐唬效率
```

### 合併範圍下注

```
合併 = 你有各種牌力都下注

小尺度下注（25-40%）：
- 用更寬的範圍下注
- 對手難以加注
- 取薄值 + 誘導
```

### 超池下注 (Overbet)

```
何時超池：
1. 你的範圍有很多堅果
2. 對手範圍被封頂
3. 牌面適合極化

範例：
你 3-bet pot，對手跟注
牌面：A♠ K♦ Q♣ J♥ T♠
你有：A♣ A♦

分析：
- 牌面有 Broadway 順子
- 你有 Set，對手可能有 AK, AQ
- 對手不太可能有順子（你 3-bet）
- 超池取值！
```

## 常見河牌場景

### 場景 1：你先行動，有中等牌

```
你：A♠ T♠
牌面：A♦ 8♣ 5♥ 3♠ 2♦

分析：
- 頂對中等踢腳
- 下注取值：KT, QT, JT 會跟注嗎？
- 可能只有 Ax, 更好的對子會跟注

選項：
A) 下注 1/3 取薄值
B) 過牌控池

建議：過牌更安全，除非對手是跟注站
```

### 場景 2：面對大注，你有 bluff catcher

```
對手下注 $100 進 $100 底池
你有：Q♠ Q♥
牌面：K♦ 9♣ 5♥ 3♠ 2♦

分析：
- MDF = 50%
- QQ 輸給 Kx, 99, 55, 33, 22
- QQ 贏詐唬（JT, AT, 等）

評估：
- 對手有多少詐唬？
- 他會用 JT 詐唬滿池嗎？

決定：根據對手類型
- vs 激進對手：跟注
- vs 保守對手：可能棄牌
```

### 場景 3：錯失聽牌，考慮詐唬

```
你：J♣ T♣
牌面：A♠ K♦ 8♣ 4♣ 2♥

分析：
- 同花聽牌沒完成
- JT 沒有攤牌價值
- J 不阻擋 AJ（對手會棄的牌）
- T 阻擋 AT

選項：
A) 放棄（過牌-棄牌）
B) 詐唬（代表 Ax 或 KQ）

詐唬考量：
- 你的故事線合理嗎？
- 對手會棄 Ax 嗎？
- 尺度多少能讓他棄牌？
```

### 場景 4：強牌，選擇尺度

```
你：K♠ K♦
牌面：K♥ 7♦ 4♣ 3♠ 2♥

分析：
- 你有 Set
- 對手可能有 Ax, 77, 44, 口袋對

選項：
A) 小注 33%：讓更多牌跟注
B) 中注 66%：標準取值
C) 大注 100%：極化，打對手 77, 44

決定因素：
- 對手類型（跟注傾向）
- 籌碼深度
- 之前的行動線
```

## 河牌讀牌

### 故事線一致性

```
你的行動需要講一個合理的故事：

一致的故事線：
翻牌：你下注（代表有牌）
轉牌：你下注（繼續代表）
河牌：你下注（合理結論）

不一致的故事線：
翻牌：你過牌
轉牌：你過牌
河牌：突然大注 ← 這是什麼牌？
```

### 對手故事線分析

```
對手的行動告訴你什麼：

翻牌：對手過牌-跟注
轉牌：對手過牌-跟注
河牌：對手下注

問：他有什麼？
- 可能慢打強牌
- 可能終於決定詐唬
- 不太可能是中等牌（之前會下注）
```

## 對手類型調整

### vs 跟注站
```
調整：
- 增加取值下注
- 減少詐唬
- 用更大尺度取值
```

### vs 激進玩家
```
調整：
- 更多過牌-跟注
- 設陷阱
- 準備好抓詐唬
```

### vs 緊玩家
```
調整：
- 更多詐唬（他們會棄牌）
- 薄取值要謹慎
- 面對他們的下注要尊重
```

## 常見錯誤

### 1. 忽略 MDF
```
❌ 每次面對下注都棄牌
✅ 用正確比例防守
```

### 2. 超級取值
```
❌ 用 A-high 在河牌下大注
✅ 評估「什麼牌會跟注？」
```

### 3. 故事線不一致
```
❌ 前面被動，河牌突然激進
✅ 保持一致的行動線
```

### 4. 不考慮尺度訊號
```
❌ 忽略對手的下注尺度
✅ 小注和大注代表不同範圍
```

## 總結

### 河牌核心原則

1. **取值下注**：問「什麼牌會跟注？」
2. **詐唬**：問「什麼牌會棄牌？」
3. **防守**：使用 MDF 計算跟注頻率
4. **尺度**：極化用大注，合併用小注
5. **故事線**：保持一致，讀對手故事

### 河牌決策清單

```
作為進攻者：
□ 我的牌有取值嗎？
□ 對手會用什麼跟注？
□ 我應該詐唬嗎？
□ 正確的尺度是什麼？

作為防守者：
□ MDF 是多少？
□ 我的牌是好的 bluff catcher 嗎？
□ 對手的詐唬頻率？
□ 我有阻擋效應嗎？
```
