# 極化範圍概念：理解下注的兩極結構

## 什麼是極化範圍？

**極化範圍 (Polarized Range)** 是只包含最強牌和最弱牌的範圍，沒有中間牌力。

```
極化範圍結構：

[堅果牌] ←──────────────────→ [詐唬牌]
   ↑                              ↑
  取值                          空氣

中間牌不在極化範圍中
```

---

## 極化 vs 合併範圍

### 極化範圍 (Polarized)

```
特徵：
- 只有很強或很弱
- 沒有中等牌力
- 通常配合大尺度下注

範例：
下注範圍 = {AA, KK, Sets} + {錯失聽牌, 空氣}
過牌範圍 = {中對, 頂對弱踢腳, 弱成牌}
```

### 合併範圍 (Merged / Linear)

```
特徵：
- 包含各種牌力
- 連續分佈
- 通常配合小尺度下注

範例：
下注範圍 = {AA 到 中對} + {部分空氣}
過牌範圍 = {弱成牌, 放棄的牌}
```

### 視覺化比較

```
牌力分佈（從強到弱）：
|──────|──────|──────|──────|──────|
堅果    強牌    中等    弱牌    空氣

極化下注：
|██████|      |      |      |██████|
 取值                         詐唬

合併下注：
|██████|██████|██████|      |██    |
 取值範圍                     詐唬
```

## 何時使用極化範圍？

### 1. 河牌決策

```
河牌是最常極化的街道：

原因：
- 沒有更多牌
- 聽牌要嘛擊中要嘛錯失
- 中等牌的價值已定

極化策略：
- 強牌大注取值
- 錯失聽牌詐唬
- 中等牌過牌
```

### 2. 對手範圍被封頂

```
當對手範圍明顯較弱時：

範例：
你 3-bet，對手只跟注
牌面：A-K-7

分析：
- 對手沒有 AA, KK（會 4-bet）
- 對手範圍封頂於 AQ, AJ 等
- 你可以用極化範圍下注

策略：
- 用 AK, Set 大注取值
- 用空氣大注詐唬
- 對手中等牌很難應對
```

### 3. 大尺度下注

```
大尺度自然要求極化：

原因：
- 大注需要足夠強的牌承受
- 中等牌下大注風險太高
- 詐唬需要尺度足夠讓對手棄牌

範例：
100% pot 下注範圍：
取值：頂 set, 同花, 順子
詐唬：完全錯失的聽牌
不下注：頂對, 第二對子
```

## 何時使用合併範圍？

### 1. 翻牌 C-bet

```
翻牌通常適合合併下注：

原因：
- 牌力未定
- 很多牌有發展空間
- 小注可以用寬範圍

範例（乾燥牌面 A-7-2）：
下注 1/3 pot：
- AK, AQ（取值）
- A8, A6（薄值）
- KQ, QJ（詐唬）
- 幾乎整個範圍都下注
```

### 2. 範圍優勢明顯

```
當你範圍明顯更強時：

範例：
你是 BTN 開池者，BB 跟注
牌面：K-K-7

分析：
- 你有更多 Kx
- 對手範圍較弱
- 可以用寬範圍小注

合併下注：
- Kx（取值）
- 所有對子（保護）
- A-high（阻止對手免費看牌）
```

### 3. 小尺度下注

```
小尺度自然適合合併：

原因：
- 風險小
- 對手難以加注
- 可以用更多牌下注

範例：
25% pot 下注：
- 可能包含 70%+ 的範圍
- 取值和詐唬界限模糊
- 主要是範圍優勢壓制
```

## 極化的數學基礎

### 取值/詐唬比例

```
平衡的極化範圍：

公式：
取值 : 詐唬 = 對手跟注率 : 棄牌率

根據下注尺度：
50% pot → MDF 67% → 取值:詐唬 = 2:1
100% pot → MDF 50% → 取值:詐唬 = 1:1
200% pot → MDF 33% → 取值:詐唬 = 1:2

意思：
大尺度可以有更多詐唬
小尺度需要更多取值
```

### 範例計算

```
河牌你下注 100% pot：

MDF = 50%
對手應該跟注 50% 範圍

你的範圍平衡：
如果取值 10 組合
詐唬應該 10 組合
總計 20 組合

這樣對手跟注/棄牌 EV 相同
```

## 極化下注的尺度選擇

### 尺度與牌力的關係

| 下注尺度 | 適合的範圍類型 |
|----------|----------------|
| 25-33% | 合併（寬範圍）|
| 50-66% | 輕度極化 |
| 75-100% | 中度極化 |
| 150%+ | 高度極化（堅果或空氣）|

### 選擇尺度的考量

```
選大尺度的情況：
✓ 你的範圍有很多堅果
✓ 對手範圍被封頂
✓ 牌面濕潤需要保護
✓ 對手傾向跟注太多

選小尺度的情況：
✓ 你有明顯範圍優勢
✓ 想用寬範圍下注
✓ 不確定牌力排名
✓ 對手傾向棄牌太多
```

## 實戰場景分析

### 場景 1：河牌極化下注

```
你：A♠ J♠
牌面：A♦ K♣ 7♥ 4♠ 2♦

你在 BTN，對手在 BB
之前行動：你 C-bet 翻牌，對手跟注
轉牌雙方過牌
河牌你先行動

分析：
- AJ 是頂對中等踢腳
- 下注會被 AK, AQ 跟注
- 下注可能被 AT, A9 跟注

這是中等牌：
→ 不適合極化下注
→ 過牌或小注取薄值
```

### 場景 2：河牌極化詐唬

```
你：Q♠ J♠
牌面：K♦ T♣ 7♥ 3♠ 2♦
（你的順子聽牌沒完成）

分析：
- QJ 沒有攤牌價值
- 這是「空氣」端
- 適合放入極化詐唬範圍

詐唬決策：
- 你下注代表 AK, KT, K7
- 用大尺度（75-100%）
- 對手中等牌會很難應對
```

### 場景 3：合併 C-bet

```
你：K♠ Q♣ 在 CO
牌面：K♦ 8♣ 3♥（對手 BB 跟注）

分析：
- 你有頂對好踢腳
- 這是乾燥牌面
- 你有範圍優勢

合併策略：
- 下注 1/3 pot
- 用整個範圍下注
- KQ 在下注範圍中間
```

## 極化範圍的防守

### 面對極化下注

```
防守策略：

1. 認識對手範圍是極化的
   - 大尺度下注
   - 河牌激進行動

2. 你的牌變成 bluff catcher
   - 不用擔心中等牌
   - 只需要贏詐唬

3. 使用 MDF 決定跟注頻率
   - 不要過度棄牌
   - 不要過度跟注
```

### 面對合併下注

```
防守策略：

1. 認識對手範圍包含各種牌力
   - 小尺度下注
   - 翻牌/轉牌行動

2. 你的中等牌可能領先
   - 考慮加注取值
   - 或跟注看下一街

3. 可以更靈活
   - 加注頻率可以更高
   - 因為對手範圍較弱
```

## 常見錯誤

### 1. 中等牌極化下注

```
❌ 用頂對中踢腳下注 100% pot
✅ 中等牌應該小注或過牌
```

### 2. 極化範圍只有取值

```
❌ 大注只用強牌，不詐唬
✅ 極化需要平衡的取值/詐唬比例
```

### 3. 合併範圍用大尺度

```
❌ 用整個範圍下注 75% pot
✅ 合併下注用小尺度
```

### 4. 不識別對手的極化

```
❌ 面對大注還擔心對手中等牌
✅ 認識大注 = 極化，你是 bluff catcher
```

## 進階概念

### 半極化範圍

```
介於極化和合併之間：

結構：
[堅果] + [中上等牌] + [少量詐唬]

使用場景：
- 轉牌決策
- 中等尺度下注
- 不確定牌力排名時
```

### 動態極化

```
範圍會隨著街道變化：

翻牌：通常合併
轉牌：開始極化
河牌：高度極化

原因：
牌力越來越確定
中等牌逐漸分化
```

## 總結

### 極化範圍核心

1. **極化 = 堅果 + 空氣**
2. **合併 = 連續分佈的牌力**
3. **河牌最適合極化**
4. **大尺度 = 極化，小尺度 = 合併**
5. **平衡取值/詐唬比例**

### 快速判斷

```
使用極化：
□ 河牌決策
□ 對手範圍封頂
□ 想用大尺度
□ 有明確取值和詐唬

使用合併：
□ 翻牌 C-bet
□ 範圍優勢明顯
□ 想用小尺度
□ 牌力不確定
```
