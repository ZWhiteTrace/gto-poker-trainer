# 對抗 Nit 玩家完整指南：偷瞎的藝術

## 什麼是 Nit？

**Nit（岩石/緊玩家）** 是撲克中最保守的玩家類型。他們的核心特徵是：**極度緊，只玩優質牌**。

```
Nit 的核心特徵：

✓ 只玩最強的起手牌
✓ 翻前棄牌率極高
✓ 面對侵略容易棄牌
✓ 下注時通常有貨
✓ 幾乎不詐唬

結果：
→ 範圍極強但透明
→ 偷盲容易
→ 他下注時要小心
→ 直接利潤有限
```

---

## 識別 Nit 玩家

### 數據特徵 (HUD)

```
典型 Nit 數據：

VPIP：8-14%
PFR：6-12%
VPIP-PFR 差距：<4%

3-bet%：3-5%（只用超強牌）
Fold to 3-bet：>70%

翻後數據：
C-bet Flop：65-80%（有牌才進池）
Fold to C-bet：>50%
WTSD：20-24%（很少攤牌）
W$SD：>54%（攤牌勝率高）
AF：2.0-3.0

關鍵指標：
VPIP < 15% + 高 Fold to 3-bet = Nit
```

### 行為特徵（無 HUD 時）

```
翻前信號：
□ 連續多圈棄牌
□ 只在好位置開池
□ 幾乎不冷跟
□ 看起來很無聊（因為一直棄牌）
□ 偶爾加注 = 幾乎一定是 premium

翻後信號：
□ C-bet 後如果被加注通常棄牌
□ 很少過牌-加注詐唬
□ 轉牌不繼續 = 真的放棄了
□ 河牌大注 = 堅果

心理特徵：
□ 害怕輸錢
□ 追求「安全」的打法
□ 寧可小贏也不願冒險
□ 可能是新手或 bankroll 緊張
```

### Nit 的亞型

```
膽小 Nit：
- 害怕輸錢
- 有強牌也不敢下大
- 面對侵略就投降
- 最容易剝削

老派 Nit：
- 認為「耐心」是美德
- 等 AA/KK 才玩
- 可能有多年經驗但策略過時
- 下注時非常危險

破產線 Nit：
- Bankroll 不夠
- 被迫打得緊
- 可能在等重建信心
- 對待同上

選擇性 Nit：
- 只在某些情況緊
- 可能因為桌上對手強
- 需要區分是永久風格還是暫時調整
```

## 核心策略：對 Nit 的調整

### 黃金法則

```
對 Nit 的三個核心原則：

1. 瘋狂偷盲
   他太常棄牌 = 你的偷盲印鈔機

2. 尊重他的侵略
   他加注/下注 = 通常有真貨

3. 不要試圖詐唬河牌
   他到河牌還在 = 範圍非常強
```

---

## 翻前策略

### 偷盲

```
Nit 在盲位時：

這是你的金礦！

標準偷盲成功率需要約 50%
Nit 棄牌率通常 > 80%

擴大開池範圍：
BTN vs Nit 盲位：開池 50%+ 的牌
CO vs Nit 盲位：開池 35%+ 的牌

適合偷盲的牌：
- 任何兩張百老匯（KT+, QT+, JT）
- 任何 Ax
- 同花連牌（65s+）
- 任何口袋對
- 甚至更弱的牌（如 J7s, Q5s）

尺度：
可以稍微減小（2-2.2BB）
反正他會棄牌，為什麼要冒更多風險？

但注意：
如果 Nit 3-bet = 縮起來
他 3-bet 的範圍是 QQ+, AK
```

### 面對 Nit 的開池

```
Nit 開池的含義：

早位開池 = QQ+, AK（超強！）
中位開池 = TT+, AQs+
後位開池 = 99+, AJs+, KQs

應對策略：

3-bet：
- 只用真正的價值牌（QQ+, AK）
- 偶爾用 A5s/A4s 平衡（他會棄掉很多）
- 不要 3-bet 詐唬太寬

跟注：
- 用投機牌（小對、同花連牌）
- 目標是中暗三或同花
- 沒中就放棄

棄牌：
- 很多通常可以玩的牌可以棄掉
- KJo、QTs 等在盲位可以棄掉
- 他的範圍太強
```

### 面對 Nit 的 3-bet

```
Nit 3-bet = 紅色警報！

他的 3-bet 範圍：
QQ+, AK（可能只有這些）

你的應對：

用 KK+ 4-bet 取值
用 AA 考慮平跟陷阱（他的範圍這麼窄）
其他都可以考慮棄牌

為什麼 JJ 也可以棄牌：
- Nit 3-bet 範圍：QQ+, AK
- JJ vs 這個範圍大約 36% equity
- 加上他翻後會打得好 = 可以棄
```

## 翻後策略

### 當你是進攻者

```
C-bet vs Nit：

頻率：可以提高（65-75%）
他翻前範圍緊 = 翻牌更常沒中

尺度：可以減小（40-50% pot）
反正他沒中會棄，有中不會棄

什麼時候停止：
□ 他跟注你的 C-bet = 他有東西
□ 他加注你 = 絕對有東西
□ 轉牌繼續要謹慎

Nit 跟注 C-bet 後的範圍：
- 頂對+
- 強聽牌
- 偶爾中對（但會在轉牌放棄）
```

### 當 Nit 開池你跟注

```
翻牌 C-bet：

Nit C-bet = 通常有牌
但 C-bet 範圍還是比較寬
可以適當浮牌

跟注的考量：
- 有位置
- 有後門聽牌
- Nit 轉牌可能放棄

Nit 不 C-bet 時：
= 他完全沒中
= 你可以下注拿下

轉牌雙桶：
Nit 轉牌繼續 = 範圍很強
除非你有強牌/強聽牌，否則放棄
```

### 多條街的策略

```
Nit 的街道分析：

翻牌：
- 會 C-bet 大部分進池範圍
- 跟注範圍還算寬

轉牌：
- 關鍵決策點
- 繼續 = 真的有牌
- 放棄 = 就是放棄

河牌：
- 到這裡還在 = 範圍極強
- 他的下注 = 幾乎都是價值
- 詐唬 Nit 的河牌幾乎沒用

結論：
對 Nit 詐唬的最佳時機是翻牌
轉牌他還在就要小心
河牌基本放棄詐唬
```

## 特殊情況處理

### 當 Nit 加注

```
Nit 加注 = 堅果警報！

翻牌加注：
- 至少兩對
- 通常是暗三+
- 偶爾超強聽牌（同花+順子組合）

轉牌加注：
- 兩對+
- 幾乎沒有詐唬

河牌加注：
- 100% 堅果
- 不要懷疑
- 不要英雄跟注

應對：
除非你有接近堅果的牌
否則棄牌是正確的
```

### 當 Nit 下注

```
Nit 主動下注的含義：

他們很少主動下注
當他們下注時：

小注（25-40% pot）：
- 可能是試探
- 可能是弱牌想便宜攤牌
- 可以考慮加注測試

標準注（50-70% pot）：
- 有價值的牌
- 在取值

大注（>75% pot）：
- 強牌
- 想拿大底池

超池（>100% pot）：
- 堅果
- 期望你有第二堅果
```

### 當 Nit 過牌-跟注

```
Nit 過牌-跟注的範圍：

這是有趣的情況
Nit 不喜歡過牌-跟注

如果他過牌-跟注：
- 可能是中等牌力控池
- 可能是在設陷阱
- 可能是聽牌

如何判斷：
- 牌面是否有聽牌可能？
- 他的翻前行動暗示什麼範圍？
- 他過牌的頻率如何？

應對：
- 如果牌面濕，他可能在抽牌
- 如果牌面乾，他可能在設套
- 謹慎繼續
```

### Nit 突然激進

```
注意異常行為：

如果一直緊的 Nit 突然：

1. 開始打更多牌
   - 可能要離開了想撈一把
   - 可能傾斜了
   - 可能發現桌上都是魚想參與

2. 開始詐唬
   - 可能學了新策略
   - 可能意識到被偷太多盲
   - 要重新評估

3. 大注突然變多
   - 還是要尊重
   - Nit 改變打法≠變成詐唬狂
   - 他的大注仍然代表強牌
```

## 利潤來源分析

### 直接利潤

```
對 Nit 的直接利潤：

盲注偷取：
- 每次偷盲成功 +1.5BB
- 假設每圈你嘗試偷 2 次
- 成功率 75%
- 每圈淨賺約 2.25BB

這就是主要收入！

從價值下注：
- 很難
- 他不會用弱牌跟注
- 除非你中了堅果對上他的強牌
```

### 間接利潤

```
對 Nit 的間接利潤：

1. 桌上形象
   當你對 Nit 擴大範圍
   其他人會注意到
   可以利用這個形象

2. 位置控制
   Nit 不會 squeeze
   讓你可以更多跟注前面的加注

3. 底池減少
   Nit 讓底池變小
   減少你和他的 variance
```

## 常見錯誤

### 錯誤 1：和 Nit 打大底池

```
❌ 「我有 KK，他 3-bet 我 4-bet」
❌ 「我 AA 想堆籌碼」
❌ 「不信他每次都有 AA」

事實：
Nit 願意打大底池 = 他有堅果級別
KK vs Nit 的 3-bet 範圍通常是 QQ+, AK
KK 的 equity 只有約 55%

✓ 正確心態：
Nit 給你大底池 = 謹慎
他的籌碼進池 = 他有強牌
用 KK 跟注 3-bet 看翻牌可能更好
```

### 錯誤 2：詐唬河牌

```
❌ 「我代表 A 高牌面的 A」
❌ 「這麼多桶他應該棄牌」
❌ 「牌面太恐怖了他不可能跟」

事實：
Nit 到河牌還在 = 範圍極強
他已經過了多個放棄點
現在他留下來是有原因的

✓ 正確心態：
Nit 在河牌 = 準備攤牌
不要浪費籌碼詐唬
```

### 錯誤 3：偷盲不夠頻繁

```
❌ 「這牌不夠好，棄掉」
❌ 「我已經連偷三次，會被抓」
❌ 「他這次應該會反擊」

事實：
Nit 的棄牌頻率極高且穩定
他不會因為你偷很多就開始報復
這是穩定的印鈔機

✓ 正確心態：
任何 +EV 的偷盲都要做
不要因為「感覺」而放棄利潤
```

### 錯誤 4：忽視他的加注

```
❌ 「他是 Nit，這次應該是詐唬」
❌ 「我頂對不可能棄牌」
❌ 「他不可能每次都有」

事實：
Nit 就是每次加注都有
這是他的定義
不要懷疑

✓ 正確心態：
Nit 加注 = 有牌
面對加注果斷棄牌
保護你的籌碼
```

## 實戰案例分析

### 案例 1：偷盲成功

```
6-max NL50，100BB 深度
你在 BTN 有 J♦5♦

翻前：
Fold 到你
SB（Nit，VPIP 11/PFR 9）
BB（普通玩家）

分析：
- SB 是 Nit，Fold to steal > 80%
- BB 普通，約 60% fold
- J5s 通常不是開池牌
- 但對這個盲位配置 = +EV

行動：開到 2.2BB
SB 棄牌
BB 棄牌

結果：贏得 1.5BB

計算：
投入 2.2BB
盲注 1.5BB
SB 棄牌率 85%，BB 棄牌率 60%
成功率約 51%（0.85 × 0.60）
EV = 0.51 × 1.5 - 0.49 × 2.2 ≈ -0.31

等等，這樣算是負 EV？

但實際上：
- 當 BB 跟注時，我們還有翻後機會
- J5s 有 playability
- 加上位置優勢
- 實際 EV 是正的

學習：
對 Nit 偷盲是基本操作
不要錯過這些小但穩定的利潤
```

### 案例 2：尊重 Nit 的 3-bet

```
6-max NL50，100BB 深度
你在 CO 有 J♠J♥

翻前：
你開到 2.5BB
BTN 棄牌
SB（Nit，VPIP 12/PFR 10，3-bet 4%）3-bet 到 9BB
BB 棄牌

分析：
- Nit 3-bet 4% = 只有 QQ+, AK
- JJ vs QQ+, AK 大約 36% equity
- 4-bet 會被 5-bet 或面對超強範圍
- 跟注看翻牌？

行動選項：
A) 4-bet 到 22BB
B) 跟注 9BB
C) 棄牌

分析各選項：
A) 4-bet：他會用 QQ+ 5-bet，AK 可能跟
   你的 JJ 處境不好

B) 跟注：看翻牌
   翻牌沒 J 很難繼續
   有位置劣勢

C) 棄牌：放棄 2.5BB
   避免後面更大損失

選擇：棄牌

這看起來很緊，但：
- JJ vs QQ+, AK = 36%
- 加上位置劣勢和翻後困難
- 長期 EV 棄牌可能最好

結果：保留籌碼等待更好機會

學習：
- 對 Nit 的 3-bet 範圍要有清醒認識
- JJ 不是什麼大牌
- 知道何時放手是智慧
```

### 案例 3：價值下注被加注

```
6-max NL50，100BB 深度
你在 BTN 有 K♠K♦

翻前：
MP（Nit，VPIP 13/PFR 11）開到 2.5BB
你 3-bet 到 8BB
Nit 跟注

翻牌：K♥ 8♦ 3♠（底池 17.5BB）

分析：
- 你中了頂暗三！
- Nit 的跟注範圍：TT-QQ、AK、AQs
- 這是一個非常好的價值情況

行動：你下注 8BB（46% pot）
Nit 跟注

轉牌：5♣（底池 33.5BB）

分析：
- 轉牌空白
- Nit 跟注 = 可能 AA、QQ、AK
- 繼續取值

行動：你下注 22BB（66% pot）
Nit 加注到 50BB

分析：
- Nit 轉牌加注！
- 他的範圍：AA、88（翻牌沒加注）、33
- 你有 KK set = 非常強

但仔細想想：
- 88 會在翻牌慢打？可能
- 33 會在翻牌慢打？可能
- AA 會在翻牌跟注轉牌加注？很合理

你的選項：
- 跟注看河牌
- 3-bet 全押

分析：
如果他有 AA，你領先
如果他有 88/33，你落後
範圍組合：AA(1)、88(3)、33(3) = 1:6
你大約 14% equity！

行動：跟注（控池看河牌）
河牌：9♥
Nit 全押剩餘約 42BB

這時候：
- 他的全押 = 幾乎 100% 88 或 33
- AA 不會在河牌全押（你 3-bet 過，他怕 KK）

選擇：棄牌

學習：
- 即使是 Set 也不是堅果
- Nit 的加注要認真對待
- 他轉牌加注河牌全押 = 至少 full house
```

## 對 Nit 策略總結

### 核心調整清單

```
翻前：
□ 瘋狂偷盲（擴大 50%+）
□ 對開池減少 3-bet 詐唬
□ 他 3-bet = 只用超強牌對抗

翻後：
□ C-bet 頻率提高，尺度減小
□ 他跟注轉牌 = 謹慎
□ 他加注 = 果斷棄牌
□ 河牌不詐唬

整體：
□ 主要利潤來自偷盲
□ 不要試圖「打敗」他
□ 避免和他打大底池
```

### 記住這些數字

```
vs Nit 的數學：

偷盲成功率：75-85%
需要成功率：約 50%
→ 幾乎任何牌都可以偷

他 3-bet 的範圍：QQ+, AK（4%）
你 JJ vs 這個範圍：36%
→ 可以考慮棄牌

他的河牌加注詐唬頻率：<5%
→ 面對加注基本都棄牌
```
