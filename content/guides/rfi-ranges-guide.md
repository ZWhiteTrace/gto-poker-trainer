# RFI 翻前開池範圍完全指南

## 什麼是 RFI？

RFI（Raise First In）指的是當前面所有玩家都棄牌時，你作為第一個加注進入底池的行為。這是撲克中最基礎也最重要的決策之一。

### 為什麼 RFI 如此重要？

1. **頻率最高**：大多數手牌你都面臨 RFI 決策
2. **奠定基礎**：正確的翻前範圍是所有後續策略的起點
3. **長期盈利**：翻前的小錯誤會在大量手數中累積成大損失
4. **心理優勢**：主動加注者在翻後擁有主動權

## 位置與範圍的關係

### 基本原則

**位置越靠後，開池範圍越寬。**

原因：
- 後位需要面對的對手更少
- 更容易偷到盲注
- 翻後有位置優勢

### 6-Max 各位置 RFI 範圍

#### UTG（槍口位）—— 最緊

**範圍：約 15-18% 的手牌**

```
對子：22+
同花：A2s+, K9s+, Q9s+, J9s+, T9s, 98s, 87s, 76s
雜色：ATo+, KJo+, QJo
```

**UTG 的特點：**
- 需要面對 5 個對手
- 被 3bet 的機率最高
- 必須有強牌支撐

#### HJ（劫位）—— 略寬

**範圍：約 20-22% 的手牌**

```
對子：22+
同花：A2s+, K8s+, Q9s+, J9s+, T8s+, 97s+, 87s, 76s, 65s
雜色：A9o+, KTo+, QTo+, JTo
```

#### CO（切斷位）—— 中等偏寬

**範圍：約 27-30% 的手牌**

```
對子：22+
同花：A2s+, K5s+, Q7s+, J8s+, T8s+, 97s+, 86s+, 75s+, 65s, 54s
雜色：A7o+, K9o+, Q9o+, J9o+, T9o
```

#### BTN（按鈕位）—— 最寬

**範圍：約 45-50% 的手牌**

```
對子：22+
同花：A2s+, K2s+, Q4s+, J6s+, T6s+, 96s+, 85s+, 75s+, 64s+, 54s, 43s
雜色：A2o+, K5o+, Q7o+, J8o+, T8o+, 98o, 87o
```

**BTN 的優勢：**
- 只需面對兩個盲注
- 翻後永遠有位置
- 可以用弱牌偷盲

#### SB（小盲位）—— 特殊處理

**範圍：約 40-45%（包含 limp）**

SB 的 RFI 策略較複雜，有兩種主流方式：

**策略 A：只加注（Raise or Fold）**
- 優點：簡單、激進
- 範圍：約 45% raise，其餘 fold

**策略 B：混合 limp**
- 優點：保護 limp 範圍
- Raise：強牌 + 部分 bluff
- Limp：中等牌力，希望看翻牌

## 加注尺度

### 標準尺度建議

| 位置 | 加注大小 | 說明 |
|------|---------|------|
| UTG-CO | 2.5bb | 標準開池 |
| BTN | 2.5bb | 標準偷盲 |
| SB | 3bb | 無位置需要更大 |

### 尺度調整因素

1. **對手類型**
   - vs 鬆玩家：加大尺度獲取價值
   - vs 緊玩家：標準尺度即可

2. **籌碼深度**
   - 深籌碼：可以用更大尺度
   - 短籌碼：考慮直接全下

3. **桌上動態**
   - 多人跟注：加大尺度
   - 頻繁 3bet：收緊範圍或用更極化策略

## 如何記憶 RFI 範圍

### 使用口訣

**UTG 口訣：「對子全開，ATs 起跳，同花連張到 76s」**

**BTN 口訣：「幾乎所有對子和 Ax，K5+ Q7+ 同花全開」**

### 分類記憶法

將手牌分為三類：

1. **核心牌**（永遠開池）
   - AA-TT, AK-AJ, KQ

2. **位置牌**（後位才開）
   - 小對子、弱同花 Ax、同花連張

3. **偷盲牌**（只在 BTN/SB 開）
   - 弱 Kx、Qx、雜色連張

### 視覺化記憶

使用 13x13 的範圍圖表：
- 對角線 = 對子
- 對角線上方 = 同花牌
- 對角線下方 = 雜色牌

## 面對 3bet 的調整

### 被 3bet 時的選擇

1. **4bet**：AA, KK, QQ, AK（價值）+ 部分 bluff
2. **Call**：JJ-99, AQs, KQs（有位置時更寬）
3. **Fold**：其餘

### 位置對應調整

| 你的位置 | 3bet 來源 | 防守範圍 |
|---------|----------|---------|
| UTG | 任何位置 | 很緊 |
| CO | BTN | 中等 |
| BTN | Blinds | 較寬 |

## 常見錯誤

### 1. UTG 開太寬
- **錯誤**：UTG 用 A9o 開池
- **正確**：UTG 只開 ATo+

### 2. BTN 開太緊
- **錯誤**：BTN 只開 20% 範圍
- **正確**：BTN 應該開 45%+ 偷盲

### 3. 忽視對手傾向
- **錯誤**：機械執行範圍
- **正確**：根據對手調整（緊桌開寬，鬆桌收緊）

### 4. SB 過度 limp
- **錯誤**：SB 每手都 limp
- **正確**：有選擇性地 raise 或 limp

## 進階概念

### 範圍平衡

GTO 要求你的範圍包含：
- **價值牌**：強牌，希望被跟注
- **Bluff 牌**：弱牌，但有阻隔效應

### 頻率調整

根據對手類型調整開池頻率：
- vs Nit：增加偷盲頻率
- vs LAG：收緊範圍，多 3bet
- vs Fish：標準範圍，價值最大化

## 總結

RFI 是撲克的基礎：

1. **掌握位置與範圍關係**：後位更寬
2. **記住標準範圍**：使用口訣和圖表
3. **了解加注尺度**：2.5-3bb 標準
4. **準備面對 3bet**：每次開池都要有計劃
5. **大量練習**：形成直覺反應
