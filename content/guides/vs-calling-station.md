# 打跟注站完整指南：讓他們付出代價

## 什麼是跟注站？

**跟注站 (Calling Station)** 是最常見也最容易剝削的玩家類型之一。他們的核心特徵是：**跟注太多，棄牌太少**。

```
跟注站的心理：
「我付錢看你的牌」
「萬一你在詐唬呢？」
「我有一對，不能棄牌」
「底池已經這麼大了...」

結果：
→ 用太弱的牌跟注到底
→ 讓對手輕鬆取值
→ 長期大量輸錢
```

---

## 識別跟注站

### 數據特徵 (HUD)

```
典型跟注站數據：

VPIP：35-60%（玩太多牌）
PFR：8-15%（很少主動加注）
VPIP-PFR 差距：>20%（大量冷跟）

翻後數據：
WTSD：>32%（攤牌率過高）
W$SD：<48%（攤牌勝率低）
Fold to C-bet：<40%（不棄牌）
Fold to River Bet：<45%（河牌也不棄）

關鍵指標：
VPIP-PFR 差距大 + WTSD 高 = 跟注站
```

### 行為特徵（無 HUD 時）

```
翻前信號：
□ 經常跛入/冷跟
□ 很少 3-bet
□ 面對加注幾乎不棄牌
□ 喜歡「看翻牌」

翻後信號：
□ 下注後很少加注
□ 面對下注幾乎都跟注
□ 很少主動詐唬
□ 河牌大注還在思考（不是秒棄）

心理信號：
□ 看起來只想娛樂
□ 輸了也不太在意
□ 喜歡看到攤牌
□ 常說「我 call」
```

### 跟注站的亞型

```
純魚型跟注站：
- 完全不懂策略
- 任何牌都可能跟
- 最容易剝削

好奇心跟注站：
- 就是想看你的牌
- 會用任何成手跟注
- 但不會用空氣跟

情緒型跟注站：
- 被詐唬過後不信任任何人
- 可能突然變得激進
- 需要注意狀態變化

膽小型跟注站：
- 跟注因為不敢加注
- 有強牌也只跟注
- 加注時要特別小心
```

## 核心策略：對跟注站的調整

### 黃金法則

```
對跟注站的三個核心原則：

1. 停止詐唬
   他不會棄牌 = 你的詐唬沒有意義

2. 瘋狂取值
   他會跟注 = 用所有能贏的牌下注

3. 增大尺度
   他對尺度不敏感 = 下更大榨取更多
```

---

## 翻前調整

### 開池範圍

```
對跟注站在後位時：

標準開池範圍可以維持
原因：
- 他會跟注 = 你能取值
- 位置優勢更重要
- 翻後能控制底池

稍微收緊的牌：
- 弱 Ax（A2o-A6o）
- 不好的同花（83s, 72s）
原因：這些牌很難取值

適合擴大的牌：
- 所有強頂對（AJ+, KQ）
- 適合取值的牌
原因：翻後能大量取值
```

### 隔離加注

```
跟注站跛入後：

大幅提高隔離頻率
目標：和他單挑

隔離範圍：
- 所有可取值的牌
- 好的頂對牌
- 減少投機牌（小對、弱同花連牌）

尺度：
標準 + 1-2BB
他反正會跟 = 建大底池

位置：
盡量在位置隔離
無位置時範圍收緊
```

### 面對 3-bet（罕見）

```
跟注站 3-bet 的含義：

非常強！
跟注站很少 3-bet
當他 3-bet 時 = AA/KK/QQ/AK

應對：
- 用頂級牌 4-bet
- 其他都棄牌
- 不要試圖詐唬
```

## 翻後調整

### C-bet 策略

```
對跟注站的 C-bet：

頻率：降低（50-60%）
但這不是因為他會加注
而是因為：沒價值的牌不用下注

什麼時候 C-bet：
✓ 有任何成手牌（頂對、中對、底對）
✓ 強聽牌
✓ 後門聽牌 + 高牌

什麼時候不 C-bet：
✗ 純空氣
✗ 弱聽牌
✗ 沒有改進空間的牌
原因：他不會棄牌，詐唬沒意義

尺度：
增大到 66-75% pot
他對尺度不敏感
```

### 取值下注

```
對跟注站擴大取值範圍：

標準取值：
頂對頂踢、超對、兩對+

對跟注站擴大：
✓ 頂對任何踢腳
✓ 中對好踢腳
✓ 底對有時也能取值

範例：
牌面 K-8-3
你有 K5s

vs 一般對手：可能過牌
vs 跟注站：下注取值！

他會用 K2、88 以下的對子、
甚至 A-high 跟注
```

### 河牌策略

```
河牌對跟注站：

取值：
- 三條街取值是標準
- 用比平常弱的牌取值
- 尺度可以更大（75-100% pot）

詐唬：
❌ 幾乎完全停止
除非：
- 明顯的威脅牌面
- 他這手牌表現特別弱
- 你有強阻擋牌

範例計算：
你河牌詐唬 75% pot
需要對手棄牌 43% 才有利
跟注站棄牌率 < 30%
→ 詐唬 = 燒錢
```

## 特殊情況處理

### 當跟注站下注時

```
跟注站主動下注的含義：

通常 = 有東西
可能是：
- 中等成手牌（不敢過牌）
- 聽牌想便宜看牌
- 偶爾的強牌

應對：
弱牌 → 棄牌或跟注
中等牌 → 跟注控池
強牌 → 加注取值（他會跟）

不要因為他是跟注站就：
- 用空氣詐唬加注
- 用弱牌跟注太多街
```

### 當跟注站加注時

```
跟注站加注 = 紅色警報！

跟注站的加注頻率非常低
當他加注時：

翻牌加注 = 兩對+、暗三、偶爾強聽牌
轉牌加注 = 至少兩對
河牌加注 = 幾乎 100% 堅果

應對：
除非你有超強牌
否則認真考慮棄牌

範例：
牌面 K-8-3-7-2
你有 KQ（頂對好踢腳）
跟注站河牌 check-raise

正確做法：棄牌！
他有 33、88、K8、87 等
```

### 多人底池中的跟注站

```
多人底池調整：

更緊的取值範圍：
- 需要更強的牌
- 跟注站 + 其他人

減少詐唬：
- 本來就不該詐唬跟注站
- 多人底池更不該詐唬

位置更重要：
- 盡量在跟注站後面行動
- 看他行動後再決定
```

## 常見錯誤

### 錯誤 1：還是想詐唬

```
❌ 「他這次一定會棄牌」
❌ 「這個牌面太恐怖了」
❌ 「我演得很像有牌」

事實：
跟注站不在乎牌面
跟注站不在乎你的行動線
跟注站只在乎自己的牌

✓ 正確心態：
他有成手就跟
他沒成手才棄
你的詐唬不影響這個
```

### 錯誤 2：取值太薄

```
❌ 「我怕他有更好的」
❌ 「他跟注應該是強牌」
❌ 「河牌太危險了」

事實：
跟注站的跟注範圍超寬
你的中等牌經常是最好的

✓ 正確心態：
vs 一般對手：頂對好踢取值
vs 跟注站：中對也能取值
```

### 錯誤 3：尺度太小

```
❌ 「下小一點他才會跟」
❌ 「我不想嚇跑他」

事實：
跟注站對尺度不敏感
他跟注 50% 也會跟注 80%

✓ 正確做法：
有牌就下大
榨取最大價值
```

### 錯誤 4：過度調整後被反殺

```
❌ 對跟注站的加注不當回事
❌ 「他是跟注站，不可能有牌」

事實：
跟注站加注 = 真的有牌
不要因為他是魚就忽略警訊

✓ 正確做法：
面對加注要重新評估
即使是跟注站也會有堅果
```

## 實戰案例分析

### 案例 1：標準取值線

```
6-max NL50，100BB 深度
你在 BTN 有 A♠J♥

翻前：
CO 跛入（已識別為跟注站）
你在 BTN 加注到 4BB
跟注站跟注

翻牌：A♣ 8♦ 3♠（底池 9.5BB）

分析：
頂對好踢腳 vs 跟注站 = 取值天堂

行動：下注 6BB（63% pot）
跟注站跟注

轉牌：5♥（底池 21.5BB）

分析：
轉牌空白，繼續取值
他會用 Ax、88、33、口袋對跟注

行動：下注 14BB（65% pot）
跟注站跟注

河牌：K♦（底池 49.5BB）

分析：
K 有點危險？
但 vs 跟注站：
- 他有 K 早就跟注了
- 他可能有 A8、A5、A3
- 繼續取值！

行動：下注 35BB（70% pot）
跟注站跟注，亮出 A4o

結果：你贏得 119.5BB 底池
```

### 案例 2：停止詐唬

```
6-max NL50，100BB 深度
你在 CO 有 7♠6♠

翻前：
你開到 2.5BB
BTN 棄牌
SB 棄牌
BB（跟注站）跟注

翻牌：K♥ 9♦ 4♣（底池 5.5BB）

分析：
完全沒中，要 C-bet 嗎？
vs 一般對手：可以詐唬
vs 跟注站：不要！

行動：過牌
BB 過牌

轉牌：2♠（底池 5.5BB）

分析：
還是空氣，後門同花
要不要再試一下？
不要！

行動：過牌
BB 下注 3BB
你棄牌

結果：省下了可能的 2-3 個詐唬下注

思考：
如果你翻牌 C-bet 4BB
轉牌繼續 10BB
河牌再詐唬 25BB
總共浪費 39BB 去詐唬一個不會棄牌的人
```

### 案例 3：面對加注的應對

```
6-max NL50，100BB 深度
你在 BTN 有 Q♥Q♦

翻前：
你開到 2.5BB
SB（跟注站）跟注

翻牌：J♠ 7♥ 3♦（底池 6BB）

行動：你下注 4BB
跟注站跟注

轉牌：5♣（底池 14BB）

行動：你下注 10BB
跟注站 CHECK-RAISE 到 28BB

分析：
跟注站 check-raise！
這是極強的信號

他的範圍：
- 77、33、55（暗三）
- J7s（兩對）
- 可能的 JJ

你的 QQ：
贏 JJ（不太可能）
輸給所有其他

行動：棄牌！

即使對手是跟注站
他的加注還是要尊重
```

## 對跟注站策略總結

### 核心調整清單

```
翻前：
□ 維持或稍微收緊開池
□ 積極隔離跛入
□ 他 3-bet = 超強牌

翻後：
□ 停止詐唬
□ 擴大取值範圍
□ 增大下注尺度
□ 三條街取值是標準

特殊情況：
□ 他下注 = 有東西
□ 他加注 = 超強
□ 多人底池更緊
```

### 記住這些數字

```
vs 跟注站的調整：

詐唬頻率：
標準 → 跟注站
30% → 5-10%

取值範圍：
標準 → 跟注站
頂對+ → 中對+

下注尺度：
標準 → 跟注站
60% → 75-100%
```

## 心態建設

```
打跟注站的正確心態：

1. 這是送錢的人
   不是要打敗他
   是要讓他送錢給你

2. 耐心等待
   不用詐唬
   等有牌再取值

3. 不要教育他
   他打得差 = 你賺錢
   教會他 = 你虧錢

4. 享受過程
   打跟注站應該是輕鬆的
   如果覺得煩躁 = 你在做錯事
```
