# SPR 深度解析：堆疊底池比的策略應用

## 什麼是 SPR？

**SPR (Stack-to-Pot Ratio)** 是有效籌碼與翻牌後底池的比值，決定了你應該如何規劃翻後的行動。

```
公式：
SPR = 有效籌碼 / 翻牌底池

範例：
有效籌碼：$100
翻牌底池：$10
SPR = 100 / 10 = 10
```

---

## SPR 的重要性

### 為什麼 SPR 決定策略？

```
核心洞見：
SPR 決定了你能「承諾」多少籌碼到底池

低 SPR（< 4）：
- 容易全進
- 頂對就是堅果
- 一對就能全進

高 SPR（> 13）：
- 需要更強的牌
- 頂對可能只是 bluff catcher
- 必須發展到兩對以上
```

### SPR 與承諾門檻

| SPR 範圍 | 全進承諾門檻 | 典型情況 |
|----------|--------------|----------|
| < 3 | 任何對子 | 短籌碼全進 |
| 3-6 | 頂對頂踢腳 | 3bet 底池 |
| 6-10 | 超對/兩對 | 單次加注底池 |
| 10-15 | 兩對/暗三 | 開池單跟 |
| > 15 | Set/順子/同花 | 深籌碼遊戲 |

---

## 低 SPR 策略 (SPR < 6)

### 低 SPR 的成因

```
1. 翻前多次加注
   - 3bet/4bet 底池
   - 底池已經膨脹

2. 籌碼相對淺
   - MTT 中後期
   - 短籌碼加入

3. 刻意建立
   - 有強牌時翻前大加注
```

### 低 SPR 策略要點

```
優勢牌型（大對子、高牌）：

✓ 頂對頂踢腳 = 有效堅果
✓ 可以激進地取值
✓ 不用擔心被反超
✓ 全進壓力有效

劣勢牌型（投機牌）：

✗ 隱含賠率不足
✗ 無法 set mine
✗ 同花連張價值降低
```

### 低 SPR 實戰範例

```
場景：你 BTN 3bet，BB 跟注
底池：$24，有效籌碼：$80
SPR = 80 / 24 ≈ 3.3

你的牌：A♠K♥
翻牌：K♦ 8♣ 4♠

策略：
- 頂對頂踢腳 = 極強
- C-bet $12-15
- 面對加注直接全進
- 不需要複雜思考

這個 SPR 下，TPTK 就是全進牌
```

---

## 中 SPR 策略 (SPR 6-13)

### 中 SPR 的平衡

```
中 SPR 最需要技巧：

- 頂對不再是自動全進
- 需要評估對手範圍
- 底池控制變重要
- 多條街規劃關鍵
```

### 中 SPR 策略要點

```
成牌時：
- 兩對以上可以舒服全進
- 頂對需要謹慎
- 觀察對手激進程度

聽牌時：
- 隱含賠率適中
- 可以半詐唬
- 注意底池控制
```

### 中 SPR 實戰範例

```
場景：你 CO 加注，BB 跟注
底池：$7，有效籌碼：$70
SPR = 70 / 7 = 10

你的牌：Q♠Q♥
翻牌：J♥ 8♣ 3♠

策略：
- 超對很強，但不是堅果
- C-bet $5（70% pot）
- 面對跟注：轉牌繼續下注
- 面對加注：需要考慮
  - 對手有沒有 JJ/88/33?
  - 你願意全進嗎？

這個 SPR 下，QQ 是取值但需要小心
```

---

## 高 SPR 策略 (SPR > 13)

### 高 SPR 的特徵

```
深籌碼遊戲：

- 單對價值大幅下降
- 投機牌價值提升
- 錯誤代價更高
- 技術差距放大
```

### 高 SPR 策略要點

```
價值體系改變：
- Set/兩對 = 有效堅果
- 超對/TPTK = Bluff Catcher
- 中對以下 = 通常棄牌

投機牌發光：
- 同花連張價值最大化
- 小對子可以 set mine
- 隱含賠率充足
```

### 高 SPR 實戰範例

```
場景：你 BTN 加注，BB 跟注
底池：$5，有效籌碼：$100
SPR = 100 / 5 = 20

你的牌：A♥A♠
翻牌：9♠ 7♠ 6♣

策略：
- AA 只是一對
- 這個濕潤牌面，對手可以有很多強牌
- C-bet 小（33-50%）
- 面對加注要非常謹慎
- 可能需要底池控制

這個 SPR 下，AA 不是堅果
可能輸給兩對、Set、順子、同花
```

## SPR 與手牌選擇

### 不同 SPR 的最佳手牌

```
低 SPR (< 6)：
冠軍：AA, KK, QQ, AK
優秀：JJ-TT, AQ, AJ
尷尬：小對子, 同花連張

原因：大對子直接贏，投機牌沒賠率
```

```
中 SPR (6-13)：
冠軍：AA-JJ, AK, AQ
優秀：TT-88, KQ, 大同花連張
可玩：77-55, 中同花連張

原因：平衡場景，各種牌都可能勝出
```

```
高 SPR (> 13)：
冠軍：Set Mining 牌 (22-99)
優秀：同花連張 (87s-JTs)
尷尬：AA-KK（難打大底池）

原因：隱含賠率驅動，堅果潛力重要
```

## SPR 規劃：翻前決策

### 用 SPR 指導翻前

```
你有 AA，想要低 SPR：
→ 翻前大加注/3bet
→ 創造 SPR 3-6 的情況
→ 翻後全進壓力

你有 65s，想要高 SPR：
→ 翻前小加注/跟注
→ 保持 SPR > 10
→ 翻後發展空間
```

### 實戰規劃範例

```
場景：你有 K♥K♠，UTG 加注 3BB

選項 A：跟注
預期底池：7BB
剩餘籌碼：97BB
SPR = 97/7 ≈ 14
→ KK 難打，需要謹慎

選項 B：3bet 到 9BB
預期底池：19BB（假設跟注）
剩餘籌碼：91BB
SPR = 91/19 ≈ 5
→ KK 可以舒服全進

結論：3bet 創造有利 SPR
```

## SPR 與下注尺度

### 低 SPR 下注策略

```
SPR 3-4 時：

翻牌下注：50-75% pot
轉牌下注：全進

原因：
- 只剩 1-2 條街
- 不需要複雜尺度
- 直接建立底池
```

### 中 SPR 下注策略

```
SPR 8-10 時：

需要規劃三條街：
翻牌：33-50% pot
轉牌：50-75% pot
河牌：75-100% pot

或者：
翻牌：大（75%）+ 轉牌全進
```

### 高 SPR 下注策略

```
SPR 15+ 時：

底池控制優先：
翻牌：25-40% pot
轉牌：視情況
河牌：價值最大化

用聽牌：
可以選擇激進建底池
或控制等待成牌
```

## SPR 動態變化

### 每條街 SPR 變化

```
翻牌 SPR = 10

你下注 75% pot，對手跟注
轉牌 SPR 計算：

翻牌底池：$10
你下注：$7.5
對手跟注：$7.5
轉牌底池：$25

剩餘籌碼：$100 - $7.5 = $92.5

轉牌 SPR = 92.5 / 25 = 3.7

→ 從高 SPR 變成低 SPR！
→ 策略也要跟著調整
```

### 規劃全局

```
關鍵思維：

翻牌時就要想到河牌
- 如果我下這個注...
- 轉牌 SPR 會變成多少？
- 河牌我能全進嗎？

範例：
初始 SPR = 10
想要河牌全進
→ 需要每條街約 75-80% pot
```

## SPR 與牌面結構

### 濕潤牌面 + 高 SPR

```
牌面：9♠ 8♠ 7♥
SPR = 15

危險組合：
- 對手範圍可以有很多堅果
- 你的超對/頂對價值降低
- 詐唬更有效（可以代表很多牌）

策略：
- 縮小下注尺度
- 準備棄牌面對加注
- 用聽牌攻擊
```

### 乾燥牌面 + 低 SPR

```
牌面：K♥ 7♣ 2♦
SPR = 4

有利情況：
- 對手不容易有怪物
- 頂對價值最大化
- 可以激進取值

策略：
- 大下注
- 準備全進
- 不要慢打
```

## 常見 SPR 錯誤

### 1. 低 SPR 慢打

```
❌ 錯誤：
SPR = 3，有 AA
翻牌過牌想「誘導」

✓ 正確：
直接下注建底池
低 SPR 你有絕對優勢
不需要複雜花招
```

### 2. 高 SPR 過度承諾

```
❌ 錯誤：
SPR = 15，有 QQ
翻牌 J 高，對手加注
你 4bet 全進

✓ 正確：
高 SPR 下 QQ 不是堅果
面對加注可以棄牌
或平跟注等待更多信息
```

### 3. 忽略 SPR 規劃

```
❌ 錯誤：
不看 SPR，隨意下注
結果河牌不知道該全進還是棄牌

✓ 正確：
翻前就規劃目標 SPR
每條街下注有明確意圖
```

## SPR 速查表

### 常見場景 SPR

```
單次加注底池（100BB）：
翻牌底池 ≈ 7BB
有效籌碼 ≈ 97BB
SPR ≈ 14

3bet 底池（100BB）：
翻牌底池 ≈ 25BB
有效籌碼 ≈ 75BB
SPR ≈ 3

4bet 底池（100BB）：
翻牌底池 ≈ 50BB
有效籌碼 ≈ 50BB
SPR ≈ 1
```

### SPR 決策快速參考

| SPR | 全進門檻 | 下注思路 |
|-----|----------|----------|
| < 3 | 任何對子 | 1-2 條街全進 |
| 3-6 | 頂對好踢腳 | 2 條街規劃 |
| 6-10 | 超對/兩對 | 3 條街規劃 |
| 10-15 | 兩對/Set | 底池控制 |
| > 15 | Set/堅果 | 深度遊戲 |

## 總結

### SPR 核心原則

1. **SPR 決定承諾門檻**
2. **低 SPR 利好大對子**
3. **高 SPR 利好投機牌**
4. **翻前規劃目標 SPR**
5. **每條街更新 SPR 計算**

### SPR 決策清單

```
翻前：
□ 評估我的牌想要什麼 SPR？
□ 調整加注尺度創造目標 SPR

翻後：
□ 計算當前 SPR
□ 確定承諾門檻
□ 規劃多條街下注
□ 每條街重新計算
```
