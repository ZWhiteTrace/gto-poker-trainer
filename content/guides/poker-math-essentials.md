# 撲克數學精要：贏家必備的計算能力

## 為什麼撲克數學重要？

**撲克數學** 是做出正確決策的基礎。

```
撲克的本質：
不確定性下的決策

數學幫助你：
1. 量化不確定性
2. 比較不同選項
3. 做出長期獲利的決定

記住：
短期是運氣
長期是數學
```

---

## 核心概念一：機率

### 基本機率計算

```
機率 = 有利結果數 / 總可能結果數

範例：
抽到 A 的機率 = 4/52 = 7.7%
翻前拿到 AA 的機率 = 6/1326 = 0.45%
```

### 撲克中的常見機率

```
翻前機率：
對子：5.9%
同花牌：23.5%
AK：1.2%
AA：0.45%

翻牌機率：
翻牌中一對（有兩張不同牌）：32.4%
翻牌中兩對或更好：2%
翻牌中同花聽牌（有兩張同花）：10.9%
```

## 核心概念二：Outs 和 Equity

### 什麼是 Outs

```
Outs = 能讓你贏牌的剩餘牌張數

常見 Outs：
同花聽牌：9 outs
兩頭順子聽牌：8 outs
內順聽牌：4 outs
一對變暗三：2 outs
兩高張變對子：6 outs
```

### Outs 轉換為 Equity

```
2-4 法則：

翻牌時（還有兩張牌）：
Equity ≈ Outs × 4

轉牌時（還有一張牌）：
Equity ≈ Outs × 2

範例：
同花聽牌 9 outs
翻牌時：9 × 4 = 36%
轉牌時：9 × 2 = 18%
```

```

```

## 核心概念三：底池賠率

### 計算底池賠率

```
底池賠率 = 跟注金額 / 總底池

範例：
底池 $100
對手下注 $50
你需要跟 $50

底池賠率 = 50 / (100 + 50 + 50) = 25%
```

### 常見下注尺寸的底池賠率

```
下注尺寸    需要 Equity
25% pot     16.7%
33% pot     20%
50% pot     25%
66% pot     28.5%
75% pot     30%
100% pot    33.3%
150% pot    37.5%
```

### 決策框架

```
如果 Equity > 底池賠率需求：
→ 跟注是 +EV

如果 Equity < 底池賠率需求：
→ 跟注是 -EV

範例：
你有同花聽牌（35% equity）
對手下注半池（需要 25%）
35% > 25% → 跟注！
```

## 核心概念四：期望值 (EV)

### 什麼是 EV

```
EV = 期望值 = 長期平均結果

公式：
EV = (贏的機率 × 贏的金額) - (輸的機率 × 輸的金額)

範例：
你有 40% 機率贏 $100
你有 60% 機率輸 $50

EV = (0.4 × 100) - (0.6 × 50)
EV = 40 - 30 = +$10
```

### 計算跟注 EV

```
範例：
底池 $100
對手全下 $50
你需要跟 $50
你的 equity：40%

計算：
贏時得到：$100 + $50 + $50 = $200
輸時損失：$50

EV = (0.4 × 200) - (0.6 × 50)
EV = 80 - 30 = +$50

淨 EV = +$50 - $50（投入）= +$0
→ 盈虧平衡（實際略正因為 40% > 需要的 25%）
```

### 計算詐唬 EV

```
範例：
底池 $100
你詐唬下注 $75
估計對手棄牌率：50%

計算：
成功時贏：$100
失敗時輸：$75

EV = (0.5 × 100) - (0.5 × 75)
EV = 50 - 37.5 = +$12.5

詐唬是 +EV！
```

## 核心概念五：隱含賠率

### 什麼是隱含賠率

```
定義：
考慮後續街可能贏得的額外籌碼

底池賠率：只看當前
隱含賠率：加上未來可贏的錢

適用場景：
- 籌碼深
- 聽牌隱蔽
- 對手會付錢
```

### 計算隱含賠率

```
範例：
底池 $50
對手下注 $50（需要 33%）
你有內順聽牌（17%）

直接賠率不夠！

但考慮：
- 對手剩餘 $300
- 中順預計再贏 $150

等效底池 = $50 + $50 + $150 = $250
等效賠率 = 50 / (50 + 50 + 50 + 150) = 16.7%

17% > 16.7% → 跟注變得可行
```

## 核心概念六：組合數學

### 手牌組合

```
基本概念：
同樣的手牌有多少種組合？

對子：6 種組合
AA = A♠A♥, A♠A♦, A♠A♣, A♥A♦, A♥A♣, A♦A♣

非對子同花：4 種組合
AKs = A♠K♠, A♥K♥, A♦K♦, A♣K♣

非對子不同花：12 種組合
AKo = 16 總 - 4 同花 = 12
```

### 阻擋效應

```
你的牌如何影響對手的組合？

範例：
你有 A♠K♠

對手 AA 的組合：
原本 6 種
你阻擋 1 張 A → 剩 3 種

對手 AK 的組合：
原本 16 種
你阻擋 A♠ 和 K♠ → 剩 9 種
```

### 範圍計數

```
範例：估算對手的範圍

對手 3-bet 範圍估計：
QQ+, AK

計算組合數：
QQ: 6
KK: 6
AA: 6
AKs: 4
AKo: 12
總計: 34 組合

你有 JJ，對手有多少比你強？
QQ+: 18 組合
AK: 16 組合（翻前約 45%）

加權計算對你的勝率...
```

## 實戰應用

### 應用 1：翻前全下決策

```
情況：
你有 JJ
對手全下 100bb
估計對手範圍：QQ+, AK

計算：
對手範圍組合：
QQ: 6, KK: 6, AA: 6, AK: 16 = 34

JJ vs 這個範圍：
vs QQ+: 約 19%（18 組合）
vs AK: 約 55%（16 組合）

加權 equity：
(18/34 × 19%) + (16/34 × 55%)
= 10% + 26% = 36%

需要 50% 才能跟注
36% < 50% → 棄牌
```

### 應用 2：河牌 Bluff-Catch

```
情況：
底池 $200
對手下注 $100
你需要 33% equity

問題：對手詐唬多少次？

計算：
如果對手從不詐唬 → 你輸
如果對手 33% 詐唬 → 盈虧平衡
如果對手 50% 詐唬 → 有利跟注

決策：
估計對手詐唬頻率
與 33% 比較
做出決定
```

### 應用 3：半詐唬下注

```
情況：
翻牌你有同花聽牌（9 outs = 35%）
底池 $50
你考慮下注 $35

分析：
如果對手棄牌 50%：
EV = 0.5 × $50 = $25（詐唬價值）

如果對手跟注 50%：
新底池 = $50 + $35 + $35 = $120
你的 equity = 35%
EV = 0.35 × $120 - $35 = $42 - $35 = $7

總 EV = 0.5 × $25 + 0.5 × $7 = $16
→ 半詐唬是好的！
```

## 快速心算技巧

### 技巧 1：簡化計算

```
複雜計算 → 簡化估算

例：36% × $180
≈ 1/3 × $180 = $60
實際：$64.8

足夠接近用於決策
```

### 技巧 2：記住關鍵數字

```
必記數字：

底池賠率：
1/3 pot = 20%
1/2 pot = 25%
Full pot = 33%

Outs：
4 outs = 翻牌 17%
8 outs = 翻牌 32%
9 outs = 翻牌 35%
```

### 技巧 3：組合計數

```
快速估算組合：

對子：6
非對子：16
同花非對子：4
不同花非對子：12
```

## 常見數學錯誤

### 1. 過度計算 Outs

```
錯誤：
「我有 15 outs！」
（但有些 outs 讓對手更強）

正確：
只算「乾淨」的 outs
考慮對手可能的牌
```

### 2. 忽略後續街

```
錯誤：
翻牌跟注用轉牌 equity 計算
（還有河牌要付錢）

正確：
考慮整個手牌的 EV
不只是一條街
```

### 3. 過度依賴隱含賠率

```
錯誤：
「我會中大牌贏很多」
（對手只剩 10bb）

正確：
隱含賠率需要：
- 深籌碼
- 對手會付錢
- 隱蔽的聽牌
```
