# 權益實現率 (EQR) 完整指南

## 什麼是權益實現率？

**Equity Realization (EQR)** 是衡量你實際能贏得多少權益的指標。原始權益 (Raw Equity) 只告訴你勝率，但 EQR 告訴你在實戰中能將多少勝率轉化為籌碼。

```
核心公式：
實際贏得 = 原始權益 × 權益實現率

範例：
原始權益：40%
EQR：90%
實際贏得 ≈ 40% × 90% = 36% 的底池
```

---

## 為什麼 EQR 這麼重要？

### 原始權益的局限性

```
假設你有 A5o vs QQ：

翻牌前權益：約 30%

但實際上：
- 沒位置時，你難以實現這 30%
- 有位置時，你能實現更多

這就是為什麼同樣的牌：
有位置 EQR ≈ 100-110%
沒位置 EQR ≈ 70-85%
```

### EQR 的影響因素

| 因素 | 提升 EQR | 降低 EQR |
|------|----------|----------|
| 位置 | 有位置 | 沒位置 |
| 技術差距 | 你更強 | 對手更強 |
| 牌的可玩性 | 同花連張 | 乾燥高牌 |
| 籌碼深度 | 深籌碼（好牌） | 淺籌碼（投機牌） |
| 底池類型 | 單挑 | 多人底池 |

## EQR 數值解讀

### 典型 EQR 範圍

```
EQR > 100%：你能贏得比原始權益更多
           （通常是有位置 + 技術優勢）

EQR = 100%：完美實現原始權益

EQR 80-100%：正常實現範圍

EQR < 80%：權益被大幅折損
           （沒位置 + 難打的牌）

EQR < 50%：嚴重無法實現
           （應該翻前棄牌）
```

### 不同手牌的 EQR

```
高 EQR 手牌：
- 口袋對子（容易打）
- 同花連張（多種贏法）
- 有位置的寬牌（控制底池）

低 EQR 手牌：
- 乾燥非同花（K7o, Q5o）
- 沒位置的邊緣牌
- 難以在翻後遊玩的牌
```

## 位置對 EQR 的影響

### 有位置 vs 沒位置

```
以 87s 為例：

有位置（BTN vs BB）：
- 可以控制底池大小
- 得到完整的信息
- 詐唬機會更多
- EQR ≈ 105-115%

沒位置（BB vs BTN）：
- 必須先行動
- 信息劣勢
- 容易被詐唬
- EQR ≈ 75-90%

差距：約 20-30%！
```

### 各位置 EQR 基準

| 位置對決 | 典型 EQR 範圍 |
|----------|---------------|
| BTN vs BB | 100-115% |
| CO vs BB | 95-110% |
| SB vs BB | 85-100% |
| BB vs BTN | 75-95% |
| BB vs CO | 80-95% |
| BB vs EP | 85-100% |

## 牌型與 EQR

### 同花連張 (Suited Connectors)

```
為什麼同花連張 EQR 高？

1. 多種成牌可能
   - 順子
   - 同花
   - 兩對/三條

2. 隱蔽性強
   - 對手難讀
   - 大底池潛力

3. 好棄牌
   - 錯過可以輕鬆棄牌
   - 不會陷入困境

87s EQR ≈ 100-120%（取決於位置）
```

### 乾燥高牌 (Dry Offsuit Hands)

```
為什麼 K7o 這類牌 EQR 低？

1. 成牌困境
   - 頂對弱踢腳
   - 難以繼續行動

2. 可玩性差
   - 沒有聽牌潛力
   - 容易被支配

3. 昂貴錯誤
   - 經常付錢給更好的頂對
   - 第二好的牌

K7o EQR ≈ 70-85%（經常應該棄牌）
```

### 口袋對子

```
小口袋對子 (22-66)：
- EQR 高度依賴籌碼深度
- 深籌碼：中暗三 = 大收益
- 淺籌碼：難以set mine

深籌碼 (100BB+)：EQR ≈ 100-110%
淺籌碼 (30BB-)：EQR ≈ 60-80%

大口袋對子 (TT+)：
- 天然高 EQR
- 容易贏小底池或輸大底池
- AA/KK EQR ≈ 95-105%
```

## 多人底池的 EQR

### 為什麼多人底池 EQR 降低？

```
原因：

1. 必須對抗多個範圍
   - 更難詐唬
   - 邊緣牌價值下降

2. 隱含賠率下降
   - 對手也更謹慎
   - 大底池難建立

3. 更常攤牌
   - 詐唬減少
   - 純靠牌力

EQR 降幅：
3 人底池：EQR × 0.85-0.95
4+ 人底池：EQR × 0.75-0.90
```

### 多人底池策略調整

```
應該做的：
✓ 收緊範圍
✓ 用更強的牌下注
✓ 減少詐唬
✓ 重視堅果牌

應該避免的：
✗ 薄取值
✗ 多條街詐唬
✗ 邊緣牌過度承諾
```

## 籌碼深度與 EQR

### 深籌碼 (100BB+)

```
高 EQR 牌型：
- 同花連張
- 小口袋對子
- 可以 set mine / 追逐大牌

低 EQR 牌型：
- 支配牌（AK vs AQ 場景）
- 單對牌（難建大底池）

策略：
- 投機牌價值提升
- 隱含賠率重要
- 可以打得更寬
```

### 淺籌碼 (30BB-)

```
高 EQR 牌型：
- 大對子
- 強高牌（AK, AQ）
- 容易全進的牌

低 EQR 牌型：
- 同花連張（沒有足夠深度）
- 小口袋對子（沒有 set mining 價值）

策略：
- 高牌價值提升
- 隱含賠率不重要
- 範圍要更緊
```

## 實戰 EQR 計算

### 翻前決策範例

```
場景：CO 加注，你在 BB 有 76s
底池：3BB，需要跟注 2BB
賠率：2 / 5 = 40%

原始權益 vs CO 範圍：約 38%

問題：該跟注嗎？

分析：
如果 EQR = 100%：
需要 40% 權益，有 38%，略虧

如果 EQR = 85%（沒位置）：
實際權益 = 38% × 85% = 32%
明顯虧損，棄牌

如果 EQR = 105%（假設你有位置）：
實際權益 = 38% × 105% = 40%
可以跟注
```

### 翻後調整

```
翻牌：9♠ 8♣ 2♦
你有：76s（開放式順子聽牌）

原始權益：約 35% vs 對手範圍

EQR 考量：
+ 清晰的牌：中就贏，不中就棄牌
+ 隱蔽性：對手難讀
+ 可以半詐唬

這種情況 EQR ≈ 100-120%
（聽牌 EQR 通常高於原始權益）
```

## EQR 與範圍構建

### 翻前範圍中的 EQR 考量

```
構建 BB 防守範圍時：

不能只看原始權益！

❌ 錯誤思維：
「K7o 有 42% 權益，可以跟注」

✓ 正確思維：
「K7o 有 42% 權益，但 EQR 約 75%
實際權益 = 42% × 75% = 31.5%
不足以跟注」
```

### 用 EQR 調整範圍

```
高 EQR 替代低 EQR：

與其用：K7o（42% 權益 × 75% EQR = 31%）
不如用：65s（35% 權益 × 110% EQR = 38%）

雖然原始權益較低
但實際收益較高！
```

## Solver 中的 EQR

### 如何從 Solver 讀取 EQR

```
Solver 輸出：
- EV：某手牌的期望值
- 權益：原始勝率

計算 EQR：
EQR = (EV / 底池) / 原始權益

範例：
底池：10BB
你的 EV：3.5BB
原始權益：40%

EQR = (3.5/10) / 0.40 = 0.35 / 0.40 = 87.5%
```

### EQR 觀察要點

```
從 Solver 學習 EQR：

1. 位置差異
   - 比較 IP vs OOP 同樣手牌的 EV

2. 牌型差異
   - 同花連張 vs 非同花高牌

3. 籌碼深度
   - 100BB vs 50BB 的 EV 變化
```

## 常見 EQR 錯誤

### 1. 只看原始權益

```
❌ 「我有 45% 權益，好牌！」
✓ 「我有 45% 權益，但 EQR 多少？」
```

### 2. 忽略位置影響

```
❌ 「這手牌在任何位置都一樣強」
✓ 「有位置 EQR 可能高 20-30%」
```

### 3. 高估邊緣牌

```
❌ 「K9o 也是高牌，應該可以玩」
✓ 「K9o EQR 很低，通常應棄牌」
```

### 4. 多人底池打太寬

```
❌ 「反正賠率好，跟注吧」
✓ 「多人底池 EQR 大幅降低，需要更強的牌」
```

## 提升 EQR 的策略

### 技術層面

```
1. 打有位置
   - 優先攻擊盲注位
   - 避免沒位置跟注

2. 選擇高可玩性手牌
   - 同花連張 > 非同花高牌
   - 避免支配問題

3. 控制底池
   - 邊緣牌小底池
   - 強牌大底池
```

### 心理層面

```
1. 認識自己的弱點
   - 哪些牌型你打不好？
   - 哪些位置對決你虧損？

2. 對手調整
   - 對弱對手 EQR 提升
   - 對強對手 EQR 降低
```
