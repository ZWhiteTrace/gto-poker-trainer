# cEV vs $EV：錦標賽進階決策指南

## 什麼是 cEV 和 $EV？

**cEV (Chip EV)** 是籌碼期望值，衡量一個決策能贏或輸多少籌碼。
**$EV (Dollar EV / ICM EV)** 是金錢期望值，衡量一個決策對你最終獎金的影響。

```
核心差異：

cEV：每一個籌碼價值相等
$EV：籌碼價值遞減（ICM 效應）

這就是為什麼：
現金遊戲 → 只看 cEV
錦標賽 → 必須考慮 $EV
```

---

## 為什麼籌碼價值會遞減？

### ICM 的核心原理

```
範例：3 人 SNG，獎金 50/30/20

起始：每人 1000 籌碼
總籌碼：3000
總獎金：$100

初始籌碼價值：
1000 籌碼 ≈ $33.33

但如果你贏到 2000 籌碼：
2000 籌碼 ≈ $50（不是 $66.66！）

因為：
- 你不能贏走所有獎金
- 第二名還是拿 $30
- 籌碼領先的邊際價值降低
```

### 籌碼價值遞減示意

```
假設 100 人 MTT，總籌碼 100000

擁有籌碼 → 每籌碼價值（相對）
1000      → 100%
5000      → 95%
10000     → 88%
20000     → 78%
50000     → 55%
100000    → 35%（全部籌碼）

籌碼越多，每個額外籌碼的邊際價值越低
```

## cEV 和 $EV 何時差異最大？

### 差異最大的情況

```
1. 泡沫期（Bubble）
   - 出局 = $0
   - 存活 = 保證獎金
   - 風險厭惡最高

2. 決賽桌
   - 獎金階梯陡峭
   - 每個晉級都很值錢

3. 面對大籌碼
   - 輸給籌碼領先者很痛
   - 贏了也只得到「便宜籌碼」

4. 你是短籌碼
   - 存活比贏更重要
   - 需要極端收緊
```

### 差異最小的情況

```
1. 錦標賽早期
   - 離獎金很遠
   - 幾乎等於 cEV

2. 你是籌碼領先者
   - 可以更激進
   - 施壓其他玩家

3. 單桌 SNG 早期
   - 獎金結構還遠
   - 打正常遊戲
```

## 泡沫策略：$EV 實戰

### 泡沫的定義

```
泡沫 = 比保證獎金多 1 人的時候

例如：
90 人獎金圈
91 人存活 = 泡沫

泡沫心理：
- 出局 = 什麼都沒有
- 再活 1 人 = 保證獎金
- 極度風險厭惡
```

### 泡沫策略調整

```
如果你是大籌碼：

cEV 思維：「這手牌有正期望，跟注」
$EV 思維：「我可以施壓所有人，不需要邊緣牌」

策略：
✓ 頻繁攻擊中短籌碼
✓ 避免和其他大籌碼對抗
✓ 利用別人的恐懼
```

```
如果你是中籌碼：

cEV 思維：「66 可以跟注這個推注」
$EV 思維：「如果輸了我出局，贏了也只是變成中偏大」

策略：
✓ 極度收緊
✓ 只玩超強牌
✓ 讓短籌碼先死
```

```
如果你是短籌碼：

cEV 思維：「我需要翻倍，K9s 夠強」
$EV 思維：「也許我能熬過泡沫」

策略：
✓ 更謹慎選擇推注時機
✓ 考慮熬過泡沫的價值
✓ 不是任何 +cEV 都該打
```

## 計算 $EV 調整

### ICM 壓力量化

```
ICM 壓力指數（簡化）：

低壓力（早期）：
$EV ≈ cEV × 0.95-1.00

中壓力（接近泡沫）：
$EV ≈ cEV × 0.80-0.90

高壓力（泡沫）：
$EV ≈ cEV × 0.50-0.70

極高壓力（FT 泡沫）：
$EV ≈ cEV × 0.30-0.50
```

### 實戰範例

```
場景：MTT 泡沫，你有 25BB
BB 推注 10BB，你在 BTN 有 A9o

cEV 計算：
對手範圍：前 25%
A9o 權益：約 55%
cEV = +1.5BB（正期望）

$EV 調整：
泡沫壓力：× 0.60
$EV = +1.5 × 0.60 = +0.9BB

但考慮到：
- 出局 = $0
- 存活 = 可能 $500+
- 風險太大

結論：棄牌！
即使 cEV 正，$EV 可能是負的
```

## 決賽桌策略

### 決賽桌獎金結構

```
典型 9 人決賽桌獎金分配：
1st: 35%
2nd: 22%
3rd: 15%
4th: 11%
5th: 7%
6th: 4%
7th: 3%
8th: 2%
9th: 1%

觀察：
9→8 差距小（1%）
3→2 差距大（7%）
2→1 差距最大（13%）
```

### 決賽桌策略調整

```
早期（9-7 人）：
- 等待短籌碼出局
- 中等籌碼要保守
- 大籌碼可以施壓

中期（6-4 人）：
- 獎金階梯變陡
- 每個位置都重要
- 更多 ICM 考量

後期（3 人）：
- 短籌碼爆發力重要
- 大籌碼控制比賽
- 準備 HU
```

## 特殊情況

### Chip Leader vs Chip Leader

```
場景：
你和另一個大籌碼對決
其他人都是短籌碼

cEV：「我有 AQ，對手 3bet，可以跟」
$EV：「讓他們去打，我收割短籌碼」

策略：
- 避免和大籌碼大對抗
- 讓短籌碼互相淘汰
- 除非有極強牌
```

### 短籌碼全壓決策

```
場景：
你有 5BB，泡沫期
拿到 K5s，前面棄牌到你

cEV：「K5s 是正期望推注」

$EV 考量：
- 還有 3 個 3BB 籌碼
- 他們可能先死
- 熬過泡沫值錢

決策：
如果短籌碼很短 → 棄牌等待
如果大家都差不多 → 推注
```

### 面對加注的防守

```
場景：
決賽桌，5 人剩餘
你 BB 有 20BB，BTN 加注 2.5BB
你有 A7o

cEV：「賠率好，可以跟注」

$EV：
- 翻後沒位置難打
- 可能付出更多
- 邊緣牌風險高

策略：
✓ 3bet 或棄牌
✗ 避免邊緣跟注
```

## 常見 $EV 錯誤

### 1. 只看 cEV

```
❌ 「我有 AK，cEV 正，全進」
✓ 「考慮 ICM，這個全進 $EV 可能是負的」
```

### 2. 泡沫期不調整

```
❌ 「我平常這樣玩，現在也這樣」
✓ 「泡沫期極度收緊，除非極強牌」
```

### 3. 決賽桌過於激進

```
❌ 「要贏就要打，A9 夠強了」
✓ 「每個位置都值錢，邊緣牌要小心」
```

### 4. 忽略籌碼分布

```
❌ 「我 20BB，正常推注範圍」
✓ 「看看誰會跟注我，他們的籌碼多少」
```

## 進階：cEV 陷阱

### 什麼是 cEV 陷阱？

```
定義：
一個決策是 +cEV 但 -$EV

發生條件：
- ICM 壓力高
- 邊緣正期望
- 風險不對稱
```

### cEV 陷阱範例

```
場景：
3 人 SNG 泡沫
籌碼：你 3000，A 4000，B 3000
獎金：$60/$30

B 推注 3000（他的全部）
你有 ATo

cEV：
ATo vs 25% 範圍 ≈ 55% 權益
cEV = +300 籌碼

$EV：
如果你棄牌：等 A 和 B 打
如果 A 淘汰 B，你保證第二（$30）
如果你全進輸了，你出局（$0）

計算：
棄牌 $EV > 全進 $EV

結論：棄牌！
這是經典 cEV 陷阱
```

## 實戰決策框架

### $EV 決策清單

```
□ 目前是什麼階段？
  - 早期 → cEV 為主
  - 泡沫 → $EV 優先
  - 決賽桌 → ICM 關鍵

□ 我的籌碼狀況？
  - 大籌碼 → 可以施壓
  - 中籌碼 → 極度謹慎
  - 短籌碼 → 選擇性激進

□ 對手籌碼狀況？
  - 短籌碼 → 可能很寬
  - 大籌碼 → 可能很緊
  - 覆蓋你 → 要小心

□ 這個決策的風險/回報？
  - 贏了得到什麼？
  - 輸了失去什麼？
  - 值得嗎？
```

## 總結

### cEV vs $EV 核心原則

1. **現金遊戲用 cEV，錦標賽用 $EV**
2. **籌碼價值遞減是 ICM 核心**
3. **泡沫期 $EV 調整最大**
4. **大籌碼施壓，中籌碼收緊**
5. **避免 cEV 陷阱**

### 快速參考

```
早期：cEV ≈ $EV
泡沫：$EV << cEV（收緊）
決賽桌：$EV 主導
短籌碼 vs 短籌碼：接近 cEV
大籌碼 vs 大籌碼：避免
```
