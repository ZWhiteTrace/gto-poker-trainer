# 底池賠率完全指南：撲克數學的核心

## 什麼是底池賠率？

**底池賠率 (Pot Odds)** 是撲克最基本也最重要的數學概念。

```
定義：
底池賠率 = 跟注金額 : 可贏金額

範例：
底池 $100，對手下注 $50
你需要跟注 $50 來贏 $150
底池賠率 = 50:150 = 1:3 = 25%

意義：
你需要 25% 的勝率才能盈虧平衡
```

---

## 底池賠率計算

### 基本公式

```
底池賠率 % = 跟注金額 / (底池 + 對手下注 + 跟注金額)

範例 1：
底池 $100，對手下注 $50
= 50 / (100 + 50 + 50)
= 50 / 200
= 25%

範例 2：
底池 $80，對手下注 $40
= 40 / (80 + 40 + 40)
= 40 / 160
= 25%
```

### 常見下注尺寸的底池賠率

```
對手下注底池 % → 你需要的勝率

25% pot  → 16.7%
33% pot  → 20%
50% pot  → 25%
66% pot  → 28.5%
75% pot  → 30%
100% pot → 33.3%
150% pot → 37.5%
200% pot → 40%
```

### 快速心算技巧

```
記住這些常見數字：

1/4 pot = 17%
1/3 pot = 20%
1/2 pot = 25%
2/3 pot = 29%
3/4 pot = 30%
Full pot = 33%

技巧：
33% 是底池下注的基準
每增加 1/3 pot 約需多 4-5% 勝率
```

## 底池賠率 vs 手牌勝率

### 基本決策框架

```
決策規則：

如果 手牌勝率 > 底池賠率需求
→ 跟注是 +EV（有利可圖）

如果 手牌勝率 < 底池賠率需求
→ 跟注是 -EV（長期虧損）

如果 手牌勝率 = 底池賠率需求
→ 盈虧平衡
```

### 實戰範例

```
範例 1：同花聽牌

你的牌：A♥K♥
公共牌：7♥2♥9♣
底池：$100
對手下注：$50（半池）

計算：
- 底池賠率需求：25%
- 同花聽牌勝率：約 35%（9 outs）
- 35% > 25% → 跟注！
```

```
範例 2：內順聽牌

你的牌：JT
公共牌：A-K-7
底池：$100
對手下注：$100（全池）

計算：
- 底池賠率需求：33%
- 內順聽牌勝率：約 17%（4 outs）
- 17% < 33% → 棄牌！
```

## Outs 與勝率換算

### 什麼是 Outs

```
Outs = 能讓你贏牌的剩餘牌張數

常見聽牌的 Outs：

同花聽牌：9 outs
（13張同花 - 手牌2張 - 公牌2張）

兩頭順子聽牌 (OESD)：8 outs
（兩端各4張）

內順聽牌 (Gutshot)：4 outs
（只有一端4張）

超對聽牌：2 outs
（一對變暗三）
```

### Outs 到勝率的轉換

```
2-4 法則：

翻牌到河牌（兩張牌）：
勝率 ≈ Outs × 4%

轉牌到河牌（一張牌）：
勝率 ≈ Outs × 2%

範例：
同花聽牌 9 outs
- 翻牌時：9 × 4 = 36%
- 轉牌時：9 × 2 = 18%
```

### 常見聽牌勝率表

```
聽牌類型        Outs    翻牌勝率    轉牌勝率
─────────────────────────────────────────
同花聽牌         9       35%        19%
兩頭順子(OESD)   8       31%        17%
同花+順子       15       54%        33%
內順聽牌         4       17%         9%
兩高張(超對)     6       24%        13%
一對變兩對       5       20%        11%
暗三聽牌         2        8%         4%
```

## 隱含賠率

### 什麼是隱含賠率

```
定義：
隱含賠率 = 考慮後續街可能贏得的額外籌碼

底池賠率：只看當前底池
隱含賠率：加上未來可能贏的錢

適用場景：
- 對手還有很多籌碼
- 你的聽牌有隱蔽性
- 中牌後容易得到 action
```

### 隱含賠率計算

```
範例：

當前情況：
- 底池 $50，對手下注 $50
- 底池賠率需求：33%
- 你有內順聽牌：17%
- 直接賠率不夠！

但考慮隱含賠率：
- 對手剩餘籌碼：$300
- 如果中順，預期再贏 $150
- 等效賠率：50/(50+50+50+150) = 16.7%
- 17% > 16.7% → 跟注變得可行
```

### 隱含賠率使用條件

```
好的隱含賠率情況：
✓ 對手籌碼深
✓ 你的聽牌隱蔽
✓ 中牌後是堅果或接近堅果
✓ 對手傾向 pay off

差的隱含賠率情況：
✗ 對手籌碼短
✗ 聽牌明顯（如明花面）
✗ 中牌後可能不是最好
✗ 對手很緊，不會付錢
```

## 反向隱含賠率

### 什麼是反向隱含賠率

```
定義：
中牌後反而輸更多的風險

範例：
你有 K♥Q♥，公牌 A♥7♣2♦
翻牌來 J♥ 讓你成同花

問題：
- 如果對手有 A♥x♥（更大同花）
- 你會在後續街輸掉更多

這就是反向隱含賠率
```

### 常見反向隱含賠率場景

```
1. 非堅果同花聽牌
   - 可能中小同花輸大同花

2. 下端順子聽牌
   - 如 54 在 876 牌面
   - 中順可能輸更大順子

3. 弱踢腳的頂對
   - 如 A2 在 A-K-7 牌面
   - 對手可能有更好 Ax

4. 低對子
   - 如 22 在 A-7-2 牌面
   - 對手可能有更大暗三
```

### 如何處理反向隱含賠率

```
策略：

1. 減少這類牌的投入
   - 不要用邊緣聽牌看太多牌

2. 注意牌面發展
   - 危險牌來時要謹慎

3. 降低中牌後的期望值
   - 計算時要打折扣

4. 優先選擇堅果聽牌
   - A♥x♥ 比 7♥6♥ 好
```

## 底池賠率在不同街的應用

### 翻牌圈

```
特點：
- 還有兩張牌要來
- 聽牌勝率較高
- 隱含賠率較大

策略：
- 可以較寬地跟注聽牌
- 考慮後續街的發展
- 用 4 倍法則
```

### 轉牌圈

```
特點：
- 只剩一張牌
- 聽牌勝率減半
- 底池通常更大

策略：
- 需要更好的賠率
- 用 2 倍法則
- 考慮是否該放棄
```

### 河牌圈

```
特點：
- 沒有更多牌了
- 純粹的對決
- 沒有隱含賠率

策略：
- 只考慮當前底池賠率
- 估算對手的範圍
- 決定是否 bluff-catch
```

## 實戰應用範例

### 範例 1：標準同花聽牌

```
情況：
你：A♠9♠
公牌：K♠7♠2♦
底池：$20
對手下注：$15

計算：
1. 底池賠率需求
   15 / (20 + 15 + 15) = 30%

2. 你的勝率
   9 outs × 4 = 36%

3. 決策
   36% > 30% → 跟注

額外考量：
- 你有 A high 可能本身就贏
- 後位有位置優勢
- 結論：明確跟注
```

### 範例 2：邊緣聽牌決策

```
情況：
你：J♥T♥
公牌：Q♠8♣3♦
底池：$50
對手下注：$40

計算：
1. 底池賠率需求
   40 / (50 + 40 + 40) = 30.8%

2. 你的 Outs
   - 順子：4 (9) = 4 outs
   - 兩對+：6 outs
   - 總計：約 10 outs

3. 勝率
   10 × 4 = 40%

4. 但要考慮：
   - 一些 outs 可能無效
   - Q 可能給對手更大順
   - 打折到 7-8 clean outs
   - 實際約 28-32%

5. 決策
   邊緣，但有位置可以跟注
```

### 範例 3：河牌 Bluff-Catch

```
情況：
你：K♦K♣
公牌：A♥9♠5♦7♣2♥
底池：$100
對手下注：$75

計算：
1. 底池賠率需求
   75 / (100 + 75 + 75) = 30%

2. 評估對手範圍
   - 價值牌：Ax, 暗三, 兩對
   - 詐唬牌：錯過的聽牌, air

3. 需要對手 30% 在詐唬
   - 如果對手很少詐唬 → 棄牌
   - 如果對手常詐唬 → 跟注

4. 決策
   取決於對手類型的讀牌
```

## 常見錯誤

### 1. 忽略底池賠率

```
錯誤：
「我有聽牌，當然要跟」

正確：
計算賠率需求
比較手牌勝率
做出 +EV 決策
```

### 2. 過度依賴隱含賠率

```
錯誤：
「我會中大牌贏很多」
（對手籌碼只剩 20bb）

正確：
隱含賠率需要：
- 深籌碼
- 對手會付錢
- 隱蔽的聽牌
```

### 3. 忽略反向隱含賠率

```
錯誤：
用 76s 跟注只為了中同花
（可能中小同花輸大同花）

正確：
考慮中牌後的情況
優先選擇堅果聽牌
適當打折勝率
```

### 4. 不調整已知資訊

```
錯誤：
「同花 9 outs」
（但對手可能有更大同花聽牌）

正確：
根據已知資訊調整 outs
考慮牌面結構
估算 clean outs
```

## 總結

### 底池賠率核心概念

```
1. 基本公式
   跟注金額 / 總底池 = 需要勝率

2. 常見記憶點
   半池 = 25%
   全池 = 33%

3. Outs 換算
   翻牌：outs × 4
   轉牌：outs × 2

4. 隱含賠率
   考慮未來可贏的錢
   但要謹慎使用

5. 反向隱含賠率
   考慮中牌後輸更多的風險
```

### 底池賠率檢查清單

```
每次面對下注時：
□ 計算底池賠率需求
□ 評估手牌勝率（outs）
□ 比較勝率和需求
□ 考慮隱含賠率（如適用）
□ 考慮反向隱含賠率
□ 做出最佳 EV 決策
```
