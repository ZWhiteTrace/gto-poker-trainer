# 轉牌打法基礎：從翻牌到河牌的橋樑

## 轉牌的重要性

**轉牌**是撲克中最具挑戰性的街道，因為：

1. **籌碼深度變淺** - 底池變大，相對籌碼變少
2. **信息更多** - 翻牌的行動提供了更多信息
3. **決策影響大** - 轉牌決定河牌的底池大小

```
籌碼深度對比（100BB 起手）：

翻前：100BB effective
翻牌後：約 80BB（加入約 20BB）
轉牌後：約 50BB（底池可能 40BB+）
河牌：可能已經 all-in 或接近

結論：轉牌決策對籌碼影響巨大
```

## 轉牌的策略分類

### 1. 作為進攻者（翻牌下注者）

#### 繼續下注 (Double Barrel)

**何時繼續？**

```
✓ 你有強牌，繼續取值
✓ 轉牌對你有利（完成聽牌、阻擋對手）
✓ 對手範圍弱（capped）
✓ 你有好的詐唬牌（權益 + 阻擋）
```

**何時停止？**

```
✗ 你沒有牌力也沒有權益
✗ 轉牌對對手有利
✗ 對手展現強度（翻牌加注後跟注）
✗ 籌碼已經接近 commitment
```

### 2. 作為防守者（翻牌跟注者）

#### 繼續防守

```
✓ 你仍有足夠權益
✓ 底池賠率有利
✓ 對手可能在詐唬
✓ 你的牌有 showdown value
```

#### 考慮棄牌

```
✗ 權益大幅下降
✗ 對手第二次下注代表強度
✗ 你的聽牌沒有進步
✗ 賠率不足
```

## 轉牌牌型分析

### 好的轉牌（對進攻者）

#### 1. 高牌（A, K, Q）

```
牌面：8♦ 7♣ 3♠ → A♥

分析：
- A 在進攻者範圍中更多
- 防守者的 88, 77 現在怕 Ax
- 好的繼續下注牌
```

#### 2. 完成聽牌的牌

```
牌面：K♠ Q♥ 3♦ → T♣

分析：
- 完成 AJ 的順子
- 進攻者範圍有更多 AJ
- 可以代表順子詐唬
```

#### 3. 配對牌面

```
牌面：J♠ 8♦ 4♣ → 4♥

分析：
- 翻牌的跟注者不太可能有 4
- 進攻者可以代表有 4
- 好的詐唬牌
```

### 差的轉牌（對進攻者）

#### 1. 中間連接牌

```
牌面：K♦ 7♣ 3♠ → 8♥

分析：
- 8 連接 7，可能完成對手順子
- 防守者範圍中有 98, 87, 76
- 應該考慮過牌
```

#### 2. 完成明顯聽牌

```
牌面：A♠ J♠ 4♦ → 8♠

分析：
- 同花完成
- 對手可能有 K♠ x♠ 或 Q♠ x♠
- 需要謹慎繼續
```

## 轉牌下注尺度

### 取值下注尺度

| 牌面類型 | 建議尺度 | 原因 |
|----------|----------|------|
| 乾燥靜態 | 50-66% | 對手範圍 capped |
| 濕潤動態 | 66-75% | 拒絕對手權益 |
| 極強成牌 | 75-100% | 建立大底池 |

### 詐唬下注尺度

```
原則：詐唬尺度應該和取值一致

如果你取值用 66% pot
你的詐唬也用 66% pot
保持尺度一致性
```

### 小尺度 vs 大尺度

```
小尺度 (33-50%)：
- 維持寬範圍
- 適合範圍優勢明顯時
- 對手難以加注

大尺度 (66%+)：
- 極化範圍
- 適合有明確取值或詐唬
- 拒絕對手權益
```

## 轉牌加注策略

### 作為進攻者面對加注

```
對手在轉牌加注 = 非常強的信號

考慮因素：
1. 對手的加注頻率（一般玩家很少詐唬加注轉牌）
2. 你的牌力（是否能繼續？）
3. 籌碼深度（加注後是否 committed？）

一般建議：
- 超強牌：再加注
- 強牌：跟注
- 中等牌：通常棄牌
- 詐唬：根據情況決定
```

### 作為防守者考慮加注

```
轉牌加注的時機：
✓ 你有超強牌（set+）
✓ 對手很可能繼續下注
✓ 你有好的半詐唬（combo draw）
✓ 阻擋對手堅果

注意：
- 轉牌加注承諾更多籌碼
- 對手範圍已經收窄
- 純詐唬加注風險高
```

## Probe Bet（探測下注）

### 什麼是 Probe Bet？

當翻牌的進攻者在轉牌過牌後，你選擇下注。

```
流程：
你 Check（翻牌）
對手 Bet（C-bet）
你 Call
轉牌出來
你 Check
對手 Check ← 對手放棄主動權
你在河牌下注 = Probe Bet（也可在轉牌）
```

### Probe Bet 的時機

```
好的 Probe 時機：
✓ 轉牌對你有利（完成聽牌）
✓ 對手 C-bet 頻率高但轉牌放棄
✓ 你有 showdown value 想取值
✓ 你想詐唬代表擊中轉牌

不好的 Probe 時機：
✗ 轉牌對對手有利
✗ 對手 C-bet 頻率低（過牌可能是陷阱）
✗ 你沒有牌力也沒有權益
```

## 面對轉牌過牌

### 對手過牌給你

```
當對手翻牌下注、轉牌過牌：

他可能有：
1. 放棄的詐唬
2. 中等牌控池
3. 陷阱（較少見）

你應該：
- 強牌：下注取值
- 中等牌：可能過牌控池
- 聽牌：看免費牌或半詐唬
- 空氣：考慮詐唬或放棄
```

### 你選擇過牌

```
何時在轉牌過牌：

作為進攻者：
- 牌面危險，繼續無利可圖
- 你的牌力下降
- 控池策略

作為防守者：
- 繼續 check-call 線路
- 準備 check-raise
- 等待河牌決策
```

## 聽牌在轉牌的處理

### 計算權益

```
轉牌後你只剩一張牌：
用 2 法則計算勝率

Outs × 2 = 大約勝率 %

範例：
9 outs 同花聽牌 → 18%
8 outs 順子聽牌 → 16%
15 outs 組合聽牌 → 30%
```

### 面對下注

```
賠率計算：
對手下注 $50 進 $100 底池
你需要跟注 $50 贏 $200
賠率 = 50/200 = 25%

如果你有 9 outs（18%）：
直接賠率不足，但考慮：
- 隱含賠率
- 河牌可能詐唬成功
```

### 半詐唬加注

```
何時考慮加注：
✓ 你有強聽牌（combo draw）
✓ 對手會棄牌一些更好的牌
✓ 即使被跟注你仍有權益
✓ 籌碼深度允許

組合聽牌範例：
牌面：K♥ Q♥ 7♣ → 4♥
你有：J♥ T♥

你的權益：
- 同花：9 outs
- 順子：6 outs（扣除重複）
- 總計：15 outs ≈ 30%

這是好的半詐唬牌！
```

## 常見轉牌場景

### 場景 1：持續下注後

```
你：A♠ K♣ 在 CO 開池
對手：BB 跟注
翻牌：J♦ 8♣ 4♥
你 C-bet 50%，對手跟注
轉牌：2♠

分析：
- 轉牌是白板，沒有改變太多
- 對手範圍：Jx, 89, 小口袋對, 聽牌
- 你只有 A-high，但對手範圍 capped

選項：
A) 繼續下注 50-66%（代表強牌）
B) 過牌放棄

建議：混合策略，稍偏向繼續
```

### 場景 2：擊中轉牌

```
你：T♠ 9♠ 在 BTN
對手：BB 跟注你的開池
翻牌：K♥ 8♦ 3♣
你過牌（pot control）
轉牌：7♠

分析：
- 你完成順子！
- 對手可能有 Kx, 88, 33
- 現在應該下注取值

行動：下注 66-75% pot
目標：從 Kx 和口袋對取值
```

### 場景 3：聽牌沒完成

```
你：A♠ 5♠ 在 BB 防守
對手：CO 開池，翻牌 C-bet
牌面：K♠ 9♠ 2♣
你跟注（同花聽牌）
轉牌：J♦

對手再次下注 60%

分析：
- 同花沒完成
- 你有 9 outs
- 需要 28.5% 勝率跟注
- 你只有 18%

決定：
- 直接賠率不足
- 考慮隱含賠率和對手籌碼
- 如果河牌完成可以贏大底池 → 跟注
```

## 籌碼深度考量

### SPR（Stack to Pot Ratio）

```
SPR = 有效籌碼 / 底池

轉牌 SPR 影響策略：

SPR > 4：仍有空間操作
SPR 2-4：接近承諾
SPR < 2：基本上要 all-in 或 fold
```

### 承諾門檻

```
一般原則：
投入超過 1/3 籌碼 = 基本承諾

轉牌計算：
如果你的轉牌下注會讓你承諾
→ 要嘛現在 all-in
→ 要嘛考慮其他線路
```

## 常見錯誤

### 1. 盲目雙管
```
❌ 翻牌下注，轉牌自動再下注
✅ 評估轉牌是否改變局勢
```

### 2. 聽牌玩得太被動
```
❌ 每次都只是跟注
✅ 考慮半詐唬加注
```

### 3. 忽略轉牌影響
```
❌ 不在乎轉牌是什麼
✅ 分析轉牌對雙方範圍的影響
```

### 4. 籌碼管理失誤
```
❌ 轉牌下注後不夠籌碼河牌施壓
✅ 計劃好三條街的籌碼分配
```

## 總結

### 轉牌核心原則

1. **評估轉牌對範圍的影響**
2. **考慮籌碼深度和承諾**
3. **強牌繼續、弱牌放棄、聽牌計算**
4. **保持取值和詐唬的平衡**
5. **為河牌做好計劃**

### 轉牌決策清單

```
□ 這張牌對誰有利？
□ 我的牌力/權益如何？
□ SPR 是多少？
□ 對手的可能範圍？
□ 我的河牌計劃是什麼？
```
