# 位置基礎入門：為什麼位置是撲克最重要的概念

## 什麼是位置？

**位置**指的是你相對於按鈕（Dealer Button）的座位。位置決定了你在每輪下注中的行動順序。

**核心原則**：後行動的玩家有優勢，因為可以先看到對手的動作再做決定。

---

## 6-Max 座位圖

```
        [SB]  [BB]
          \    /
           \  /
    [BTN]-------[UTG]
           /  \
          /    \
        [CO]  [HJ]
```

| 位置 | 全名 | 行動順序 | 相對優勢 |
|------|------|----------|----------|
| UTG | Under The Gun | 第1 | 最差 |
| HJ | Hijack | 第2 | 差 |
| CO | Cutoff | 第3 | 好 |
| BTN | Button | 第4 | 最佳 |
| SB | Small Blind | 第5* | 差 |
| BB | Big Blind | 第6* | 中等 |

*翻前 SB/BB 最後行動，但翻後第一個行動

## 為什麼位置如此重要？

### 1. 信息優勢

**後位玩家**可以：
- 看到對手先行動
- 根據對手動作調整策略
- 更準確地讀取對手範圍

```
範例：
你在 BTN，對手在 CO 加注
→ 你知道他有強牌範圍
→ 你可以根據這個信息決定跟注、3-bet 或棄牌
```

### 2. 底池控制

**有位置時**：
- 可以選擇下注取值或過牌控池
- 對手很難知道你的牌力
- 可以在最後關閉行動

**沒有位置時**：
- 必須先行動，暴露意圖
- 對手可以根據你的動作調整
- 更難實現詐唬

### 3. 實現權益 (Realize Equity)

**有位置**：更容易把手牌的理論勝率轉化為實際獲利

```
範例：你拿著 76s
- 在 BTN：可以看到對手反應，靈活遊玩 → 高實現率
- 在 UTG：面對多人，容易被擠壓 → 低實現率
```

## 各位置策略概要

### UTG (Under The Gun) - 最緊

**特點**：
- 最早行動，還有 5 人待行動
- 被 3-bet 的機率最高
- 只玩最強的手牌

**RFI 範圍**：約 15% 的手牌
- 所有口袋對
- AQs+, AKo
- 少量同花連牌 (JTs+)

**心態**：「我的牌要強到足以面對 3-bet」

---

### HJ (Hijack) - 較緊

**特點**：
- 比 UTG 稍寬，但仍需謹慎
- 主要威脅來自 CO、BTN、盲位

**RFI 範圍**：約 18% 的手牌
- UTG 範圍 + AJo, KQs
- 更多同花連牌

---

### CO (Cutoff) - 開始放寬

**特點**：
- 只有 BTN 和盲位在後
- 可以開始偷盲
- 攻擊性開始提升

**RFI 範圍**：約 25% 的手牌
- HJ 範圍 + 更多同花 Ax
- 低對子
- 連牌

---

### BTN (Button) - 最寬

**特點**：
- 最佳位置！
- 翻後永遠最後行動
- 可以大量偷盲

**RFI 範圍**：約 40-50% 的手牌
- 幾乎所有同花牌
- 所有對子
- 大量連牌

**心態**：「這是印鈔位置，積極參與」

---

### SB (Small Blind) - 複雜位置

**特點**：
- 翻前最後行動（有利）
- 翻後第一個行動（不利）
- 已投入半個盲注

**策略**：
- 對前位開池：3-bet 或棄牌，少 call
- 面對 BTN 開池：大量 3-bet 防守

**注意**：SB 是長期虧損位置，減少損失是目標

---

### BB (Big Blind) - 防守位置

**特點**：
- 已投入 1BB，有底池賠率
- 翻前最後行動
- 需要防守一定範圍

**策略**：
- 利用底池賠率擴大防守範圍
- 面對小額開池：寬跟注
- 面對大額開池：收緊

---

## 位置對範圍的影響

### 開池範圍比較

| 位置 | 開池比例 | 範例手牌 |
|------|----------|----------|
| UTG | ~15% | AA-22, AKs-AQs, AKo |
| HJ | ~18% | + AJo, KQs, QJs |
| CO | ~25% | + A9s, KTs, 87s |
| BTN | ~45% | + K5s, Q8s, J7s |
| SB | ~40% | 開池或 limp (爭議策略) |

### 視覺化

```
開池範圍寬度:
UTG [====]                           15%
HJ  [=====]                          18%
CO  [========]                       25%
BTN [================]               45%
```

## 實戰應用

### 場景 1：相同手牌，不同位置

**手牌：A♠ T♣**

| 位置 | 動作 | 原因 |
|------|------|------|
| UTG | 棄牌 | 容易被主導 |
| HJ | 棄牌/邊緣 | 仍然危險 |
| CO | 開池 | 可接受 |
| BTN | 開池 | 標準打法 |

### 場景 2：面對開池

**對手 UTG 開池，你拿著 JJ**

| 你的位置 | 建議動作 | 原因 |
|----------|----------|------|
| HJ | 跟注/3-bet | 都可以 |
| CO | 跟注/3-bet | 稍偏向 3-bet |
| BTN | 3-bet | 利用位置優勢 |
| BB | 3-bet | 不想多人底池 |

## 常見錯誤

### 1. 忽略位置玩太多牌
❌ 在 UTG 玩 KTo
✅ 只在 CO+ 玩 KTo

### 2. 有位置卻不利用
❌ 有位置時總是被動
✅ 有位置時積極下注和加注

### 3. 沒位置卻太激進
❌ 在 SB 用邊緣牌 3-bet
✅ 在 SB 要嘛強 3-bet，要嘛棄牌

### 4. 不根據位置調整
❌ 每個位置同樣打法
✅ 根據位置調整範圍和策略

---

## 位置優勢量化

根據統計，同樣技術水平的玩家：

| 位置 | 每 100 手期望值 |
|------|-----------------|
| BTN | +10 到 +15 BB |
| CO | +5 到 +8 BB |
| HJ | +2 到 +4 BB |
| UTG | +1 到 +3 BB |
| BB | -20 到 -30 BB |
| SB | -35 到 -45 BB |

**BTN 的 EV 可能是 SB 的 3-4 倍！**

---

## 總結

1. **位置 = 信息 = 金錢**
2. **後位可以玩更多牌**
3. **前位需要更強的牌**
4. **BTN 是最賺錢的位置**
5. **SB/BB 的目標是減少損失**
