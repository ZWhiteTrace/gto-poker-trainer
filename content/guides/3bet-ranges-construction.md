# 3-Bet 範圍建構：從防守到攻擊的完整指南

## 什麼是 3-Bet？

**3-Bet** 是在某人開池加注（2-bet）後的再加注動作。

```
例子：
盲注 $1/$2
UTG 開池 $6（2-bet）
你在 BTN 加注到 $18（3-bet）
```

**為什麼叫 3-bet？**
- 1st bet = 大盲注
- 2nd bet = 開池加注
- 3rd bet = 對開池的再加注

---

## 3-Bet 的目的

### 1. 取值 (Value)
用強牌建立大底池

```
手牌：AA, KK, QQ, AKs
目的：被較差的牌跟注或 4-bet
```

### 2. 詐唬 (Bluff)
用弱牌逼迫對手棄牌

```
手牌：A5s, 76s, K9s
目的：直接偷取底池，或在翻後有後門權益
```

### 3. 隔離 (Isolate)
減少後面的玩家參與

```
情況：UTG 開池，你在 CO
3-bet 可以趕走 BTN 和盲位
達成 vs UTG 的單挑局面
```

---

## 極化 vs 線性範圍

### 極化範圍 (Polarized)

**定義**：只用最強和最弱的牌 3-bet，中等牌跟注

```
3-Bet：AA, KK, QQ, AKs + A5s, A4s, 76s, 65s
Call：JJ, TT, AQs, AJs, KQs, QJs
```

**適用場景**：
- 對抗開池範圍寬的位置（CO, BTN）
- 有位置時
- 深籌碼

**優點**：
- 詐唬牌有阻擋效應和後門權益
- 跟注範圍隱蔽性強
- 平衡取值和詐唬比例

---

### 線性範圍 (Linear / Merged)

**定義**：用所有強牌和中強牌 3-bet，不包含純詐唬

```
3-Bet：AA-JJ, AKs-ATs, AKo-AJo, KQs
不 3-Bet：A5s, 76s（這些棄牌或跟注）
```

**適用場景**：
- 對抗開池範圍緊的位置（UTG, HJ）
- 沒有位置時（盲位）
- 短籌碼

**優點**：
- 翻後更容易遊玩
- 面對 4-bet 不會陷入困境
- 對抗 station 類型對手更有效

## 各位置 3-Bet 策略

### BTN vs CO/HJ 開池

**建議範圍**：極化

```
取值 3-Bet：
AA, KK, QQ, JJ, TT
AKs, AKo, AQs, AQo

詐唬 3-Bet：
A5s, A4s, A3s, A2s（阻擋 Ax）
76s, 65s, 54s（同花連牌）
K9s, Q9s（有一定權益）

跟注：
99-66, AJs, ATs, KQs, KJs, QJs, JTs
```

**3-Bet 比例**：約 10-12%

---

### CO vs UTG/HJ 開池

**建議範圍**：線性偏極化

```
取值 3-Bet：
AA, KK, QQ, JJ
AKs, AKo, AQs

詐唬 3-Bet（少量）：
A5s, A4s

跟注：
TT, 99, AQo, AJs, KQs
```

**3-Bet 比例**：約 6-8%

---

### SB vs 各位置

**原則**：SB 沒有位置優勢，應該更極化

#### vs BTN 開池
```
3-Bet 範圍擴大：
取值：AA-99, AKs-ATs, AKo-AJo
詐唬：A5s-A2s, K5s-K3s, Q5s-Q3s

3-Bet 比例：約 12-15%
跟注：很少（因為位置劣勢）
```

#### vs UTG 開池
```
只用純取值 3-bet：
AA, KK, QQ, AKs, AKo

3-Bet 比例：約 4-5%
其餘強牌：跟注
```

---

### BB vs 各位置

**原則**：BB 已投入 1BB，有底池賠率，防守範圍最寬

#### vs BTN 開池
```
3-Bet：
取值：AA-TT, AKs-ATs, AKo-AQo
詐唬：A5s-A2s, K9s-K7s, 87s-65s

跟注範圍（超寬）：
幾乎所有對子、同花牌、連牌
```

#### vs UTG 開池
```
3-Bet（緊）：
AA, KK, QQ, JJ, AKs, AKo

跟注：
TT-22, AQs, AJs, KQs, 適合的同花連牌
```

## 3-Bet Sizing（尺度選擇）

### 基本公式

```
有位置 3-Bet = 開池尺度 × 3
沒位置 3-Bet = 開池尺度 × 3.5-4

範例：
開池 $6
→ 有位置 3-bet 到 $18
→ 沒位置 3-bet 到 $21-24
```

### 調整因素

| 因素 | 調整方向 |
|------|----------|
| 對手跟注頻率高 | 增加尺度 |
| 多人已跟注 | 增加尺度 |
| 短籌碼 | 減少尺度（或直接全下） |
| 取值手牌 | 可略增尺度 |

### 特殊情況

**Squeeze（擠壓）**：
當開池者後面有跟注者時，你的 3-bet 稱為 Squeeze

```
UTG 開池 $6
HJ 跟注 $6
你在 BTN：3-bet 到 $24-28（4x+ 開池）

原因：對抗更多玩家需要更大隔離
```

## 面對 4-Bet 的應對

### 4-Bet 後的策略

當你 3-bet 後被 4-bet：

```
繼續（5-bet/Call）：
AA, KK, QQ（永遠）
AKs（通常）
JJ, AKo（看情況）

棄牌：
詐唬 3-bet 牌（A5s, 76s 等）
中等取值牌（AQo, TT）
```

### 4-Bet 防守比例

根據對手 4-bet 尺度計算需要的防守頻率：

| 4-Bet 尺度 | 需防守比例 |
|------------|------------|
| 2.2x | ~45% |
| 2.5x | ~40% |
| 3x | ~33% |

## 常見錯誤

### 1. 詐唬比例過高
❌ 3-bet 範圍 70% 是詐唬
✅ 詐唬應約占 40-50%

### 2. 不根據位置調整
❌ 每個位置用同樣的 3-bet 範圍
✅ vs UTG 緊，vs BTN 寬

### 3. 尺度洩漏
❌ 強牌大 3-bet，弱牌小 3-bet
✅ 統一尺度，不暴露信息

### 4. 忽略對手類型
❌ 對所有人用同樣策略
✅ vs 緊手用線性，vs 寬手用極化

## 對手類型調整

### vs 緊玩家 (Nit)
```
減少詐唬 3-bet
用線性範圍
他們 4-bet = 給他們尊重
```

### vs 鬆玩家 (Fish)
```
增加取值 3-bet
減少詐唬（他們會跟注）
尺度加大
```

### vs 激進玩家 (LAG)
```
陷阱更多（跟注 AA/KK）
增加 4-bet 詐唬頻率
預期更多對抗
```

## 練習：建構你的 3-Bet 範圍

### 步驟 1：確定取值範圍
```
問：哪些牌你願意 5-bet 全下？
答：AA, KK, QQ, AKs
這些是你的核心取值 3-bet
```

### 步驟 2：添加次級取值
```
問：哪些牌 3-bet 後願意跟注 4-bet？
答：JJ, TT, AKo, AQs
這些可以加入取值範圍
```

### 步驟 3：選擇詐唬牌
```
選擇標準：
✓ 有阻擋效應（Ax 阻擋 AA, AK）
✓ 有後門權益（同花、順子）
✓ 能在翻後繼續施壓

好的詐唬牌：A5s, A4s, 76s, 65s
```

### 步驟 4：平衡比例
```
取值 : 詐唬 ≈ 60 : 40

如果取值有 6 種組合
詐唬應有 4 種組合
```

## 總結

1. **極化範圍**用於對抗寬範圍、有位置時
2. **線性範圍**用於對抗緊範圍、沒位置時
3. **尺度**：有位置 3x，沒位置 3.5-4x
4. **平衡**取值和詐唬比例約 60:40
5. **根據對手類型調整**策略
