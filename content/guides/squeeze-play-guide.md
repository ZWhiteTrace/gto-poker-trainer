# Squeeze Play 完全指南：利用壓力的進階武器

## 什麼是 Squeeze Play？

**Squeeze Play（擠壓）** 是撲克中最有力的翻前武器之一。

```
定義：
當有人開牌，一人或多人跟注後
你發動一個大的 3-bet
= Squeeze Play

名稱來源：
你把開牌者和跟注者「擠壓」在中間
兩邊都受到壓力
```

---

## Squeeze Play 為何有效？

### 死錢效應

```
範例場景：

CO 開牌到 $6
BTN 跟注 $6
SB 棄牌
你在 BB

底池已有：$6 + $6 + $0.5 + $1 = $13.5（死錢）

如果你 squeeze 到 $25：
- CO 面對 $19 加注進入 $38.5 底池
- BTN 面對 $19 加注進入 $38.5 底池
- 死錢讓你的攻擊更有利可圖
```

### 範圍優勢

```
為什麼跟注者的範圍弱：

開牌者的範圍：
- 可以是很寬的牌
- 包含強牌和投機牌

跟注者的範圍：
- 不夠強到 3-bet
- 不夠弱到棄牌
- 通常是中等強度牌

結論：
跟注者的範圍最容易被壓縮
他們很難面對大的 3-bet
```

### 位置壓力

```
當你 squeeze：

開牌者的困境：
- 後面還有跟注者
- 不知道跟注者會怎麼做
- 傾向棄掉邊緣牌

跟注者的困境：
- 前面還有開牌者
- 可能被夾在中間
- 只能用很強的牌繼續

結果：
兩人都會棄掉很多牌
你經常直接贏下底池
```

## Squeeze Play 的條件

### 最佳情況

```
✓ 好的條件：

1. 多個跟注者
   - 更多死錢
   - 更高 fold equity

2. 緊的開牌者
   - 容易棄牌
   - 4-bet 範圍窄

3. 弱的跟注者
   - 跟注範圍弱
   - 不會反擊

4. 你有位置
   - BB squeeze 最常見
   - SB 也可以

5. 適當的籌碼深度
   - 40-100bb 最佳
   - 太深風險大，太淺沒意義
```

### 避免的情況

```
✗ 不好的條件：

1. 開牌者很鬆且會 4-bet
   - 你的 fold equity 低

2. 跟注者很強或很黏
   - 他們不會輕易棄牌

3. 你沒有位置且籌碼很深
   - 翻後很難打

4. 只有一個跟注者且範圍強
   - 更像普通 3-bet
```

## Squeeze 尺寸

### 尺寸計算

```
基本公式：
Squeeze 大小 = 3-4x 開牌 + 每個跟注者 1x 開牌

範例：
CO 開 $6，BTN 跟 $6，你在 BB

計算：
- 基礎：3.5 × $6 = $21
- 加跟注者：$21 + $6 = $27
- 合理範圍：$24-$30

有多個跟注者：
CO 開 $6，BTN 跟 $6，SB 跟 $6

計算：
- 基礎：3.5 × $6 = $21
- 加跟注者：$21 + $6 + $6 = $33
- 合理範圍：$30-$36
```

### 尺寸調整因素

```
使用更大尺寸：

- 對手很黏（不願棄牌）
- 有多個跟注者
- 你沒有位置
- 想要極化範圍

使用較小尺寸：

- 對手很緊
- 你有位置
- 想要更多跟注
- 籌碼較淺
```

## Squeeze 範圍構建

### 價值 Squeeze

```
核心價值牌：
AA, KK, QQ, JJ, TT
AKs, AKo, AQs

這些牌：
- 對抗 4-bet 有優勢
- 被跟注也很強
- 是 squeeze 範圍的基礎
```

### 詐唬 Squeeze

```
好的詐唬牌：

Ax 同花（阻擋牌）：
A5s, A4s, A3s, A2s
- 阻擋 AA, AK
- 有 equity 和可玩性

同花連張：
76s, 65s, 54s
- 被跟注時有 equity
- 不怕對子

Kx 同花（部分）：
K9s, K8s
- 阻擋 KK
- 有一定可玩性
```

### 平衡的 Squeeze 範圍

```
範例 BB squeeze 範圍：

價值端（約 4-5%）：
TT+, AQs+, AKo

詐唬端（約 3-4%）：
A5s-A2s, K9s-K8s
87s-54s, J9s, T9s

總計：約 7-9% 範圍

調整：
- 面對緊玩家可以加寬
- 面對鬆玩家要收緊
```

## 不同位置的 Squeeze

### BB Squeeze

```
最常見的 squeeze 位置

優勢：
- 有底池賠率優勢
- 封閉行動
- 最多死錢

劣勢：
- 被跟注後沒位置
- 需要較大尺寸

範圍：
- 可以較寬
- 包含較多詐唬
```

### SB Squeeze

```
較少使用但有效

優勢：
- 有位置對 BB
- 可以利用位置

劣勢：
- BB 還沒行動
- 可能被 BB cold 4-bet

範圍：
- 應該較緊
- 減少詐唬比例
```

### Button Squeeze

```
在 BTN 對 EP 開牌 + MP 跟注

特點：
- 翻後有位置
- 但對手範圍較強

範圍：
- 主要價值導向
- 詐唬要謹慎選擇
```

## 面對 Squeeze 的應對

### 作為開牌者

```
被 squeeze 時的選擇：

1. 棄牌
   - 大部分邊緣牌
   - 沒有好的翻後勝算

2. 跟注
   - 有位置時
   - 對手 squeeze 太寬
   - 能實現 equity 的牌

3. 4-bet
   - 頂級價值牌
   - 部分詐唬（平衡用）

防守頻率：
通常只防守 15-25% 範圍
```

### 作為跟注者

```
被 squeeze 時：

大部分情況 → 棄牌
- 你的範圍最弱
- 前後都有壓力

少數繼續的牌：
- 強對子（JJ+）
- AK
- 偶爾的詐唬 4-bet

重要原則：
不要輕易跟注 squeeze
要麼 fold，要麼 4-bet
```

## 實戰範例

### 範例 1：經典 Squeeze

```
情況：
100bb 深度
CO（緊玩家）開 2.5bb
BTN（普通玩家）跟注
SB 棄牌
你在 BB 拿 A♠5♠

分析：
✓ 有兩個對手
✓ 開牌者緊，容易棄
✓ BTN 範圍弱
✓ A5s 有阻擋和可玩性

決策：
Squeeze 到 10-11bb
預期經常直接贏
```

### 範例 2：不適合 Squeeze

```
情況：
100bb 深度
BTN（激進玩家）開 2.5bb
SB（跟注站）跟注
你在 BB 拿 K♠8♠

分析：
✗ BTN 很激進，可能 4-bet
✗ SB 是跟注站，不會棄牌
✗ K8s 阻擋效應不強

決策：
棄牌或跟注
Squeeze 沒有好的 fold equity
```

### 範例 3：價值 Squeeze

```
情況：
80bb 深度
HJ 開 2.5bb
CO 跟注
BTN 跟注
SB 棄牌
你在 BB 拿 Q♠Q♣

分析：
✓ 三個對手，大量死錢
✓ QQ 對抗這些範圍很強
✓ 要建立底池

決策：
Squeeze 到 14-16bb
用大尺寸建立價值
準備好面對 4-bet
```

## 常見錯誤

### 1. 尺寸太小

```
錯誤：
CO 開 2.5bb，BTN 跟
你在 BB squeeze 到 6bb

問題：
- 給對手太好的賠率
- 經常被跟注
- 失去 fold equity

正確：
至少 squeeze 到 10bb+
```

### 2. 範圍太寬

```
錯誤：
用任何兩張牌 squeeze
「反正他們會棄牌」

問題：
- 被 4-bet 時很難處理
- 被跟注時沒有 equity
- 長期虧損

正確：
選擇有 blocker 或 equity 的牌
```

### 3. 不考慮對手類型

```
錯誤：
對跟注站 squeeze 詐唬

問題：
- 他們不會棄牌
- 浪費資金

正確：
根據對手調整範圍
對跟注站減少詐唬
```

### 4. 忽略籌碼深度

```
錯誤：
25bb 時還用大尺寸 squeeze

問題：
- 籌碼淺沒必要
- 可能直接 shove 更好

正確：
短籌碼考慮 shove
深籌碼使用標準尺寸
```

## 進階概念

### 反向 Squeeze

```
定義：
開牌者故意讓你 squeeze
然後 4-bet 你

識別信號：
- 對手突然開很多牌
- 面對你的 squeeze 常 4-bet
- 4-bet 範圍看起來很寬

應對：
- 收緊 squeeze 範圍
- 準備好打 5-bet pot
- 增加價值牌比例
```

### Squeeze 後的翻後策略

```
被跟注後：

有位置：
- 可以繼續施壓
- 利用位置優勢
- C-bet 頻率可以高

沒有位置：
- 更謹慎地選擇 C-bet
- 用強牌建立底池
- 準備放棄弱牌
```

## 總結

### Squeeze Play 核心

```
1. 利用死錢
   多個跟注者 = 更多價值

2. 攻擊弱範圍
   跟注者的範圍最容易被擠壓

3. 正確尺寸
   3-4x 開牌 + 每個跟注者 1x

4. 平衡範圍
   價值牌 + 有 blocker 的詐唬牌

5. 考慮對手
   對緊玩家更有效
```

### Squeeze 檢查清單

```
決定 squeeze 前：
□ 有多個跟注者嗎？
□ 開牌者容易棄牌嗎？
□ 跟注者範圍弱嗎？
□ 我的牌有 blocker 或 equity 嗎？
□ 尺寸足夠大嗎？
□ 我準備好面對 4-bet 嗎？
```
