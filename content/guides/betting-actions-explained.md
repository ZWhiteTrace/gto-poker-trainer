# 撲克下注動作完全詳解

## 基本下注動作

德州撲克有 5 種基本動作：

| 動作 | 英文 | 說明 | 何時可用 |
|------|------|------|----------|
| 棄牌 | Fold | 放棄手牌和已投入的籌碼 | 任何時候 |
| 過牌 | Check | 不下注，將行動權交給下家 | 沒人下注時 |
| 跟注 | Call | 匹配前面的下注金額 | 有人下注後 |
| 下注 | Bet | 第一個投入籌碼 | 沒人下注時 |
| 加注 | Raise | 增加下注金額 | 有人下注後 |

---

## 詳細動作解說

### 1. 棄牌 (Fold)

**定義**：放棄本輪，將手牌丟入棄牌堆

**使用時機**：
- 手牌太弱，不值得繼續
- 面對太大的下注，沒有足夠勝率
- EV（期望值）為負時

**重要原則**：
- 棄牌後無法拿回已投入的籌碼
- 棄牌是撲克中最常見的動作
- 好玩家知道何時該棄牌

```
情境：你拿著 72o，對手全下
正確動作：棄牌 ✓
錯誤思考：「我已經投了盲注，不如跟一下」✗
```

---

### 2. 過牌 (Check)

**定義**：不下注，將行動權傳給下一位

**使用時機**：
- 沒有人在你之前下注
- 你想要控制底池大小
- 你想要設陷阱（check-raise）
- 你的牌不值得下注取值

**重要原則**：
- 只有在沒人下注時才能過牌
- 過牌不代表軟弱
- 策略性過牌是進階技巧

```
翻牌後，你是大盲位：
牌面：K♠ 7♦ 2♣
手牌：A♠ A♥
考慮：過牌讓對手 bluff，或下注取值？
```

---

### 3. 跟注 (Call)

**定義**：投入與前面下注相同的金額

**使用時機**：
- 你有足夠的勝率繼續
- 你想要控制底池大小
- 你不想暴露手牌強度
- 底池賠率有利

**常見錯誤**：
- 沒有計劃地「跟到底」
- 忽略隱含賠率
- 用邊緣牌過度跟注

```
底池：$100
對手下注：$50
你需要跟注：$50
底池賠率：3:1 (需要 25% 勝率)
```

---

### 4. 下注 (Bet)

**定義**：在沒人下注時，第一個投入籌碼

**下注原因**：
1. **取值 (Value)**：你有好牌，想被較差的牌跟注
2. **詐唬 (Bluff)**：你沒有好牌，想讓較好的牌棄牌
3. **保護 (Protection)**：阻止對手免費看牌

**常見下注尺度**：

| 尺度 | 底池比例 | 使用場景 |
|------|----------|----------|
| 小注 | 25-33% | 乾燥牌面、範圍下注 |
| 中注 | 50-66% | 標準取值或詐唬 |
| 大注 | 75-100% | 極化範圍、濕潤牌面 |
| 超池 | >100% | 最大取值、河牌詐唬 |

---

### 5. 加注 (Raise)

**定義**：增加前面的下注金額

**加注原因**：
1. **取值**：你有強牌，想建立更大底池
2. **詐唬**：你想逼迫對手棄牌
3. **隔離**：只想和一位對手單挑
4. **獲取信息**：測試對手手牌強度

**加注規則**：
- 最小加注 = 前面下注的 2 倍
- 例：對手下注 $10 → 最小加注到 $20

**常見加注尺度**：

| 術語 | 尺度 | 說明 |
|------|------|------|
| Min-raise | 2x | 最小加注 |
| 標準加注 | 2.5-3x | 翻前常見 |
| 大加注 | 4x+ | 施壓或隔離 |
| All-in | 全部 | 全下 |

## 進階動作組合

### Check-Raise（過牌加注）

**流程**：先過牌 → 對手下注 → 你加注

**使用時機**：
- 設陷阱：你有強牌，想讓對手先下注
- 詐唬：代表強牌，逼迫對手棄牌
- 平衡：避免過牌總是顯得軟弱

```
你在大盲位，拿著 Set
翻牌：7♠ 7♥ 2♣
你：過牌
對手：下注 $30
你：加注到 $90 ← Check-Raise
```

---

### Donk Bet（反主動下注）

**定義**：沒有主動權的玩家先下注

**範例**：
- 翻前你跟注了對手的加注
- 翻牌你先下注（而不是過牌給加注者）

**使用時機**：
- 牌面對你範圍非常有利
- 你有極強或極弱的牌
- 改變遊戲節奏

---

### Float（飄注）

**定義**：用弱牌跟注，計劃在後面街道奪取底池

**範例**：
- 翻牌你沒有擊中，但跟注對手的 C-bet
- 轉牌對手過牌，你下注奪池

**使用條件**：
- 對手 C-bet 頻率高
- 你有位置優勢
- 牌面適合詐唬

## 下注尺度策略

### 翻前下注尺度

| 場景 | 建議尺度 |
|------|----------|
| RFI (首位開池) | 2-2.5BB |
| 3-Bet (對開池加注) | 3x 開池尺度 |
| 4-Bet | 2.2-2.5x 3bet 尺度 |
| 跛入後加注 | 3BB + 1BB/跛入者 |

### 翻後下注尺度

| 牌面類型 | 建議尺度 | 原因 |
|----------|----------|------|
| 乾燥 (A72r) | 25-33% | 範圍優勢大 |
| 中等 (KT6) | 50-66% | 平衡範圍 |
| 濕潤 (JT9) | 66-75% | 保護+取值 |
| 配對 (K77) | 25-33% | 範圍優勢 |

---

## 常見錯誤

### 1. 被動玩法
❌ 總是跟注，很少加注
✅ 主動下注和加注

### 2. 尺度不一致
❌ 強牌大注，弱牌小注
✅ 相同尺度平衡範圍

### 3. 忽略位置
❌ 在不利位置過度參與
✅ 根據位置調整動作

### 4. 不考慮對手
❌ 對所有人用同樣策略
✅ 根據對手類型調整

## 動作決策流程

```
1. 評估手牌強度
   ↓
2. 考慮位置
   ↓
3. 分析對手範圍
   ↓
4. 計算底池賠率
   ↓
5. 決定動作和尺度
   ↓
6. 執行
```

---

## 總結

| 動作 | 主要目的 | 關鍵考量 |
|------|----------|----------|
| Fold | 止損 | 勝率不足時果斷棄牌 |
| Check | 控池/陷阱 | 不暴露手牌強度 |
| Call | 繼續參與 | 確保有足夠賠率 |
| Bet | 取值/詐唬 | 選擇正確尺度 |
| Raise | 建池/施壓 | 考慮對手反應 |
