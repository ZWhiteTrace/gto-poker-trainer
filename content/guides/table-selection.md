# 選桌策略：找到最有利可圖的牌桌

## 為什麼選桌重要？

**選桌 (Table Selection)** 是撲克獲利的隱藏關鍵。

```
選桌的重要性：

一個簡單事實：
在強手滿桌的牌桌 → 你是最弱的
在弱手滿桌的牌桌 → 你是最強的

你的贏率差異：
好牌桌：可能 +10BB/100
差牌桌：可能 -2BB/100

選對牌桌 = 自動獲利更多
不用改變任何策略
```

---

## 好牌桌的特徵

### 看牌率高

```
什麼是高看牌率：

VPIP（自願投入底池率）：
- 行業平均：20-25%
- 好牌桌：30%+
- 夢幻牌桌：40%+

為什麼好：
- 對手玩太多牌
- 容易犯錯
- 你的價值牌得到支付

識別方法：
- 觀察幾手牌
- 多少人看翻牌
- 翻牌圈是否經常多人
```

### 底池大小

```
底池大小指標：

觀察：
- 平均底池是多少大盲
- 是否經常有全下
- 河牌是否常有大注

好的跡象：
- 平均底池 > 15BB
- 經常有人跟到河牌
- 有大的展示牌

壞的跡象：
- 小底池居多
- 翻前就結束
- 很少看到攤牌
```

### 玩家行為

```
業餘玩家特徵：

明顯的告訴：
- 拿到好牌很興奮
- 聊天愉快
- 喝酒

打法特徵：
- 跟注太多
- 不會棄掉對子
- 詐唬時機不對

桌上氣氛：
- 輕鬆友好
- 在開玩笑
- 不太認真
```

## 壞牌桌的特徵

### 職業玩家聚集

```
識別職業玩家：

外觀：
- 戴耳機
- 不聊天
- 穿著撲克周邊

打法：
- VPIP 低於 20%
- 有位置意識
- 詐唬時機正確

行為：
- 快速決策
- 不顯示情緒
- 認真專注
```

### 緊而無聊

```
緊牌桌的問題：

特徵：
- 很少人看翻牌
- 大部分手牌翻前結束
- 底池都很小

問題：
- 難以取值
- 對手都會棄牌
- 沒有魚

解決：
離開找更好的牌桌
```

## 線上選桌技巧

### 使用軟體工具

```
選桌軟體功能：

顯示的統計：
- 平均底池大小
- 看牌率
- 每小時手數
- 空座位

過濾功能：
- 按看牌率排序
- 找高平均底池
- 標記已知魚

等待名單：
- 加入好牌桌等待
- 多桌時優先選擇
```

### 多桌策略

```
多桌選桌原則：

數量 vs 質量：
- 8 張差牌桌 < 4 張好牌桌
- 不要為了量犧牲質

輪換策略：
- 定期檢查牌桌質量
- 魚離開就換桌
- 保持最優配置

建議：
- 每 30 分鐘檢查一次
- 至少保持 2-3 張好桌
- 寧缺毋濫
```

## 現場選桌技巧

### 觀察大廳

```
進入撲克室前：

觀察：
- 哪些桌子最熱鬧
- 哪些有歡笑聲
- 籌碼分布如何

詢問：
- 問員工哪桌最好玩
- 問常客推薦
- 了解今天的情況

等待：
- 不要隨便坐
- 等好位置
- 選擇對的桌子
```

### 座位選擇

```
好座位的重要性：

相對位置：
- 魚在你右邊最好
- 強手在你右邊不好
- 你要對他們有位置

具體策略：
- 找到最弱玩家
- 坐在他左邊
- 這樣你永遠有位置

如果不能選：
- 至少避開強手左邊
- 保護你的盲注
```

### 換座和換桌

```
何時換座：

好的換座時機：
- 你右邊的魚離開
- 新的魚坐在別處
- 強手坐到你左邊

何時換桌：
- 桌上魚都離開了
- 變成職業玩家桌
- 氣氛變得緊張

不要猶豫：
選桌是合法且重要的
不要因為「不禮貌」而留在差桌
```

## 不同級別的選桌

### 低級別選桌

```
低級別特點（$1/$2, $2/$5）：

機會多：
- 更多桌可選
- 更多業餘玩家
- 週末更好

選桌重點：
- 找喝酒的玩家
- 找剛坐下的新手
- 避開一直在玩的常客

時機：
- 週五晚上最佳
- 大型體育賽事時
- 假日晚上
```

### 高級別選桌

```
高級別特點（$5/$10+）：

更困難：
- 桌數較少
- 強手比例高
- 選擇有限

策略：
- 更注重座位
- 願意等待
- 考慮私人遊戲

額外考量：
- 認識場上的人
- 建立人脈
- 知道誰是魚
```

## 選桌心理學

### 魚的心理

```
業餘玩家的動機：

他們想要：
- 娛樂和樂趣
- 社交互動
- 刺激感

他們不在意：
- 贏率最大化
- 完美策略
- 長期結果

利用方法：
- 保持友好氣氛
- 不要讓他們感覺被欺負
- 讓他們想繼續玩
```

### 保護魚

```
為什麼要保護魚：

長期利益：
- 魚開心會繼續來
- 輸得舒服會再買入
- 壞氣氛趕走魚

具體做法：
- 不要批評打法
- 不要慢拖贏的底池
- 保持愉快氣氛

禁止行為：
- 嘲笑差的打法
- 過度慶祝勝利
- 創造敵對氣氛
```

## 常見選桌錯誤

### 1. 不選桌就坐

```
錯誤：
走進撲克室就坐下
不管牌桌質量

問題：
- 可能坐進強手桌
- 錯過好機會
- 降低贏率

正確：
花 5-10 分鐘觀察
選擇最好的選項
```

### 2. 留在差桌太久

```
錯誤：
「我已經坐下了」
「換桌太麻煩」

問題：
- 在差桌浪費時間
- 錯過好桌機會
- 贏率大幅降低

正確：
定期評估牌桌質量
隨時準備換桌
不要有沉沒成本心態
```

### 3. 挑戰強手

```
錯誤：
「我要證明我能贏他」
選擇和強手同桌

問題：
- 降低贏率
- 增加變異
- 浪費精力

正確：
錢都是一樣的
從魚那裡贏更容易
不需要證明什麼
```

### 4. 忽略位置

```
錯誤：
不在意座位選擇
只看牌桌整體

問題：
- 強手在你左邊
- 對魚沒有位置
- 難以發揮優勢

正確：
座位和牌桌一樣重要
永遠想要對弱手有位置
```
