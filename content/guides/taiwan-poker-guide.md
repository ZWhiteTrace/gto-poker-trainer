## ⚠️ 法律聲明

### 台灣撲克法規現況

```
重要提醒：

台灣目前對撲克的法律定位較為模糊：

1. 賭博罪相關
   - 刑法第 266 條：在公眾場所賭博可處罰
   - 但「技巧性遊戲」與「純賭博」的界線不明確

2. 協會/俱樂部形式
   - 多數線下場所以「協會」或「運動俱樂部」名義經營
   - 強調「競技」而非「賭博」性質
   - 使用籌碼而非現金進行

3. 線上平台
   - 伺服器設在海外的平台，法律適用較複雜
   - 玩家責任自負

本文僅提供資訊參考，不構成法律建議。
參與任何形式的撲克活動前，請自行評估風險。
```

## 線下撲克：台灣實體場所

### 撲克協會與俱樂部

台灣主要城市都有撲克協會或俱樂部，通常以「競技撲克」形式經營。

```
常見經營模式：

1. 會員制
   - 需要加入會員
   - 繳交會費或時間費
   - 參與俱樂部內比賽

2. 錦標賽制 (MTT)
   - 固定買入金額
   - 打到出局為止
   - 獎金依名次分配

3. 現金桌 (Cash Game)
   - 隨時加入離開
   - 籌碼可兌換
   - 通常有時間費或抽水
```

### 台北地區

```
台北是台灣撲克最活躍的地區：

特色：
- 俱樂部數量最多
- 常有國際賽事
- 各級別都有選擇

尋找方式：
- Facebook 搜尋「台北 撲克」「Taipei Poker」
- PTT Poker 版
- 撲克社群推薦

新手建議：
- 先從小級別錦標賽開始
- 觀察環境再決定是否常駐
- 注意俱樂部風評
```

### 其他地區

```
台中：
- 有數家撲克俱樂部
- 規模較台北小
- 社群相對緊密

高雄：
- 南部主要撲克據點
- 俱樂部選擇較少
- 有定期錦標賽

新竹/桃園：
- 竹科/工業區玩家多
- 小型聚會為主
- 可關注當地社群
```

### 國際賽事在台灣

```
曾在台灣舉辦的賽事：

APT (Asian Poker Tour)
- 亞洲最大巡迴賽之一
- 不定期在台灣舉辦
- 買入從小到大都有

WPT (World Poker Tour)
- 世界級賽事
- 偶爾有台灣站

其他區域賽事
- 各俱樂部自辦大型賽事
- 跨俱樂部聯賽
```

## 線上撲克：平台介紹

### 亞洲主流平台

#### Natural8 (N8)

```
特色：
- GGNetwork 旗下，亞洲最大網絡之一
- 台灣玩家眾多
- 介面支援繁體中文
- 軟體功能完整

級別範圍：
- 微額 (NL2-NL10)
- 小額 (NL25-NL100)
- 中額以上都有

適合：
✓ 台灣玩家首選
✓ 想打亞洲玩家池
✓ 軟體體驗重要
```

#### GGPoker

```
特色：
- 與 N8 同網絡
- 更國際化的品牌
- 大型錦標賽多
- WSOP 線上賽事合作

適合：
✓ 想打國際大賽
✓ MTT 玩家
✓ 追求賽事獎池
```

### 全球性平台

#### PokerStars

```
特色：
- 全球最大撲克平台
- 歷史最悠久
- 玩家池最大
- 軟體穩定

注意：
- 對手平均較強
- 亞洲時段人較少
- 部分地區可能有限制

適合：
✓ 想挑戰國際玩家
✓ 追求最大玩家池
✓ 經驗較豐富的玩家
```

#### 888poker

```
特色：
- 老牌平台
- 休閒玩家多
- 介面友善

適合：
✓ 休閒娛樂為主
✓ 不想太競爭
```

### 俱樂部制 App

#### PPPoker / ClubGG / PokerBros

```
特色：
- 俱樂部制，需要加入特定俱樂部
- 各俱樂部獨立運作
- 靈活的級別設定
- 手機為主

注意事項：
⚠️ 俱樂部品質參差不齊
⚠️ 需要找可信任的俱樂部
⚠️ 出入金流程各俱樂部不同
⚠️ 風險較傳統平台高

適合：
✓ 有認識的俱樂部
✓ 喜歡手機遊玩
✓ 想要特定的遊戲環境
```

## 新手入門建議

### 線下新手須知

```
第一次去俱樂部：

事前準備：
□ 先在網路上了解該俱樂部
□ 確認營業時間和地點
□ 準備好預算（建議 1-2 個買入）
□ 帶身分證件（可能需要）

到場注意：
□ 先觀察環境
□ 詢問工作人員規則
□ 從小級別開始
□ 不懂就問

禮儀須知：
□ 輪到你時要快速行動
□ 不要評論別人的打法
□ 保持籌碼整齊
□ 尊重發牌員和其他玩家
```

```
現場 vs 線上的差異：

1. 速度慢很多
   - 線上每小時 60-80 手
   - 現場每小時 25-35 手
   → 需要更多耐心

2. 讀人重要
   - 可以觀察對手表情、動作
   - 肢體語言是額外資訊
   → 練習觀察

3. 籌碼操作
   - 需要自己計算底池
   - 要會處理籌碼
   → 事先練習

4. 社交環境
   - 會有聊天互動
   - 可能有壓力
   → 保持冷靜
```

### 線上新手須知

```
開始線上撲克：

選擇平台：
□ 從主流大平台開始
□ 確認支援你的地區
□ 下載官方軟體

帳號設定：
□ 完成身份驗證
□ 設定存款限額
□ 了解出金流程

開始遊玩：
□ 先打 Play Money 熟悉介面
□ 從最小級別開始（NL2）
□ 設定自動補碼
□ 學會使用 HUD（可選）
```

```
線上安全須知：

⚠️ 只用官方軟體
⚠️ 不要共享帳號
⚠️ 定期更換密碼
⚠️ 小心釣魚網站
⚠️ 出入金用安全管道
```

### 資金管理

```
新手資金管理建議：

線上現金桌：
最低資金 = 20 個買入
→ NL2 (買入 $2) = $40 起步
→ NL5 (買入 $5) = $100 起步
→ NL10 (買入 $10) = $200 起步

線上錦標賽：
最低資金 = 50-100 個買入
→ $1 MTT = $50-100 起步

線下：
視俱樂部級別而定
→ 建議帶 2-3 個買入
→ 輸完就離開，不要追
```

```
重要原則：

1. 只用閒錢
   - 不影響生活的錢
   - 輸了不會心痛

2. 設定止損
   - 每次最多輸 X 元就走
   - 嚴格執行

3. 分離帳戶
   - 撲克資金和生活費分開
   - 不要混用

4. 記錄結果
   - 追蹤盈虧
   - 了解自己的表現
```

## 常見問題

### Q: 在台灣打撲克合法嗎？

```
A: 法律定位模糊。
線下俱樂部多以「協會」形式經營，
強調競技而非賭博。線上平台伺服器
在海外。建議自行評估風險。
```

### Q: 新手應該先打線上還是線下？

```
A: 建議先從線上開始。
- 成本較低（微額級別）
- 可以累積大量手牌
- 熟悉規則和策略
- 再嘗試線下體驗
```

### Q: 線上平台會作弊嗎？

```
A: 正規大平台有監管和認證。
- 選擇有執照的平台
- 避免不知名小平台
- RNG（隨機數生成器）有第三方認證
- 但俱樂部 App 風險較高，需謹慎
```

### Q: 如何找到台北的撲克俱樂部？

```
A: 幾種方式：
1. Facebook 搜尋相關社團
2. PTT Poker 版發問
3. 詢問撲克圈朋友
4. 線上社群推薦

記得先做功課，了解風評。
```

### Q: 需要多少錢才能開始？

```
A: 依級別而定：
- 線上微額：$50-100 美金起
- 線下小級別：台幣 3000-5000 起
- 建議從能承受損失的金額開始
```
