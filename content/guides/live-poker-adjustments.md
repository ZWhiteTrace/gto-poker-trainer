# 現場撲克調整：從線上到現場的轉變

## 現場撲克的根本差異

**現場撲克 (Live Poker)** 與線上撲克有本質上的不同，需要相應的策略調整。

```
關鍵差異：

線上撲克：
- 速度快（每小時 60-100 手）
- 對手匿名
- 多桌同時進行
- 樣本量大

現場撲克：
- 速度慢（每小時 25-35 手）
- 可以觀察對手
- 單桌專注
- 樣本量小
```

---

## 玩家類型差異

### 現場玩家的特點

```
普遍傾向：

1. 更被動
   - 很少 3-bet 詐唬
   - 傾向跟注而非加注
   - 河牌大注通常是強牌

2. 更鬆散
   - 翻前玩太多牌
   - 喜歡看翻牌
   - 「既然來了就要玩」心態

3. 更少調整
   - 不使用 HUD
   - 策略相對固定
   - 容易被剝削
```

### 常見現場玩家類型

| 類型 | 特徵 | 調整策略 |
|------|------|----------|
| 娛樂玩家 | 來玩樂，不在乎輸贏 | 多取值，少詐唬 |
| 退休族 | 保守，只玩強牌 | 尊重他們的大注 |
| 狂野玩家 | 激進，經常詐唬 | 設陷阱，抓詐唬 |
| 職業玩家 | 技術好，但少見 | 謹慎，避免複雜場景 |

## 翻前調整

### 1. 收緊開池範圍

```
原因：
- 現場很少棄牌（跛入多）
- 經常變成多人底池
- 需要更好的牌

調整：
線上 BTN 開池：45%
現場 BTN 開池：35-40%

尤其收緊：
- 前位開池
- 多人跛入後的加注
```

### 2. 增加加注尺度

```
原因：
- 現場玩家不在乎尺度
- 跟注率幾乎不變
- 可以建立更大底池

建議尺度：
線上：2-2.5BB 開池
現場：3-5BB 開池

有跛入者：
基礎 + 每個跛入者 1BB
例：3BB + 3 跛入者 = 6BB
```

### 3. 減少 3-bet 詐唬

```
原因：
- 現場很少 4-bet 詐唬
- 對手經常跟注
- 詐唬效率低

調整：
線上 3-bet 範圍：12% (含詐唬)
現場 3-bet 範圍：8% (主要取值)

保留詐唬：
只對特定激進對手使用
```

## 翻後調整

### 1. 增加 C-bet 取值

```
原因：
- 對手跟注範圍寬
- 第二對子、底對都會跟
- 取值範圍可以擴大

調整：
線上 C-bet：65% 頻率
現場 C-bet：55% 頻率，但取值更寬

範例：
線上：A9 在 A-7-2 可能不 C-bet
現場：A9 在 A-7-2 應該 C-bet
```

### 2. 減少詐唬頻率

```
原因：
- 現場玩家愛跟注
- 「我付錢看你的牌」心態
- 詐唬成功率低

調整：
河牌詐唬頻率：
線上：平衡（30-40%）
現場：極少（10-20%）

只在特定情況詐唬：
- 對手明顯很弱
- 有明確的威脅牌面
- 對手是可以棄牌的類型
```

### 3. 增加取值尺度

```
原因：
- 既然對手愛跟注
- 就讓他們付更多
- 最大化取值

調整：
線上取值下注：66% pot
現場取值下注：75-100% pot

超池取值也可以：
有些現場玩家會跟注 150% pot
```

## 觀察與讀牌

### 利用身體語言

```
現場獨有優勢：
可以觀察對手的身體語言

常見信號（謹慎使用）：

強牌信號：
- 手抖（腎上腺素）
- 假裝不在意
- 過於放鬆

弱牌信號：
- 故意展現強勢
- 盯著你看
- 快速下注
```

### 下注動作觀察

```
觀察重點：

下注方式：
- 猶豫下注 → 可能在詐唬
- 迅速下注 → 可能想嚇你
- 數籌碼方式 → 反映心理狀態

籌碼擺放：
- 整齊 → 可能是常客
- 雜亂 → 可能是新手
- 籌碼量 → 影響他的決策

切記：
這些只是輔助信息
不要過度依賴
```

### 聊天信息

```
對話中的線索：

「我可能有同花」→ 通常沒有
「我很弱」→ 可能很強
說太多話 → 可能在掩飾不安
沉默 → 可能在專注

談話策略：
- 保持友好但不透露信息
- 可以試探性提問
- 注意對手的反應變化
```

## 現場特有場景

### 1. 跛入底池

```
現場經常出現：
多人跛入的底池

應對策略：

隔離加注：
- 用更強的範圍
- 用更大的尺度
- 目標減少對手

在位置跛入：
- 偶爾可以接受
- 有高隱含賠率的牌
- 如小對子、同花連牌
```

### 2. 盲注攻防

```
現場盲注戰：
通常更保守

偷盲：
- 可以更頻繁
- 但尺度要夠大
- 對手很少防守

防守：
- 可以更緊
- 對手偷盲範圍窄
- 面對大注要尊重
```

### 3. Straddle

```
什麼是 Straddle：
UTG 位置盲注翻倍

應對：
- 收緊範圍
- 增加開池尺度
- 利用位置優勢

Straddle 環境：
底池更大 → 變異更大
需要更好的資金管理
```

## 資金管理

### 現場的變異

```
現場變異更大：

原因：
- 每小時手數少
- 底池平均更大
- 好壞運影響明顯

建議 Bankroll：
線上現金：20-30 買入
現場現金：30-50 買入

原因：
- 單次 session 樣本小
- 需要更大緩衝
```

### 場次管理

```
單次 Session：
- 設定時間限制（4-6 小時）
- 設定止損點
- 疲勞時離開

好的離場時機：
- 達到目標盈利
- 感到疲勞
- 遊戲變差
- 情緒波動
```

## 社交與禮儀

### 保持友好

```
現場撲克是社交遊戲：

為什麼重要：
- 娛樂玩家喜歡友好氛圍
- 他們會留更久
- 他們會輸更多給你

如何做：
- 不要教育對手
- 不要展現不滿
- 保持禮貌和尊重
- 適度聊天
```

### 避免的行為

```
不要做：
❌ 批評對手的打法
❌ 慢捲（Slow roll）
❌ 過度慶祝
❌ 抱怨運氣
❌ 角度射擊（Angle shooting）

要做：
✓ 保護你的手牌
✓ 行動要清楚
✓ 尊重發牌員
✓ 控制你的情緒
```

## 常見錯誤

### 1. 用線上策略打現場

```
❌ 高頻詐唬
✅ 減少詐唬，增加取值
```

### 2. 開池尺度太小

```
❌ 現場還是 2.5BB 開池
✅ 增加到 4-5BB 或更多
```

### 3. 忽略對手信息

```
❌ 只看自己的牌
✅ 觀察對手的行為和傾向
```

### 4. 過度依賴讀牌

```
❌ 完全根據「感覺」決定
✅ 讀牌只是輔助，基礎仍是策略
```

## 總結

### 現場調整核心

1. **翻前更緊，尺度更大**
2. **減少詐唬，增加取值**
3. **利用觀察，但不過度依賴**
4. **保持友好，享受遊戲**
5. **資金管理更保守**

### 現場撲克檢查清單

```
翻前：
□ 範圍夠緊嗎？
□ 加注尺度夠大嗎？
□ 3-bet 以取值為主嗎？

翻後：
□ 取值範圍擴大了嗎？
□ 詐唬頻率降低了嗎？
□ 尺度夠大嗎？

觀察：
□ 注意到對手的傾向了嗎？
□ 有特殊的身體語言嗎？
□ 下注模式有變化嗎？
```
