# 多人底池調整：當不只是單挑時

## 多人底池的根本差異

**多人底池 (Multiway Pot)** 有三個或更多玩家參與，這從根本上改變了策略。

```
核心差異：

單挑 (Heads-Up)：
- 只需要擊敗一個對手
- 可以用更寬的範圍參與
- 詐唬更有效

多人底池：
- 需要擊敗多個對手
- 需要更強的牌才能贏
- 詐唬價值大幅下降
```

---

## 為什麼多人底池不同？

### 1. 權益分配

```
單挑時：
你 vs 對手 = 50% vs 50% 平均

多人時：
你 vs 對手A vs 對手B = 33% vs 33% vs 33% 平均

但實際更複雜：
- 有人會擊中強牌
- 權益不均勻分佈
- 你需要更強的牌才能贏
```

### 2. 有人擊中的機率

```
計算：假設翻牌對每人擊中機率 30%

單挑：
對手擊中機率 = 30%

三人底池：
至少一人擊中 = 1 - (0.7 × 0.7) = 51%

四人底池：
至少一人擊中 = 1 - (0.7 × 0.7 × 0.7) = 66%

結論：人越多，越可能有人擊中強牌
```

### 3. 詐唬效果下降

```
單挑詐唬成功：
需要讓 1 人棄牌

三人底池詐唬成功：
需要讓 2 人都棄牌

如果每人棄牌率 50%：
單挑成功率 = 50%
三人成功率 = 50% × 50% = 25%

詐唬價值大幅下降！
```

## 多人底池的翻前調整

### 1. 收緊開池範圍

```
當有多人跟注預期時：

標準 BTN 開池：~45%
有人已跟注：~35%
兩人已跟注：~25%

原因：
- 需要更強的牌
- 位置優勢降低
- 攤牌價值更重要
```

### 2. 避免被主導的手牌

```
危險手牌：
❌ KJo, QTo, J9o
→ 容易被 KQ, KT, QJ 主導

更安全：
✅ 同花連牌 (87s, 76s)
→ 可以做成隱蔽的強牌

✅ 小對子 (44, 55)
→ 要嘛擊中 set，要嘛棄牌
```

### 3. 利用隱含賠率

```
多人底池的優勢：
底池更大 → 隱含賠率更高

適合的牌：
- 同花連牌：可以做大同花
- 小對子：擊中 set 贏大池
- 同花 Ax：堅果同花聽牌

不適合的牌：
- 高牌非同花：容易被主導
- 大對子：難以判斷是否領先
```

## 多人底池的翻後調整

### 1. C-bet 頻率大幅下降

```
單挑 C-bet：約 65-70%

三人底池 C-bet：約 30-40%

四人底池 C-bet：約 20%

原因：
- 更可能有人擊中
- 詐唬效果差
- 需要更強的牌下注
```

### 2. 下注尺度調整

```
多人底池下注尺度：

傾向使用更大尺度：
- 保護牌力
- 減少看牌人數
- 定義對手範圍

範例：
單挑：可以用 33% pot C-bet
三人：建議用 50-66% pot
```

### 3. 價值取向

```
多人底池策略核心：

減少詐唬，增加取值

原因：
1. 詐唬成功率低
2. 更可能有人跟注
3. 強牌可以獲得更多價值

實戰應用：
- 有強牌 → 下注取值
- 沒有牌 → 過牌放棄
- 聽牌 → 評估賠率
```

## 位置的影響

### 夾在中間 (Sandwich Position)

```
情況：你在兩個對手中間

問題：
- 你行動後還有人
- 可能被後面的人擠壓
- 信息劣勢

策略：
- 更加謹慎
- 用更強的牌下注
- 考慮過牌讓後面先行動
```

### 最後位置 (Last to Act)

```
情況：你在所有對手後面行動

優勢：
- 可以看到所有人的行動
- 決策信息最完整
- 可以控制底池

策略：
- 利用信息優勢
- 可以適度詐唬（如果前面都過牌）
- 取值更準確
```

## 牌面紋理的影響

### 乾燥牌面 (Dry)

```
牌面：A♠ 7♦ 2♣

多人底池分析：
- 擊中 A 的人會想取值
- 沒擊中的人會棄牌
- 不適合詐唬

策略：
- 有 A 就下注
- 沒有 A 通常過牌
- 詐唬只在特定情況
```

### 濕潤牌面 (Wet)

```
牌面：J♠ T♥ 8♦

多人底池分析：
- 很多人可能擊中
- 順子、兩對、同花聽牌
- 需要保護強牌

策略：
- 強牌必須下注
- 尺度要夠大
- 聽牌要評估賠率
```

### 配對牌面

```
牌面：K♠ K♦ 5♣

多人底池分析：
- 很少人有 K
- Trips 極為少見
- 可以適度施壓

策略：
- 可以增加 C-bet 頻率
- 但對手過牌-加注要尊重
- 保持警惕
```

## 多人底池常見場景

### 場景 1：你有頂對

```
你：A♠ J♣ 在 CO
牌面：A♦ 8♣ 4♥（三人底池）

分析：
- 頂對好踢腳
- 在單挑很強，多人要謹慎
- 可能有人有 A8, 44

策略：
- 下注約 50% pot
- 準備面對加注
- 不要過度承諾籌碼
```

### 場景 2：你有 Set

```
你：7♠ 7♥ 在 BTN
牌面：K♦ 7♣ 3♠（四人底池）

分析：
- Set 在多人底池很強
- 可以取大值
- 對手可能有 Kx 會跟注

策略：
- 下注 66-75% pot
- 建立大底池
- 準備在轉牌繼續
```

### 場景 3：你有同花聽牌

```
你：Q♥ J♥ 在 HJ
牌面：A♥ 8♥ 3♠（三人底池）
前面玩家下注 50% pot

分析：
- 9 outs 同花聽牌
- 大底池 = 好的賠率
- 如果擊中可以贏大池

計算：
底池 $30，對手下注 $15
你需要跟注 $15 贏 $60
賠率 = 25%，你有 ~35%（翻牌後）

策略：跟注（賠率有利）
```

### 場景 4：你沒有擊中

```
你：K♠ Q♣ 在 CO
牌面：J♦ 7♣ 4♠（四人底池）

分析：
- 你只有兩張高牌
- 四人底池很可能有人擊中
- 詐唬成功率極低

策略：
- 過牌棄牌
- 不要嘗試詐唬
- 等待更好的機會
```

## 進階概念

### 保護權益 vs 控制底池

```
多人底池的平衡：

保護權益：
- 下注讓聽牌付出代價
- 減少看牌人數
- 定義對手範圍

控制底池：
- 避免底池失控
- 保持靈活性
- 等待更多信息

建議：
中等牌力 → 適度保護 + 控制
強牌 → 積極建池
弱牌 → 放棄或等待
```

### 隔離下注

```
目的：減少對手數量

範例：
翻牌：K♦ 9♣ 5♥（四人底池）
你有：K♠ Q♠

策略：
- 下注 66-75%
- 希望只留下一個跟注者
- 變成單挑更容易遊玩
```

### 讀取下注意圖

```
多人底池下注的意義：

進攻者下注：通常有真正的牌力
（因為詐唬效果差）

多人過牌後下注：
可能是延遲的取值
或在利用弱勢

面對下注：
要比單挑更尊重對手
```

## 常見錯誤

### 1. 用單挑策略玩多人

```
❌ 像單挑一樣 C-bet 70%
✅ 多人底池 C-bet 降到 30-40%
```

### 2. 過度詐唬

```
❌ 「我來把大家詐走」
✅ 認識到詐唬成功率很低
```

### 3. 高估一對的價值

```
❌ 「我有頂對，應該下三街」
✅ 多人底池一對要謹慎
```

### 4. 忽略位置劣勢

```
❌ 在中間位置激進
✅ 考慮後面還有人要行動
```

## 總結

### 多人底池核心原則

1. **需要更強的牌**才能繼續
2. **詐唬價值大幅下降**
3. **C-bet 頻率降低**，尺度增大
4. **保護強牌**，放棄弱牌
5. **利用位置**和隱含賠率

### 多人底池檢查清單

```
翻前：
□ 這手牌適合多人底池嗎？
□ 我容易被主導嗎？
□ 有隱含賠率嗎？

翻後：
□ 這個牌面我應該 C-bet 嗎？
□ 我的牌力足以多人取值嗎？
□ 詐唬有意義嗎？
□ 我的位置如何？
```
