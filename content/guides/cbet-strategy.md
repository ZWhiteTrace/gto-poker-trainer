# C-bet 策略完全指南：翻後主動權的藝術

## 什麼是 C-bet？

**C-bet (Continuation Bet)** 是翻前加注者在翻牌圈繼續下注的行為。

```
定義：
你翻前加注
對手跟注
翻牌你繼續下注
= C-bet

為什麼重要：
1. 利用翻前主動權
2. 建立翻後優勢
3. 給對手壓力
4. 收取底池或建立價值
```

---

## C-bet 的目的

### 價值型 C-bet

```
目的：用強牌榨取價值

範例：
你在 BTN 開牌 A♠K♥
BB 跟注
翻牌：A♦7♣2♠

分析：
- 你有頂對頂 kicker
- 對手可能有 Ax, 中對, 聽牌
- 下注獲取價值
```

### 詐唬型 C-bet

```
目的：讓對手棄掉更好的牌

範例：
你在 CO 開牌 9♠8♠
BB 跟注
翻牌：K♦Q♣3♥

分析：
- 你完全沒中
- 但這個牌面對你有利
- 下注可能讓對手棄掉小對子
```

### 保護型 C-bet

```
目的：保護中等強度的手牌

範例：
你在 BTN 開牌 J♥J♠
BB 跟注
翻牌：T♦7♣4♥

分析：
- 你有超對但不是堅果
- 很多牌在後續街能超過你
- 下注保護，拒絕免費看牌
```

## C-bet 頻率

### 不同牌面的 C-bet 頻率

```
高頻 C-bet 牌面（70-100%）：

乾燥 A 高：A-7-2, A-9-3
- 你的範圍有更多 Ax
- 對手很難跟注

乾燥 K 高：K-8-3, K-7-2
- 類似道理
- 對手範圍較弱

配對牌面：7-7-2, K-K-5
- 你的範圍更可能有三條
- 對手很難連接
```

```
中頻 C-bet 牌面（40-70%）：

中等連接：J-T-5, Q-9-4
- 雙方都可能連接
- 需要更謹慎

低牌面：8-6-3, 7-5-2
- 對手的跟注範圍可能更好
- 適合混合策略
```

```
低頻 C-bet 牌面（20-40%）：

高度連接：J-T-9, 9-8-7
- 對手很可能連接
- 你的範圍優勢小

同花牌面：8♠5♠2♠
- 對手可能有同花聽牌
- 容易被 check-raise
```

## C-bet 尺寸

### 小注 C-bet（25-33% pot）

```
適用場景：

1. 乾燥牌面
   - A-7-2, K-8-3
   - 對手範圍弱，小注即可

2. 高頻率策略
   - 用很多牌下注
   - 小注減少風險

3. 三條街策略
   - 保留籌碼深度
   - 為後續街留空間

範例：
底池 $20，下注 $6-7
```

### 中等 C-bet（50-66% pot）

```
適用場景：

1. 中等結構牌面
   - J-T-5, Q-9-4
   - 需要更多保護

2. 取值為主
   - 有強牌時使用
   - 建立底池

3. 標準策略
   - 適用大多數情況

範例：
底池 $20，下注 $10-13
```

### 大注 C-bet（75-100% pot）

```
適用場景：

1. 濕潤牌面
   - J♠T♠9♦
   - 拒絕對手便宜看牌

2. 極化策略
   - 只用堅果和詐唬
   - 高風險高回報

3. SPR 較低時
   - 籌碼較淺
   - 準備全下

範例：
底池 $20，下注 $15-20
```

## 不同位置的 C-bet 策略

### IP C-bet（有位置）

```
優勢：
- 後行動，掌握更多資訊
- 可以控制底池大小
- 對手過牌你可以免費看牌

策略：
- 可以更高頻 C-bet
- 使用更多不同的尺寸
- 利用位置優勢發展
```

### OOP C-bet（沒有位置）

```
劣勢：
- 先行動，資訊較少
- 被 raise 時處於劣勢
- 無法免費看牌

策略：
- 頻率應該降低
- 選擇更有利的牌面
- 使用更極化的範圍
```

## 單挑 vs 多人底池

### 單挑底池 C-bet

```
特點：
- 只需要讓一人棄牌
- 成功率更高
- 可以更頻繁

策略：
- 高頻率（60-70% 整體）
- 根據牌面調整尺寸
- 考慮對手類型
```

### 多人底池 C-bet

```
特點：
- 需要讓多人棄牌
- 至少有一人中牌機率高
- 詐唬效率低

策略：
- 大幅降低頻率（30-40%）
- 主要用價值牌下注
- 詐唬需要更好的牌面配合
```

## 面對不同對手的調整

### 面對跟注站

```
特徵：
- 很少棄牌
- 愛看所有翻牌
- 到河牌才放棄

調整：
- 減少詐唬 C-bet
- 擴大取值範圍
- 用更大的尺寸
```

### 面對緊手被動玩家

```
特徵：
- 很少加注
- 沒中就棄牌
- 好打

調整：
- 增加 C-bet 頻率
- 任何牌面都可下注
- 面對加注要尊重
```

### 面對激進玩家

```
特徵：
- 頻繁 check-raise
- 喜歡施壓
- 不容易放棄

調整：
- 降低 C-bet 頻率
- 用更強的範圍下注
- 準備好面對加注
```

## C-bet 後的後續街

### 轉牌策略

```
翻牌 C-bet 後被跟注：

對手範圍縮窄為：
- 成手牌（對子以上）
- 聽牌
- 還在浮游的牌

你的選擇：

1. 二次開火 (Double Barrel)
   - 你有強牌
   - 好的轉牌（完成聽牌或嚇人牌）
   - 對手範圍弱

2. 過牌控池
   - 中等強度手牌
   - 不好的轉牌
   - 底池已經夠大

3. 放棄
   - 完全的空氣
   - 沒有改進機會
   - 對手表現很強
```

### 河牌策略

```
考量因素：

1. 對手的跟注範圍
   - 經過兩條街還在的牌
   - 通常較強

2. 你的手牌強度
   - 能打敗對手範圍嗎？
   - 是否值得第三次下注？

3. 詐唬的可行性
   - 牌面結構
   - 你的形象
   - 對手類型
```

## C-bet 常見錯誤

### 1. 過度 C-bet

```
錯誤：
每個翻牌都 C-bet
不考慮牌面結構

問題：
- 被對手利用
- 浪費籌碼
- 無法平衡範圍

正確：
根據牌面調整頻率
有些翻牌應該過牌
```

### 2. 尺寸不當

```
錯誤：
永遠用同樣尺寸
不考慮牌面特性

問題：
- 乾燥面下太大浪費錢
- 濕潤面下太小給免費牌

正確：
乾燥面用小注
濕潤面用大注
```

### 3. 不考慮對手

```
錯誤：
對跟注站詐唬 C-bet
對激進玩家頻繁 C-bet

問題：
- 詐唬沒效果
- 被 check-raise 壓制

正確：
根據對手類型調整
觀察並適應
```

### 4. 多人底池過度 C-bet

```
錯誤：
3-4 人底池還像單挑一樣 C-bet

問題：
- 成功率極低
- 經常遇到抵抗
- 浪費資金

正確：
多人底池大幅收緊
主要用價值牌
```

## 進階 C-bet 概念

### 範圍優勢

```
定義：
你的整體範圍在這個牌面比對手強

範例：
你在 BTN 開牌，BB 跟注
翻牌：A-K-7

分析：
- 你的範圍有更多 Ax, Kx, AK
- BB 的跟注範圍較弱
- 你有明顯的範圍優勢
- → 高頻小注 C-bet
```

### 堅果優勢

```
定義：
你的範圍有更多超強牌組合

範例：
你在 BTN 開牌，BB 跟注
翻牌：8-7-6

分析：
- 這個牌面 BB 可能有更多順子
- 你的範圍有較少的 T9, 54
- BB 有堅果優勢
- → 降低 C-bet 頻率
```

### 阻擋效應

```
定義：
你的手牌阻擋對手可能的強牌

範例：
你有 A♠5♠，翻牌 A♦7♣2♥
你 C-bet，對手跟注

轉牌 K♣，你繼續下注

分析：
- 你的 A 阻擋對手的 Ax
- 對手更可能有 Kx 或聽牌
- 阻擋牌讓你的詐唬更有效
```

## C-bet 練習框架

### 牌面分析步驟

```
1. 評估牌面結構
   - 乾燥還是濕潤？
   - 高牌還是低牌？
   - 有沒有同花可能？

2. 比較範圍
   - 誰有範圍優勢？
   - 誰有堅果優勢？

3. 考慮位置
   - 你有位置嗎？
   - SPR 是多少？

4. 對手類型
   - 會經常棄牌嗎？
   - 會頻繁加注嗎？

5. 決定頻率和尺寸
   - 綜合以上因素
   - 做出決策
```

## 總結

### C-bet 核心原則

```
1. 根據牌面調整
   - 乾燥面高頻小注
   - 濕潤面選擇性大注

2. 考慮範圍優勢
   - 有優勢時更積極
   - 沒優勢時謹慎

3. 適應對手
   - 對跟注站多取值
   - 對激進玩家謹慎

4. 位置影響
   - IP 可以更積極
   - OOP 需要更謹慎

5. 人數影響
   - 單挑可以更頻繁
   - 多人要大幅收緊
```

### C-bet 檢查清單

```
每次決定 C-bet 時：
□ 牌面結構是乾燥還是濕潤？
□ 我有範圍優勢嗎？
□ 對手會經常棄牌嗎？
□ 我有位置嗎？
□ 這是單挑還是多人底池？
□ 我的尺寸合適嗎？
```
